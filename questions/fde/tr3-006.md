# PagedAttention v2 相比 v1 做了哪些关键改进？它主要解决了什么性能瓶颈？

PagedAttention v1 核心解决了KV Cache显存碎片化问题，提出了将KV块分页存储的机制。而 v2 主要针对计算开销进行了优化，特别是在处理请求并发时上下文切换的开销。v1 中每个批次处理可能需要重算某些KV块或频繁的数据搬运。v2 引入了更高效的内核实现，支持更灵活的块共享和并行解码优化，减少了CPU和GPU之间的通信开销，并显著提高了Cache命中时的计算吞吐。它主要解决了在极高并发场景下，虽然显存利用率高了（得益于v1），但计算速度反而受限于内存读写带宽和调度开销的问题。
