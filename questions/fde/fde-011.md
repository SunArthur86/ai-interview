# 客户的 AI 应用回答效果不好,你会从哪些角度排查和优化

- **AI 效果优化排查 SOP(5 层模型)**

- **Layer 1: 数据层(最常见)**
- 知识库覆盖够不够?补充文档
- 数据分块合不合理?调整 chunk_size/overlap
- 向量化模型好不好?换更好的 Embedding

- **Layer 2: 检索层**
- 检索结果准不准?人工检查 Top-5
- 是否需要 Rerank?加 Cross-Encoder
- Query 是否需要改写?HyDE/子问题分解

- **Layer 3: Prompt 层**
- System Prompt 够不够清晰?
- 是否给了足够的上下文?
- 有没有约束幻觉的指令?
- Few-shot 示例是否有帮助?

- **Layer 4: 模型层**
- 模型能力够不够?换更大的模型
- Temperature 合不合适?事实问答用 0-0.3
- 上下文窗口够不够?分段处理

- **Layer 5: 评估层**
- 有没有评估体系?建立黄金测试集
- 用户反馈机制?点赞/点踩收集
- 定期回归测试?每次改动跑测试集

- **经验**:80% 的问题出在数据层和 Prompt 层,不要急于换模型.
