# 如何帮客户计算 AI 项目的 ROI(投资回报率)

- **AI 项目 ROI 计算 FDE 模板**

- **成本侧**
- FDE 实施(一次性 2-4周)
- API 费用(按 token 计算月度成本)
- GPU 服务器(5000-20000元/月)
- 向量数据库(1000-5000元/月)
- 运维人力(5000元/月)

- **收益侧**
- AI 客服:替代 N 个客服 x 月薪
- 文档处理:原工时 x 时薪 - AI 处理成本
- 代码助手:开发者人数 x 30%效率提升 x 时薪
- 销售辅助:增量订单 x 利润率

- **ROI 示例**
- AI 客服:月成本 15000 元(API+服务器+运维),月收益 45000 元(替代3个客服),ROI=200%,回本周期约 2 个月

- **关键技巧**
1. 量化时用客户自己的数据
2. 保守估算(收益打 7 折,成本加 30%)
3. 展示'不做的成本'(竞争对手已经在用 AI)
