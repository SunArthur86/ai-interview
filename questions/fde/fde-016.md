# 如何帮客户计算 AI 项目的 ROI(投资回报率)

- **AI 项目 ROI 计算 FDE 模板**

- **成本侧**
1. **FDE 实施 (一次性)**: 需求分析、数据清洗、POC 开发，通常耗时 2-4 周。可按人天计算。
2. **推理 API 费用 (持续性)**: 按 Token 计算月度成本。需估算日均调用量、平均上下文长度。公式：`Cost = (Input Tokens * Price_in + Output Tokens * Price_out) * RPM * 60 * 24 * 30`。
3. **GPU 服务器 (持续性)**: 若私有化部署，需考虑硬件折旧或租赁。含算力卡（5000-20000元/月，视显卡型号而定）。
4. **存储与数据库 (持续性)**: 向量数据库（1000-5000元/月）、对象存储（S3/OSS）费用。
5. **运维人力 (持续性)**: 系统监控、Prompt 迭代优化、版本更新（建议预留 5000+元/月或 0.5 FTE）。

- **收益侧**
1. **AI 客服**: 
   - 收益 = `(替代的客服人数 * 月薪) + (提升转化率带来的增收)`。
   - 注意：通常 AI 客服作为 "Level 1" 支持，能拦截 60%-80% 常见问题。
2. **文档处理 (OCR/抽取)**: 
   - 收益 = `原人工处理工时 * 人数 * 时薪 - (AI 处理成本 + 人工复核成本)`。
   - 关键点：需计算 "人工抽检/纠错" 的残留成本。
3. **代码助手**: 
   - 收益 = `开发者人数 * (效率提升百分比 20-30%) * (时薪 + 机会成本)`。
   - 量化指标：代码生成速度、减少 Context Switch 时间。
4. **销售/知识辅助**: 
   - 收益 = `增量订单数 * 平均利润率`。
   - 或减少销售培训周期的时间成本。

- **ROI 示例**
- **场景**: 50 人客服团队，引入 AI 客服拦截 60% 咨询。
- **成本**: 
  - API+服务器+运维: 15000 元/月。
  - 实施一次性投入: 50000 元。
- **收益**: 
  - 节省 30 人 (60%拦截率 * 50人) * 平均月薪 8000 元 = 240,000 元/月。
  - **年化收益**: 240,000 * 12 = 2,880,000 元。
  - **年化成本**: 15,000 * 12 + 50,000 = 230,000 元。
  - **ROI**: (2,880,000 - 230,000) / 230,000 * 100% ≈ 1152%。
  - **回本周期**: 50,000 / (240,000 - 15,000) ≈ 0.22 个月 (约 7 天)。

- **## 常见考点**
1. **隐性收益的量化**: 面试官常问如何量化 "用户体验提升" 或 "品牌形象"，可回答通过客户满意度评分 (CSAT) 变化、NPS (净推荐值) 或复购率波动来关联估算。
2. **Token 成本估算细节**: 需区分 Input 和 Output Token 价格差异（通常 Output 贵 2-3 倍），以及是否计入 Context Window 的上下文缓存成本。
3. **风险与沉没成本**: 回答应提到 AI 模型的不稳定性（如幻觉处理的人力成本）作为潜在风险，以及技术选型错误导致的模型切换沉没成本。
4. **私有化 vs API 的 ROI 拐点**: 能够解释在什么调用量级下，自建 GPU 集群的折旧成本会低于调用 OpenAI/商业 API 的成本（通常是数亿 Token/月级别）。
