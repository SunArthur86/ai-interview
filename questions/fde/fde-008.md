# 如何把一个 AI 应用快速部署到客户的服务器上

- **AI 应用客户部署 Checklist**

- **1. 容器化(最重要)**
```dockerfile
FROM python:3.11-slim
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```

- **2. docker-compose 一键启动**
```yaml
**version：** '3'
**services：** 
**app：** 
**build：** .
**ports：** ["8000:8000"]
**environment：** 
- API_KEY=${API_KEY}
**depends_on：** [redis]
**redis：** 
**image：** redis:7-alpine
```

- **3. 部署步骤**
```
# 客户服务器上
git clone <repo>
cd <project>
cp .env.example .env  # 填入 API Key
docker-compose up -d
# 验证
curl localhost:8000/health
```

- **4. 常见坑**
- 客户服务器是 ARM 架构 → Docker buildx 跨平台构建
- 客户没有公网 → 用内网 IP + 反向代理
- 客户防火墙限制 → 只开放必要端口
- GPU 驱动版本不匹配 → 提前确认 CUDA 版本

- **FDE 经验**:永远准备一个 `install.sh` 脚本,客户只需一行命令就能装好.
