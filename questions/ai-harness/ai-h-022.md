# LangGraph在生产环境中部署需要注意什么?如何管理Agent状态

- **LangGraph生产部署要点:**

- **1. 状态持久化**
生产环境用PostgreSQL/Redis而非MemorySaver

- **2. 超时与限制**
- 设置最大递归深度
- 单步超时 + 全局超时

- **3. 人机协作**
在关键节点暂停,等待用户输入

- **4. 并行执行**
使用Send API并行分发任务,结果自动合并到状态

- **5. 可观测性**
LangSmith自动追踪 + 自定义callback记录指标

- **6. 错误恢复**
从最近checkpoint恢复 + 工具调用幂等设计

- **部署架构:**
用户 → FastAPI → LangGraph Worker → PostgreSQL(状态) → Redis(缓存) → LLM API
