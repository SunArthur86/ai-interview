# Semantic Router(语义路由)是什么?它如何高效做意图分类

- **Semantic Router:** 基于语义相似度的意图路由,无需训练分类器.

- **核心思想:**
- 预定义一组「意图描述」
- 用户输入与意图描述做embedding相似度
- 最相似的意图 = 路由目标

- **示例:**
```python from semantic_router import Route

# 定义路由
billing = Route( name="billing", utterances=[ "我的账单有问题", "怎么退款", "支付失败", "发票相关", ] )

tech = Route( name="tech_support", utterances=[ "代码报错了", "API调用失败", "如何集成SDK", ] )

# 路由
router = SemanticRouter(routes=[billing, tech]) result = router("我想取消订阅")  # → billing ```

- **优势:**
1. **无需训练** - 只需几条示例
2. **速度快** - embedding比对,比LLM调用快100倍
3. **可扩展** - 新增意图只需加示例
4. **零样本** - 不需要标注数据

- **应用场景:**
- 客服意图路由
- Agent工具选择
- RAG索引选择
- 内容分类
