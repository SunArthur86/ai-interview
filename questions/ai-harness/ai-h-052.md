# Claude Code 的架构是怎样的?它的 Agent Loop 是如何设计的

- **Claude Code** 是 Anthropic 推出的 CLI AI 编程助手,其架构是现代 Agent 设计的参考标杆.

- **核心架构:**

- **1. Agent Loop 设计:**
- 接收用户指令 -> LLM 推理(thinking)-> 工具调用 -> 观察结果 -> 继续推理
- 最大 25 轮迭代
- 支持上下文压缩(超长时自动摘要)
- 每轮都带完整上下文(无状态请求)

- **2. 工具集:**
- 文件操作:Read、Write、Edit、Glob
- 执行:Bash 命令执行
- 搜索:Grep 代码搜索
- MCP:通过 MCP 协议连接外部工具

- **3. 系统提示词关键设计:**
- 角色定义:你是 Claude Code,Anthropic 的 AI 助手
- 工具使用规则:优先搜索再修改、改动后验证
- 安全约束:不执行危险命令、需用户确认

- **4. 上下文管理:**
- 每轮带完整对话历史
- Prompt Caching 减少重复计算
- 超过阈值时自动压缩(保留摘要)

- **设计哲学:** 无状态请求 + 前缀缓存 + 上下文压缩,而非复杂的有状态 Agent.
