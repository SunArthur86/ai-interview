# Mastra框架的核心设计是什么?它和LangChain(Python生态)有什么区别

**Mastra:TypeScript原生LLM应用框架**

**核心定位:** 为TS/JS全栈开发者设计的AI应用框架

**核心功能:**
1. **Workflow** - 类型安全的工作流编排
2. **Agent** - 内置Agent抽象
3. **RAG** - 文档处理+检索
4. **Memory** - 短期/长期记忆
5. **Tool** - 类型安全的工具定义

**vs LangChain:**

| | LangChain (Python) | Mastra (TypeScript) |
|--|---------------------|---------------------|
| 语言 | Python | **TypeScript** |
| 类型安全 | 弱 | **强** |
| 前端集成 | 需要Vercel AI SDK | **原生支持** |
| 部署 | Python服务器 | **Edge/Serverless** |
| 生态 | 最大 | 增长中 |

**适合场景:**
- Next.js/Remix全栈应用
- Edge函数部署
- TS/JS技术栈团队
- 需要端到端类型安全
