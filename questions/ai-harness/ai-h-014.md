# 如何处理RAG中的表格、图表、公式等非文本内容

- **非文本内容RAG方案:**

1. **表格处理**
- 提取为Markdown/CSV格式
- 用LLM生成表格摘要
- 存储为:摘要(embedding) + 原始表格(精确返回)
- 查询时:摘要检索 → 原始表格注入Prompt

2. **图表/图片处理**
- VLM(如GPT-4V/LLaVA)生成图片描述
- 描述做embedding
- 原图URL/路径存储

3. **公式处理**
- MathPix API转换为LaTeX
- LaTeX文本做embedding
- 渲染后的公式图片用VLM描述

4. **PDF复杂版面**
- 使用Unstructured.io/Marker解析
- 保留页面结构(标题/段落/表格/图)
- 带页面号的metadata

- **多模态RAG架构:**
```
文档 → 解析器(文本/表格/图)
→ 分别处理(文本embedding/VLM描述/表格摘要)
→ 统一向量索引
查询 → 检索 → 多模态结果聚合 → LLM生成
```
