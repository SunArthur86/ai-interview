# LLM应用如何实现多租户隔离和权限控制?RAG知识库如何做文档级权限

- **多租户隔离方案:**

| 层级 | 方案 | 示例 | |------|------|------| | 数据隔离 | 租户独立的向量索引 | tenant_a_index, tenant_b_index | | 元数据过滤 | 同一索引+metadata过滤 | filter: {tenant_id: 'a'} | | 应用隔离 | 每租户独立部署 | Docker/K8s namespace |

- **文档级权限控制:**

1. **Metadata标注**
```json { **"content"：** "高管薪酬报告", **"metadata"：** { **"doc_id"：** "doc_123", **"access_level"：** "executive", **"department"：** "finance", **"tenant_id"：** "company_a" } } ```

2. **检索时过滤**
```python
# Qdrant
results = qdrant.search( collection_name="docs", query_vector=embedding, query_filter={ **"must"：** [ {"key": "tenant_id", "match": {"value": user.tenant_id}}, {"key": "access_level", "match": {"any": user.permissions}} ] } ) ```

3. **RBAC + ABAC**
- RBAC: 基于角色(admin/editor/viewer)
- ABAC: 基于属性(部门/项目/安全等级)
