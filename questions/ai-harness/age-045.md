# 生产 AI 应用中,如何做 Token 成本归因和优化

- **Token 成本管理** 是生产 AI 应用的核心运营指标.

- **成本归因:**

  - **1. 请求级追踪:** 每次请求记录 input_tokens、output_tokens、模型、单价,计算实际费用.

  - **2. 功能级分账:** 按 API 端点或功能模块分组统计.如 RAG 检索花了多少、代码生成花了多少.

  - **3. 用户级分账:** 按用户/租户统计用量,用于计费和配额管理.

  - **4. 成本标签:** 给每个请求打标签(项目、环境、实验版本),支持多维度切片分析.

- **优化手段:**

  - **1. 模型降级:** 简单任务用小模型(GPT-4o-mini、DeepSeek-V3),复杂任务才用大模型

  - **2. Prompt 压缩:** 减少冗余上下文、精简 System Prompt

  - **3. 缓存:** Prefix Caching、语义缓存(相似请求返回缓存结果)

  - **4. 批处理:** Batch API 通常有 50% 折扣

  - **5. 上下文管理:** 定期压缩对话历史、限制工具返回长度
