# 什么是 Code World Model(CWM)?它为什么是代码智能的突破

- **Code World Model(CWM)** 是 Meta 于 2025 年 11 月提出的代码世界模型,是首个能理解代码执行过程的模型.

- **核心突破:**

传统代码模型(如 Codex、StarCoder)只理解代码的文本语法,不理解代码执行后的行为.CWM 理解代码的执行轨迹(Execution Trace).

- **工作原理:**
1. 训练时加入代码执行轨迹数据
2. 模型不仅学习代码文本,还学习代码运行后的状态变化
3. 能预测代码执行结果,而不只是生成代码

- **关键能力:**
1. 执行结果预测:给定代码和输入,预测输出
2. Bug 检测:通过对比预期执行轨迹和实际执行轨迹发现 bug
3. 代码修复:基于执行理解生成修复方案
4. 测试生成:自动生成覆盖关键执行路径的测试

- **Benchmark 表现:** SWE-bench 通过率 65.8%,大幅超越之前的方法.

- **意义:** 从理解代码文本 -> 理解代码行为,是代码智能从语言模型走向世界模型的关键一步.
