# CrewAI在生产中如何使用?有哪些常见的坑

- **CrewAI生产最佳实践:**

- **1. Agent设计** - 角色明确、工具精确

- **2. Task设计** - 明确expected_output格式

- **3. 常见的坑:**

| 坑 | 症状 | 解决 |
|----|------|------|
| 角色重叠 | Agent互相抢任务 | 明确职责边界 |
| 无限委派 | Agent不停互相委派 | allow_delegation=False |
| Token爆炸 | 上下文无限增长 | max_iter限制 |
| 格式不一致 | 下游Agent无法解析 | 用output_schema |
| 工具不可用 | Agent调错误工具 | 精确描述工具 |

- **4. 性能优化:**
- 用haiku/flash做简单步骤
- 缓存中间结果
- 并行执行独立任务
- 设置max_rpm限流
