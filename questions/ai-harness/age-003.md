# 为什么说本地 tiktoken 计数只能「估算」

实际计费依赖服务商的分词器版本、特殊 token、多模态输入等；不同模型与版本可能不一致。本地计数用于预算控制与截断，最终应以 API 返回的 usage 与账单对账。
