# Ragas框架如何评估RAG系统?有哪些核心指标?如何部署

- **Ragas评估指标体系:**

- **检索质量指标:**
1. **Context Precision** - 检索到的上下文中相关部分的比例
2. **Context Recall** - 金标准答案所需的信息是否被检索到
3. **Context Relevancy** - 检索上下文与问题的相关性

- **生成质量指标:**
1. **Faithfulness** - 答案是否忠于检索到的文档(无幻觉)
2. **Answer Relevancy** - 答案是否回应了用户问题

- **使用方式:**
```python
from ragas import evaluate
from ragas.metrics import (
faithfulness, answer_relevancy,
context_precision, context_recall
)

result = evaluate(
dataset=eval_dataset,  # 包含question/answer/context/ground_truth
metrics=[
faithfulness,
answer_relevancy,
context_precision,
context_recall
]
)
print(result.scores)  # 每个指标的分数
```

- **评估数据集构建:**
- 人工标注(question + ground_truth)
- LLM自动生成(从文档生成问答对)
- 从用户日志中提取

- **最佳实践:** 建立持续评估Pipeline,每次RAG配置变更后自动运行评估
