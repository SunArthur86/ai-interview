# LLM应用的成本优化策略有哪些?Token级别如何精细化管理

- **成本优化全景:**

- **1. 模型路由(省40-70%)**
简单分类用Flash/Haiku,复杂推理用Sonnet/Opus

- **2. Prompt Caching(省50-80%系统prompt)**
OpenAI/Anthropic支持前缀缓存,系统prompt只计费一次

- **3. 上下文压缩(省30-50%)**
对话摘要替代完整历史,工具输出截断

- **4. 批量推理(省50%)**
OpenAI Batch API(24小时内返回),价格减半

- **5. 结构化输出省Token**
JSON比自然语言更紧凑,用schema约束输出长度

- **6. 模型蒸馏**
用大模型生成训练数据,蒸馏到小模型

- **成本监控:**
每次调用记录cost = input_tokens * input_price + output_tokens * output_price
