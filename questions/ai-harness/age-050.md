# OPRO 和 DSPy 是什么?它们如何实现自动 Prompt 优化

- **OPRO(Optimization by PROmpting)** 和 **DSPy** 是两种自动优化 Prompt 的方法.

- **OPRO(Google DeepMind):**
- 核心思想:用 LLM 自己来优化 Prompt
- 流程:给 LLM 一批训练样本 + 当前 Prompt + 历史尝试 -> LLM 提出新 Prompt -> 在测试集上评估 -> 保留最优
- 本质是元启发式搜索(类似遗传算法),LLM 是变异算子

- **DSPy(Stanford):**
- 核心思想:把 Prompt Engineering 变成编程问题
- 流程:定义模块(Signature)-> 组合管道(Module)-> 自动编译(Compiler)优化
- 编译器会自动尝试不同的 Prompt 策略(Few-shot、Chain-of-Thought 等)
- 使用 Bootstrap Few-Shot:自动选择最佳示例

- **对比:**
- OPRO:优化自然语言 Prompt 文本,适合单步 Prompt
- DSPy:优化整个 Pipeline(Prompt + 示例 + 步骤),适合复杂流水线

- **效果:** 在多数 benchmark 上,自动优化超过人工调优 5-20%.
