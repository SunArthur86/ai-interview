# 大模型推理时为什么会出现重复输出?有哪些缓解方法

- **重复原因:**

1. **概率分布尖锐化** - 模型对某些token概率过高
2. **训练数据偏差** - 训练数据中有重复模式
3. **上下文强化** - 之前生成的重复内容强化后续重复

- **缓解方法:**

| 方法 | 原理 | 参数 |
|------|------|------|
| **repetition_penalty** | 降低已出现token的概率 | 1.1-1.3 |
| **no_repeat_ngram_size** | 禁止重复n-gram | 3-4 |
| **frequency_penalty** | 按出现次数惩罚 | 0.1-0.5 |
| **presence_penalty** | 出现过就惩罚 | 0.1-0.5 |
| **Temperature调高** | 增加随机性 | 0.7-0.9 |
| **Top-p调大** | 扩大候选集 | 0.92-0.95 |

- **no_repeat_ngram_size 原理:**
- 设 n=3
- 每次生成新token时,检查是否与已生成内容形成3-gram重复
- 如果重复,跳过该token

- **生产建议:** repetition_penalty=1.1 + temperature=0.7 + top_p=0.92
