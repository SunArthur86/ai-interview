import torch
import torch.nn as nn
class LayerNorm(nn.Module):
def __init__(self, normalized_shape, eps=1e-5, elementwise_affine=True):

super(LayerNorm, self).__init__()
self.eps = eps
self.elementwise_affine = elementwise_affine
# 如果需要可学习参数,则初始化 gamma 和 beta
if self.elementwise_affine:
self.gamma = nn.Parameter(torch.ones(normalized_shape))
self.beta = nn.Parameter(torch.zeros(normalized_shape))
**else：**
self.gamma = None self.beta = None
def forward(self, x):
# 计算均值和方差
mean = x.mean(dim=-1, keepdim=True)
var = x.var(dim=-1, keepdim=True, unbiased=False)
# 进行标准化
x_normalized = (x - mean) / torch.sqrt(var + self.eps)
# 如果有 gamma 和 beta,则进行缩放和平移
if self.elementwise_affine:
x_normalized = self.gamma * x_normalized + self.beta
return x_normalized

3. 逐行解释
normalized_shape : 指的是要标准化的维度大小(通常是最后一维).
elementwise_affine : 控制是否进行可学习的缩放和平移(默认为True).
初始化时,我们需要根据 elementwise_affine 的值来决定是否引入可学习的参数 gamma
和 beta .gamma 用于缩放,beta 用于平移,它们的形状都与输入数据的最后一维相同.
**计算均值：** x.mean(dim=-1, keepdim=True) 计算输入 x 在最后一维的均值,使用
keepdim=True 保持原有的维度以便后续操作.
**计算方差：** x.var(dim=-1, keepdim=True, unbiased=False) 计算输入 x 的方
差,其中 unbiased=False 确保我们使用有偏方差估计(在深度学习中更常用).

**标准化：** 将 x 的每个元素减去均值,然后除以标准差(即方差的平方根),并且加上一个小
**缩放和平移：** 如果 elementwise_affine 为 True ,则对标准化后的结果进行缩放(乘
以 gamma )和平移(加上 beta ).
3. 返回结果:最后,标准化后的张量被返回,经过可选的缩放和平移.
