# 什么是 Embedding(向量化)?它在 AI 中有什么用

- **Embedding** 是将文本(或图片)转换为固定维度的向量(如 768 维浮点数组),使语义相近的内容在向量空间中距离也近.

- **用途**
  1. **语义搜索**:'苹果手机' 和 'iPhone' 向量距离近
  2. **RAG 检索**:从知识库中找最相关的文档
  3. **聚类分类**:相似内容自动分组
  4. **推荐系统**:基于向量相似度推荐

- **直觉理解**
  - king - man + woman ≈ queen
  - 向量空间中'语义关系'可以用方向表示

- **常用模型**:BGE、text-embedding-ada-002、E5、M3E

- **一句话理解**:Embedding = 把文字变成'坐标',让计算机用距离来衡量'语义相似度'.
