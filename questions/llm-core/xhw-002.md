# LoRA是怎样进行初始化矩阵的

LoRA通过低秩分解进行高效微调，其初始化策略至关重要：

1. **A矩阵**：使用随机高斯分布初始化，提供模型更新所需的随机性和方向。
2. **B矩阵**：初始化为零矩阵。

**原理**：初始时 $\Delta W = A \times B = 0$，确保微调开始时模型行为与预训练模型完全一致，避免初始阶段对模型造成剧烈扰动，保证训练的稳定性。
