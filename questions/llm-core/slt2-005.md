# FlashAttention的原理是什么？为什么能加速？

FlashAttention通过优化GPU内存访问模式来加速Attention计算。

问题：标准Attention需要将N×N的attention矩阵写入HBM（高带宽内存），N很大时内存读写成为瓶颈。

FlashAttention核心优化：
1. Tiling（分块）：将Q、K、V分成小块加载到SRAM（片上缓存）中，在SRAM内完成计算
2. Recomputation（重计算）：不存储中间attention矩阵，反向传播时重新计算。用计算换内存
3. Kernel Fusion（算子融合）：将QKᵀ → softmax → dropout → AV融合成一个CUDA kernel，减少HBM读写

FlashAttention v2改进：
- 减少非矩阵乘法运算
- 优化并行度（沿序列维度并行）
- 更好的线程块分配

效果：训练速度提升2-4倍，显存占用从O(N²)降到O(N)。长序列提升更明显。
