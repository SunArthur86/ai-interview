# 多模态大模型的架构有哪些？

多模态LLM让模型理解和生成文本、图像、音频等多种模态：

**1. 视觉-语言模型（VLM）：**

**架构一：投影层**
- Vision Encoder（CLIP/ViT）提取图像特征
- 线性层/MLP将视觉特征投影到文本特征空间
- 与文本token一起输入LLM
- 代表：LLaVA、Qwen-VL
- 简单有效，但投影层信息损失

**架构二：交叉注意力**
- LLM中插入cross-attention层
- 视觉特征作为K、V
- 代表：Flamingo

**架构三：原生多模态**
- 图像token和文本token在同一个Transformer中处理
- 代表：GPT-4V、Gemini

**2. 音频-语言模型：**
- Audio Encoder → LLM
- Whisper encoder + LLM

**3. 全模态：**
- Gemini Ultra：原生支持文本、图像、音频、视频

**趋势：**从拼接式（projector）走向原生多模态，统一token空间。
