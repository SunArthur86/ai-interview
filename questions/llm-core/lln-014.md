# 什么是Self-consistency和Best-of-N

两种方法都是基于聚合的测试时计算方法，即在大模型推理阶段通过生成多个回答并聚合以获得最终输出。

Self-consistency：让大模型生成多个CoT推理路径及多个答案，通过多数投票选出最终答案。

Best-of-N：让大模型生成N个回答，使用Verifier、ORM或PRM给每个回答评分，选择分数最高的作为最终回答。
