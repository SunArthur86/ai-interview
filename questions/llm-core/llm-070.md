# Transformer模型中计算量(FLOPs)的分析

### 1. FLOPs 基础计算
**FLOPs** (Floating Point Operations) 指浮点运算次数，用于衡量计算量。
*   **矩阵乘法：** 对于矩阵 $A \in \mathbb{R}^{m \times n}$ 和 $B \in \mathbb{R}^{n \times k}$，计算 $C=AB$ 约需 $2mnk$ 次浮点运算（$mnk$ 次乘法 + $mnk$ 次加法）。

### 2. Transformer 单层 FLOPs 分析
假设参数：Batch Size=$b$, 序列长度=$s$, 隐藏层维度=$d_{model}$, FeedForward 维度=$d_{ff}$ (通常为 $4d_{model}$), 词表大小=$V$。

#### (1) Embedding 层
虽然本质是查表操作，但若视为矩阵乘法 ($OneHot \times EmbeddingMatrix$)，计算量约为 **$2bshV$**。

#### (2) Self-Attention 部分
*   **计算 Q, K, V：** 三个线性层，输入 $[b, s, d_{model}]$。每层计算量为 $2bsd_{model}^2$。三层共约 **$6bsd_{model}^2$**。
*   **Attention Scores ($QK^T$)：** 计算量约为 $2bs^2d_{model}$。
*   **Attention 加权 ($Softmax \times V$)：** 计算量约为 $2bs^2d_{model}$。
*   **输出投影 ($O$)：** 计算量约为 $2bsd_{model}^2$。
*   **Attention 小计：** 约 **$4bs^2d_{model} + 8bsd_{model}^2$**。

#### (3) Feed-Forward (MLP) 部分
*   两次线性变换（升维和降维），维度为 $d_{model} \to 4d_{model} \to d_{model}$。
*   计算量约为 $2 \cdot 2bsd_{model}(4d_{model}) = 16bsd_{model}^2$。

### 3. 总结
在 Transformer 中，计算量主要由** Attention 中的 $s^2$ 项（序列长度平方）** 和 **MLP 中的 $d_{model}^2$ 项** 构成。随着序列长度 $s$ 增加，Attention 的计算开销增长最快。
