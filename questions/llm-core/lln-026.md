# SFT中的loss是怎么做计算的

SFT中loss仅计算response部分与label的交叉熵，通过loss_mask屏蔽prompt部分，具体流程：1.get_batch获取tokens、labels、loss_mask等；2.model forward得到output_tensor；3.loss_func用loss_mask加权计算损失。
