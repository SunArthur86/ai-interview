# DeepSeek V3 的 MTP(多令牌预测)是什么?和 Medusa 有什么区别

- **MTP (Multi-Token Prediction):**

传统模型一次预测1个token.MTP一次预测多个未来token.

- **DeepSeek V3 的 MTP 实现:**
1. **主模型** 预测第 i 个token
2. **MTP模块** 预测第 i+1, i+2... 个token
3. MTP模块共享主模型的embedding层
4. 训练时用MTP loss辅助训练
5. **推理时可丢弃MTP模块**(不增加推理成本)

- **vs Medusa:**
| | Medusa | DeepSeek MTP |
|--|--------|-------------|
| 多头位置 | 解码器上 | 独立MTP模块 |
| 训练目标 | 推测解码 | 辅助训练+推测解码 |
| 训练方式 | 后加微调 | 联合从头训练 |
| 推理 | 增加额外计算 | 可选启用 |

- **核心价值:** 训练时提升表征学习能力,推理时可选启用加速.
