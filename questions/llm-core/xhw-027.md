# Llama如何处理梯度消失/爆炸

🦄在深度神经网络的训练过程中，梯度消失（vanishing gradients）和梯度爆炸（exploding 
gradients）是常见的挑战。它们会导致模型训练困难，无法有效学习到数据的特征。下面将
从根源上逐个概念进行解释，并探讨Llama模型是如何应对这些问题的。
39.1 梯度消失和梯度爆炸的根源
梯度消失
- 原理： 在反向传播过程中，梯度需要通过链
式法则层层传递。如果激活函数的导数小于1
（如Sigmoid或Tanh），多个小于1的数相
乘，导致梯度指数级衰减，最终接近于零。
- 影响： 梯度过小，权重更新缓慢，网络无法
有效学习深层次的特征。
梯度爆炸
- 原理： 相反，如果激活函数的导数大于1，多
个大于1的数相乘，梯度会指数级增长，导致
数值不稳定，甚至出现NaN。
- 影响： 梯度过大，权重更新剧烈，模型容易
陷入不稳定状态，无法收敛。
39.2 Llama模型的架构特点
Llama模型是基于Transformer架构的大型语言模型，具有以下特点：
- 多头自注意力机制（Multi-Head Self-Attention）： 能够捕捉序列中不同位置的依赖关系。
- 残差连接（Residual Connections）： 直接将输入添加到输出，缓解梯度消失问题。
- 层归一化（Layer Normalization）： 在每一层对神经元进行归一化，稳定网络训练。
