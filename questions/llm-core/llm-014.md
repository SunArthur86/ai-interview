# GLM的自回归空白填充方法与BERT中的使用遮蔽语言模型(MLM)

有什么不同,为自然语言理解(NLU)任务带来了哪些优势 🐠在自然语言处理领域,预训练语言模型是提升模型性能的关键技术.BERT和GLM是两种常见 的预训练模型,它们在预训练任务上采用了不同的方法:BERT使用了遮蔽语言模型 (Masked Language Modeling,MLM),而GLM采用了自回归的空白填充方法.理解这两 种方法的区别以及GLM在自然语言理解任务中的优势,对于深入掌握预训练模型的原理和应 用具有重要意义. 关键概念对照表

**概念**
概念 解释 预训练语言模型 在大规模文本上训练的语言模型,用于下游任务 自编码模型 通过重建输入来学习表示,如BERT

自回归模型 基于先前输出预测下一个元素,如GPT 遮蔽语言模型(MLM) 随机遮蔽输入中的词,模型预测被遮蔽的词 空白填充方法 在输入中引入空白,模型自回归地填充空白
1. 基础概念解释
预训练语言模型 预训练语言模型通过在大规模无监督文本数据上进行训练,学习语言的语法和语义信息.然后,可以 将预训练得到的模型应用于各种下游任务,如文本分类、问答系统等,以提高模型的性能. 自编码模型与自回归模型 • 自编码模型(Autoencoding Models):如BERT,使用双向上下文信息,通过重建被遮蔽的输入 来学习语言表示. • 自回归模型(Autoregressive Models):如GPT,使用单向上下文信息,基于前面的词预测下一 个词.
2. BERT的遮蔽语言模型(MLM)
工作原理 BERT的MLM方法是在输入序列中随机遮蔽一些词,然后让模型根据上下文预测被遮蔽的词. 示意图 **输入序列：** "我爱[MASK]天" **模型预测：** "我爱北京天" 1 2

**特点**
特点 • **双向性：** 模型可以同时利用左侧和右侧的上下文信息. • **信息泄露问题：** 在训练过程中,模型可以看到被遮蔽词的未来信息,这在生成任务中不适用.
3. GLM的自回归空白填充方法
工作原理 GLM的自回归空白填充方法是在输入序列中引入空白,让模型以自回归的方式逐步填充这些空白. 示意图 **输入序列：** "我爱 [ ] 天"
**填充过程：**
1 2

**步骤1：** 预测"北" => "我爱 北 [ ] 天" **步骤2：** 预测"京" => "我爱 北京 天" 3 4

**特点**
特点 • **自回归填充：** 模型以自左向右的方式,逐步预测并填充空白. • **统一的生成和理解：** 同时适用于自然语言理解和生成任务.
4. 两种方法的区别
训练目标 **BERT的MLM：** 重建被遮蔽的 词,训练目标是最大化被遮蔽 词的概率. **GLM的自回归填充：** 预测空白 处的词序列,训练目标是最大 化空白填充序列的概率. 上下文利用 **BERT：** 利用双向上下文,但存 在信息泄露的问题. **GLM：** 以自回归方式填充,避 免了信息泄露. 应用范围 **BERT：** 主要用于理解任务,如 文本分类、问答匹配等. **GLM：** 兼顾理解和生成任务, 如机器翻译、文本摘要等.
5. GLM在自然语言理解任务中的优势
更好的上下文建模 GLM通过自回归的方 式,能够更好地捕捉上 下文之间的依赖关系, 提升对长距离依赖的建 模能力. 统一的框架 GLM采用统一的预训练 任务,兼容了自编码和 自回归模型的优势,使 其在多种下游任务中表 现出色. 避免信息泄露 自回归的填充方式确保 模型在预测时只利用过 去的信息,避免了 MLM方法中可能存在 的信息泄露问题. 提升模型泛化能力 由于GLM在预训练时需 要预测连续的空白序 列,模型被迫学习更深 层次的语言结构和语义 信息,从而提升了泛化 能力. 🍋GLM的自回归空白填充方法通过结合自编码和自回归模型的优势,提供了一种统一的预训练 框架.相比于BERT的MLM方法,GLM在避免信息泄露的同时,更好地建模了上下文依赖关 系,提升了模型在自然语言理解任务中的性能.
