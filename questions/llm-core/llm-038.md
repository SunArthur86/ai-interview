# 介绍一下Llama的结构

LLaMA 模型在标准 Transformer 架构的基础上进行了多项改进，以提升训练稳定性和性能。以下是主要结构特点：

### 1. RMS Pre-Norm (预归一化)
*   **RMSNorm**：LLaMA 使用 **RMSNorm** 替代传统的 Layer Norm。RMSNorm 去掉了均值中心化的减法操作，仅基于均方根进行缩放。公式为 $\bar{x}_i = \frac{x_i}{\sqrt{\frac{1}{n}\sum x_j^2}} \cdot \gamma_i$。计算更高效，且效果相当。
*   **Pre-Norm**：将归一化层放在注意力层和 FFN 层**之前**（即 $x' = x + Sublayer(LayerNorm(x))$）。这有助于稳定深层网络的梯度传播，缓解梯度消失或爆炸问题。

### 2. SwiGLU 激活函数
*   LLaMA 在前馈神经网络（FFN）中使用了 **SwiGLU** 替代传统的 ReLU。
*   SwiGLU 结合了 Swish 激活函数和门控线性单元（GLU），虽然计算量增加了约 1/3，但显著提升了模型的性能和收敛速度。

### 3. RoPE (Rotary Positional Embeddings)
*   使用 **旋转位置编码** 替代绝对位置编码。RoPE 通过旋转矩阵将位置信息注入到 Query 和 Key 的向量中，能够更好地捕捉相对位置关系，且具有外推性。

### 4. 其他细节
*   **GQA (Grouped-Query Attention)**：在 LLaMA 2/3 中采用了 GQA 来加速推理（如上文所述）。

### 6. 关键细节补充
*   **SwiGLU 结构细节**：FFN 层的维度计算为 $FFN_{dim} = \frac{2}{3} \times 4 \times h_{dim}$（向上取整以适应 Tensor Core）。具体操作为 $SwiGLU(x, W, V, W_2) = (Swish(xW) \otimes xV)W_2$。
*   **RoPE 实现**：在 Attention 计算前，对 Q 和 K 分别应用旋转矩阵。$f(x, m) = R_{\Theta, m} x$，其中 $R$ 是根据位置 $m$ 生成的旋转矩阵。
*   **Bias=False**：LLaMA 架构中去除了所有的 Dense 层偏置项，旨在简化计算并利用某些硬件优化。

## 常见考点
1.  **Pre-Norm 与 Post-Norm 的区别**：为什么 Pre-Norm 有助于训练深层网络？（梯度不经过 Norm 流动，梯度更平稳）。
2.  **RoPE 的相对位置编码原理**：RoPE 如何体现相对位置？（通过复数域的乘法性质：$q_m^T k_n = (R_m q) \cdot (R_n k) \propto \text{relative position}(m-n)$）。
3.  **SwiGLU 的参数量计算**：相比标准 FFN (4h -> 4h)，SwiGLU 的参数量增加了多少？（标准是 $W_1(4h\times h), W_2(h \times 4h)$，SwiGLU 是 $W(4h\times h), V(4h\times h), W_2(h\times 4h)$，参数量约 $1.5\times$，虽然计算量大但效果好）。
