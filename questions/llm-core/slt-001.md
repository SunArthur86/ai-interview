# LoRA的原理是什么？为什么参数高效？

LoRA（Low-Rank Adaptation）冻结预训练权重W，额外引入低秩矩阵A和B来学习增量。

公式：W' = W + ΔW = W + B·A
其中A∈ℝ^(d×r)，B∈ℝ^(r×d)，r远小于d（如r=8）。

参数量对比：
- 全量微调：d×d参数（如4096×4096 = 16.7M）
- LoRA r=8：2×d×r = 2×4096×8 = 65K（减少256倍）

初始化：A用高斯随机初始化，B初始化为零。训练开始时ΔW=0，不改变模型输出。

为什么有效？
1. 大模型权重更新本质是低秩的（Aghajanyan等研究证明）
2. 预训练模型已有丰富知识，微调只需少量参数调整
3. 推理时可以将B·A合并回W，不增加推理延迟

QLoRA：先用NF4量化基座模型（减少显存），再用LoRA微调。使65B模型在单张48GB GPU上可微调。
