# ALiBi (Attention with Linear Biases)思路是什么

### 1. ALiBi 是什么
**ALiBi (Attention with Linear Biases)** 是一种不使用嵌入向量的位置编码方法。它通过在 Attention 的计算分数（Logits）上直接减去一个与距离成正比的惩罚项来实现位置感知。

### 2. 核心思想
*   **无需位置 Embedding：** 去掉了输入层或注意力层的绝对/相对位置向量加法操作。
*   **线性偏置惩罚：** 在计算 $QK^T$ 得到分数矩阵后，对每个位置对 $(i, j)$ 施加一个偏置 $b = m \cdot |i - j|$（其中 $i$ 是 query 位置，$j$ 是 key 位置）。
*   **距离衰减：** 距离越远，减去的值越大，导致 Softmax 后的权重越小。这意味着模型被鼓励更多地关注邻近的 Token。

### 3. 为什么具备长度外推能力
*   **基于相对距离：** 无论序列多长，第 $i$ 个 token 和第 $i+10$ 个 token 的相对距离始终是 10，惩罚值固定。模型不需要学习“位置 5000”是什么样的，只需要学习“距离 10”的影响。
*   **无长度限制：** 理论上可以外推到任意长度，只需继续延长距离计算即可。

### 4. 优势
*   **训练与推理一致性：** 短序列训练，长序列推理，无需修改模型结构。
*   **实现简单：** 仅需修改 Attention Score 的计算逻辑几行代码。
