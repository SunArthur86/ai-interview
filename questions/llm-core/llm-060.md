# 手撕Transformer中FFN代码

Transformer中的前馈神经网络(FFN)由两个线性层和一个非线性激活函数组成，用于对每个位置的特征进行非线性变换。公式：FFN(x) = max(0, xW1 + b1)W2 + b2。

Python代码实现（包含 SwiGLU 变体，这在现代 LLM 如 Qwen/Llama 中更常用）：

```python
import torch
import torch.nn as nn

class FeedForwardNetwork(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim, use_swiglu=False):
        super(FeedForwardNetwork, self).__init__()
        self.use_swiglu = use_swiglu
        
        if not self.use_swiglu:
            # 标准 Transformer (ReLU/GELU)
            self.linear1 = nn.Linear(input_dim, hidden_dim)
            self.relu = nn.ReLU() # 或 nn.GELU()
            self.linear2 = nn.Linear(hidden_dim, output_dim)
        else:
            # SwiGLU (Llama/Qwen 风格)
            # SwiGLU 需要三个矩阵，或者将 hidden_dim 视为门控和值的合并维度
            # 实际上通常 dim_input -> 3 * hidden_dim, 然后切片
            self.gate_proj = nn.Linear(input_dim, hidden_dim, bias=False)
            self.up_proj = nn.Linear(input_dim, hidden_dim, bias=False)
            self.down_proj = nn.Linear(hidden_dim, output_dim, bias=False)
            self.act = nn.SiLU() # Swish = SiLU

    def forward(self, x):
        if not self.use_swiglu:
            x = self.linear1(x)
            x = self.relu(x)
            x = self.linear2(x)
        else:
            # SwiGLU: Swish(xW_g) * (xW_u)
            gate = self.act(self.gate_proj(x))
            up = self.up_proj(x)
            x = self.down_proj(gate * up)
        return x
```

**## 常见考点**
*   **维度变化**：FFN 中隐藏层维度通常是输入维度的 4 倍，这个比例是如何确定的？
*   **SwiGLU 参数量**：相比 ReLU 版本，SwiGLU 增加了多少参数量？为什么 SwiGLU 效果更好？（门控机制增加了非线性表达能力）
*   **逐位置特性**：FFN 是对整个序列做运算还是对每个 Token 独立做运算？这如何体现其作用？
