# 以跳过sft阶段直接进行RLHF么

不建议跳过 SFT。

1. **模仿学习的作用**：SFT 类似于 RL 中的模仿学习。在 AlphaGo 和自动驾驶等复杂环境任务中，通常先通过模仿学习（监督学习）缩小搜索空间，再进行强化学习（RL）。
2. **搜索空间与成本**：LLM 的生成空间极其庞大。若直接进行 RL，搜索空间过大，导致训练极不稳定且资源消耗巨大。
3. **Reward Model 的局限**：AlphaGo Zero 的成功依赖于客观的围棋规则作为 Reward，且不计成本搜索。而在 LLM 场景下，Reward Model 需要人工标注构建，成本高且难以做到完全客观和精确。

因此，先用 SFT 让模型学会基本的对话和指令遵循能力，缩小搜索空间，再利用 RLHF 进行价值观对齐是必要且高效的路径。
