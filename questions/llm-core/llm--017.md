# 什么是模型微调(Fine-tuning)?为什么不直接用预训练模型

- **微调 (Fine-tuning)** 是在预训练模型基础上,用特定领域数据继续训练,使其适应下游任务.

- **为什么不直接用预训练模型?**
1. 预训练模型是'通才',缺乏领域深度
2. 不会遵循指令格式(如输出 JSON)
3. 回复风格不符合业务需求
4. 缺乏私有知识

- **微调方式**
- **全量微调**:更新所有参数(效果好,成本高)
- **LoRA**:只训练低秩适配器(成本低,效果接近)
- **QLoRA**:4bit 量化 + LoRA(极致省显存)

- **一句话理解**:预训练模型 = 大学毕业生;微调 = 入职培训.
