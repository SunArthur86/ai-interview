# 预训练数据处理有哪些关键步骤？

高质量预训练数据是大模型能力的基础。

关键步骤：

1. 数据收集：网页爬取（Common Crawl）、书籍（Books3）、代码（GitHub）、论文（arXiv）

2. 数据清洗：
- 去重（MinHash LSH）
- 去除低质量内容（困惑度过滤、启发式规则）
- 去除有害内容（安全过滤）
- 语言识别和分类

3. 数据配比：
- 网页文本50-60%，代码10-20%，书籍10-15%，学术论文5%
- 代码数据对推理能力提升显著

4. Tokenization：BPE/BBPE编码

5. 课程学习：
- 先学简单/通用数据，后学复杂/专业数据
- 逐步提升数据质量

数据量级：Llama 3训练了15T tokens。数据质量比数量更重要。
