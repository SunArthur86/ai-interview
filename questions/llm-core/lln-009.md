# 为什么BERT在第一句前会加一个[CLS]标志? BERT输出的两个函数是什么

BERT在第一句前会加一个[CLS]标志,最后一层该位置对应的向量可以作为整句话的语义表示,从而用于下游的分类任务等.与文本中已有的其它词相比,这个无明显语义信息的符号会更“公平”地融合文本中各个词的语义信息,从而更好的表示整句话的语义.具体来说,self-attention是用文本中的其它词来增强目标词的语义表示,但是目标词本身的语义还是会占主要部分的,因此,经过BERT的12层,每次词的embedding融合了所有词的信息,可以去更好的表示自己的语义.而[CLS]位本身没有语义,经过12层,得到的是attention后所有词的加权平均,相比其他正常词,可以更好的表征句子语义.当然,也可以通过对最后一层所有词的embedding做pooling表征句子语义。

在 BERT 源码及实际应用中，两个主要的输出函数（或层逻辑）如下：

1. **`get_pooled_output()` (Sentence-level Output)**
   - **来源**：取序列第一个 Token（即 [CLS]）在最后一层的 Hidden State。
   - **操作**：通常接一个全连接层进行线性变换（有时还接 Tanh 激活函数），矩阵维度为 `[hidden_size] x [hidden_size]`，用于适配下游分类任务的输入空间。
   - **输出 Shape**：`[batch_size, hidden_size]`。主要用于文本分类、语义匹配（ entailment ）等句子级任务。

2. **`get_sequence_output()` (Token-level Output)**
   - **来源**：取最后一层 Transformer 所有 Token 的 Hidden State。
   - **操作**：直接输出或进行简单的轻量变换。它保留了每个 Token 的上下文语义信息。
   - **输出 Shape**：`[batch_size, seq_length, hidden_size]`。该输出包含了 [CLS] 以及句子中所有词（包括 [PAD] 等）的向量。主要用于序列标注（如 NER、词性标注）、SQuAD 问答等 Token 级或 Span 级任务。

```text

输入: [CLS] 我 爱 AI [SEP] [PAD]
  │
  ▼
┌─────────────────────────────────────┐
│         BERT Encoder (L=12)         │
│  (Self-Attention: Full Attention)   │
└─────────────────────────────────────┘
  │                    │
  ▼                    ▼
┌─────────┐    ┌───────────────────────┐
│ [CLS]   │    │   我, 爱, AI, [SEP]    │
│ Vector  │    │   (All Tokens)        │
│ (1x768) │    │   (Seq_Len x 768)     │
└────┬────┘    └───────────┬───────────┘
     │                    │
     ▼                    ▼
  Tanh/Dense           (Raw Output)
     │                    │
     ▼                    ▼
 get_pooled_out()   get_sequence_out()
 [batch, hidden]   [batch, seq, hidden]
     │                    │
     ▼ (用于分类)           ▼ (用于标注/QA)
  Softmax           CRF / Span Predictor
```

## 常见考点
1. **[CLS] 融合信息的局限性**：虽然[CLS]理论上是全局信息的聚合，但实际研究表明，在深层网络中，[CLS] 与特定 Token 的 Attention 相关性可能并不总是最高的。为什么有些任务（如长文本分类）用 [CLS] 效果不如 Mean Pooling？
2. **Next Sentence Prediction (NSP) 的作用**：[CLS] 除了用于分类，原版 BERT 中还用于 NSP 任务。但在 RoBERTa 等后续模型中移除了 NSP，这是否影响了 [CLS] 的语义表征能力？
3. **不同 Pooling 策略**：除了取 [CLS] 向量，还有哪些获取句向量的方法？（如 Mean-Pooling, Max-Pooling, SBERT 的推导）。
