# 手撕RMSNorm

📌RMSNorm(Root Mean Square Normalization)是LayerNorm的一种变体,它的不同点在
于不计算均值,而是直接使用输入的范数(即均方根,Root Mean Square)来进行标准化.
这种方法减少了计算开销,同时也能提供类似的归一化效果.
1. RMSNorm的数学公式
与 LayerNorm 不同,RMSNorm 只计算输入向量的均方根,公式如下:

= x^i

x^i 其中 是一个小的常数,避免除零错误. ϵ RMSNorm不包含偏移项 ,只引入了可学习的缩放参数

2. 手撕RMSNorm代码
import torch
import torch.nn as nn
class RMSNorm(nn.Module):
def __init__(self, normalized_shape, eps=1e-5, elementwise_affine=True):
super(RMSNorm, self).__init__()
self.eps = eps
self.elementwise_affine = elementwise_affine
# 如果需要可学习参数,则初始化 gamma
if self.elementwise_affine:
self.gamma = nn.Parameter(torch.ones(normalized_shape))
**else：**
self.gamma = None
def forward(self, x):
# 计算均方根(RMS)

rms = torch.sqrt(torch.mean(x ** 2, dim=-1, keepdim=True) + self.eps)
# 进行标准化
x_normalized = x / rms
# 如果有 gamma,则进行缩放
if self.elementwise_affine:
x_normalized = self.gamma * x_normalized
return x_normalized

3. 逐行解释
normalized_shape : 要标准化的维度大小.
elementwise_affine : 控制是否引入可学习的缩放参数(默认值为True).
如果 elementwise_affine 为 True ,初始化可学习的参数 gamma ,用于缩放.
计算均方根(RMS):通过 torch.sqrt(torch.mean(x ** 2, dim=-1,
keepdim=True) + self.eps) 计算输入的均方根,x ** 2 表示对输入张量的每个元
素平方,torch.mean 则计算均值,torch.sqrt 取平方根.我们加上一个小的 eps 来
**标准化：** 将输入 x 的每个元素除以均方根 rms ,实现标准化.
**缩放：** 如果 elementwise_affine 为 True ,将标准化后的结果乘以 gamma 进行缩
3. 返回结果:经过标准化并且可能缩放后的张量被返回.
📚RMSNorm与LayerNorm的主要区别,即不再计算均值,只计算均方根,从而降低了计算开
销,同时保持标准化效果.
