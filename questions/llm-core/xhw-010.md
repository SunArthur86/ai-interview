# Transformer基本结构和原理

Transformer 是一种完全基于注意力机制的神经网络，主要分为 Encoder 和 Decoder 两部分。

**1. 核心组件**
- **自注意力机制**：通过 Query、Key、Value 计算序列中词与词的相关性，捕捉长距离依赖。
- **前馈神经网络 (FFN)**：对每个位置的向量进行非线性变换。
- **位置编码**：由于没有循环结构，注入位置信息以保留序列顺序。
- **残差连接与层归一化**：稳定深层网络的训练。

**2. 结构分工**
- **Encoder**：由多层自注意力+FFN 组成，输入序列并行处理，提取深层语义特征。
- **Decoder**：包含掩码自注意力（防止看到未来词）、Encoder-Decoder Attention（关注输入）、FFN。用于自回归生成。

**3. 优势**
- 并行计算能力强，训练效率高。
- 能够有效捕捉长距离依赖。

**4. 关键细节与原理补充**
- **多头注意力**：
    - 将 $d_{model}$ 维度的向量切分为 $h$ 个头，每个头在不同子空间计算注意力。例如 $d_{model}=512, h=8$，每个头维度为 64。
    - 公式：$MultiHead(Q,K,V) = Concat(head_1, ..., head_h)W^O$，其中 $head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)$。
    - 作用：让模型能够同时关注不同位置的不同子空间信息（如一个头关注语法结构，另一个头关注语义关联）。
- **Layer Norm（层归一化）位置**：
    - **Pre-Norm**（如 GPT-2, LLaMA）：Norm 在子层输入之前。训练更稳定，适合深层网络，利于梯度回传。
    - **Post-Norm**（原始 Transformer）：Norm 在子层输出+残差之后。需要 Warmup。
- **Positional Encoding（位置编码）**：
    - **Sinusoidal（正弦/余弦）**：原始论文使用，利用三角函数的相对位置性质，具有外推性（可处理比训练时更长的序列）。
    - **Learned（可学习）**：如 BERT，直接学习位置向量，但外推性差。
    - **RoPE（旋转位置编码）**：目前主流（LLaMA），通过旋转矩阵将位置信息注入 Query 和 Key，具有更好的相对位置感知和外推性。

## 常见考点
1. **Self-Attention 计算复杂度**：为何是 $O(N^2)$，以及 FlashAttention 等优化手段如何通过 IO 感知来加速。
2. **残差连接的作用**：解决深层网络梯度消失问题，允许模型堆叠更多层（如 100+ 层）。
3. **Pre-Norm vs Post-Norm**：为何现在的 LLM（如 LLaMA） 都使用 Pre-Norm 结构。
