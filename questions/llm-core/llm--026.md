# 预训练数据的质量如何影响模型能力?高质量数据管道的关键步骤有哪些

- **数据质量 = 模型质量**(Data is all you need)

Llama 3 的经验:用 15T tokens 高质量数据训练 70B 模型,效果超过用 2T tokens 训练的更大模型.

- **高质量数据管道的关键步骤**

1. **数据采集**
- 网页(Common Crawl)→ 需重度清洗
- 书籍/论文 → 高质量
- 代码(GitHub)→ 推理能力关键
- 对话数据 → 指令跟随

2. **去重 (Deduplication)**
- URL 去重:同一页面只保留一次
- 文档级去重:MinHash/SimHash 近似去重
- 行级去重(Llama 3 创新):逐行去重,去除模板化内容
- 经验:去重可去除 30-50% 的数据

3. **质量过滤**
- 启发式:去除过短/过长/重复/乱码文本
- 模型过滤:用小模型给文本打质量分
- 安全过滤:去除 PII、有害内容
- 语言识别:过滤非目标语言

4. **数据配比 (Data Mixing)**
- Llama 3 配比:~50% 网页 + ~25% 代码 + ~17% 数学/推理 + ~8% 多语言
- 代码和数学数据对推理能力至关重要
- 配比需要消融实验确定

5. **课程学习 (Curriculum)**
- 先训简单/通用数据,再加难/专业数据
- Llama 3 三阶段:初始预训练 → 长文本预训练 → 退火训练

6. **合成数据**
- 用强模型生成高质量训练数据
- DeepSeek-R1 的蒸馏数据显著提升小模型推理
- 风险:模型坍缩(反复用自身输出训练会退化)
