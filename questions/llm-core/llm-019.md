2. 多头注意力的目的与优势:
捕捉不同的特征信息(Representation Diversity):单个注意力头只能聚焦于输入序列中的某些 特定关系,而通过多个头,模型能够并行地从不同的“视角”进行注意力计算.这些视角可能代表不 同的上下文关系、语义层次或距离信息.换句话说,多头机制允许模型在同一层次上学到多种不同的 依赖关系,提高了模型对复杂依赖的表达能力. 增强模型的学习能力(Better Feature Learning):每个注意力头会使用不同的投影矩阵 、

、 ,这意味着每个头会生成不同的查询、键和值向量.这种多样化的投影可以帮助模型在更 高维度上捕捉更多的信息特征,进而提升模型的表达能力. W

3. 并行处理提升效率(Parallelization for Efficiency):
多头注意力机制允许不同的注意力头进行并行计算,这相较于传统的序列模型(如 RNNs)显著提高了 训练和推理的效率.在多头机制中,模型能够在同一时间捕捉到多个依赖关系,而不是逐步处理,从 而大大加速了计算.
4. 避免信息瓶颈(Mitigating the Bottleneck of Single Attention Head):
使用单个注意力头可能会导致信息过于集中,模型在学习复杂上下文时难以捕捉到多维度的依赖关 系.多头注意力通过引入多个注意力头,降低了单一头在学习复杂关系时可能遇到的信息瓶颈.
5. 平滑注意力分布(Smoother Attention Distribution):
在实际应用中,单个注意力头的权重分布可能过于集中,导致模型对某些特定词语的过度依赖.而多 头注意力通过多个头的组合,能够使注意力分布更加均衡和平滑,从而提高模型对不同上下文的整体 理解能力.
6. 总结:
📌Transformer 使用多头注意力机制是为了提升模型捕捉复杂依赖关系的能力,增强表达能 力,避免信息瓶颈,并且通过并行计算提高模型的计算效率和注意力分布的多样性.
