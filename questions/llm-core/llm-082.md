# 不同Attention之间的区别是什么

88. 1 各类 Attention 的关系与比较
🥥Self-Attention vs Cross-Attention **Self-Attention：** 同一序列内各位置彼此建模 **Cross-Attention：** 不同序列之间信息交互 都是最基础的关注“相关性”的手段 Sparse / Linear Attention vs 普通 Attention **目的相同：** 处理超长序列时减少 O(n²) 负担 **Sparse：** 通过设计“稀疏模式”跳过大部分计算 **Linear：** 通过核技巧或矩阵分解把二次复杂度转为线性 Multi-head Attention vs Multi-query / Grouped-query Attention **Multi-head：** 每个 head 都有独立 Q/K/V,表达力强,但开销大 **Multi-query：** 只留一套(或少数)K/V,大幅降低显存占用 **Grouped-query：** 分组共享 K/V,兼顾一定的表达力与效率 Multi-head Latent Attention 在多头的基础上引入“隐变量/全局潜在表示” 进一步增强模型的全局表达能力
