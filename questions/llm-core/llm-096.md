# DeepSeek-R1-Zero里的Zero含义

**一句话回答：** Zero来自Alpha Zero,特指无需人类数据,即可从零实现模型自我迭代.
🥥在人工智能领域,DeepSeek-R1-Zero模型的“Zero”一词源自AlphaZero,意指模型无需人
类数据即可从零开始自我迭代.DeepSeek-R1-Zero通过纯强化学习(Reinforcement

Learning,RL)训练,完全跳过了传统的监督微调(Supervised Fine-Tuning,SFT)阶
段.这种方法使模型能够在没有人工标注数据的情况下,通过与环境的交互自我进化,展现
出强大的推理能力.
🍁在训练过程中,DeepSeek-R1-Zero采用了基于规则的奖励系统,主要包括两种类型的奖励:
1. 准确率奖励:评估模型的回答是否正确.例如,在数学问题中,要求模型按指定格式给出
最终答案,以便基于规则可靠地验证正确性.
2. 格式奖励:促使模型将思考过程置于特定标签之间,以规范化输出格式.
通过这种奖励机制,模型能够在没有人工标注数据的情况下,通过强化学习自我进化,展现
出强大的推理能力.
然而,DeepSeek-R1-Zero在可读性和语言一致性方面存在挑战.为了解决这些问题,研究团
队开发了DeepSeek-R1模型.该模型首先通过少量冷启动数据对基础模型进行微调,然后采
用与R1-Zero相同的强化学习过程,并引入语言一致性奖励,以提高模型的可读性和推理能
力.
DeepSeek-R1-Zero的“Zero”含义强调了模型在没有人类数据的情况下,通过纯强化学习从零开始
自我迭代,展现出强大的推理能力.这一方法突破了传统的监督学习依赖,展示了强化学习在大语言
模型训练中的巨大潜力.
