# 讲讲工业界RAG具体如何优化

工业界RAG优化关注内容供给和归纳总结能力。检索优化包括Query2Doc、HyDE、PROMPTAGATOR、Step-back-prompt；生成优化涉及重排、矛盾解决、噪声识别、递归摘要、压缩、重排序、softmax改进；微调优化包括RAFT、AAR、UPRISE、PRCA。
