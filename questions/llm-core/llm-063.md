# 手撕flash-attention

FlashAttention是一种通过分块计算和在线更新优化注意力机制的算法，减少内存访问开销并加速计算，特别适合长序列任务。核心步骤包括：1）将Q、K、V分块处理；2）在线计算softmax（逐步更新最大值和指数和）；3）逐块累加输出。
