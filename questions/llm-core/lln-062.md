# 什么是 DyT(Dynamic Tanh)?它为什么能替代 LayerNorm

### DyT (Dynamic Tanh)

**DyT** 是一种归一化替代方案，旨在解决 LayerNorm 在某些场景下的局限性。

### 核心思想
用一个可学习的 tanh 激活函数替代 LayerNorm。

公式：$x_{\text{out}} = \gamma \cdot \tanh(\alpha \cdot x) + \beta$

其中 $\alpha, \gamma, \beta$ 均为可学习参数。

### 为什么能替代 LayerNorm
1. **约束范围**：LayerNorm 的核心作用是约束激活值范围，而 tanh 天然有 $(-1, 1)$ 的挤压效果。
2. **计算更简单**：不需要计算均值和方差，计算量更小。
3. **不依赖 Batch**：不依赖 batch 内的统计信息，训练更稳定。
4. **实验效果**：在 Transformer、ViT 等模型上效果相当甚至更好。

### 优势
- 计算更简单（无均值/方差计算）。
- 推理更快。
- 对 batch size 不敏感。
- 跨模态通用性好。
