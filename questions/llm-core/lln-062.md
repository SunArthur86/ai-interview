# 什么是 DyT(Dynamic Tanh)?它为什么能替代 LayerNorm

- **DyT(Dynamic Tanh)** 是 Meta 于 2025 年提出的归一化替代方案.

- **核心思想:** 用一个可学习的 tanh 激活函数替代 LayerNorm,公式为 x_out = gamma * tanh(alpha * x) + beta,其中 alpha、gamma、beta 均为可学习参数.

- **为什么能替代 LayerNorm:**
1. LayerNorm 的核心作用是约束激活值范围,而 tanh 天然有挤压效果
2. DyT 不需要计算均值和方差,计算量更小
3. 实验表明在 Transformer、ViT 等模型上效果相当甚至更好
4. 训练更稳定,且不依赖 batch 内的统计信息

- **优势:** 计算更简单(无均值/方差计算)、推理更快、对 batch size 不敏感、跨模态通用性好.
