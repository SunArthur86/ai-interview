# 大模型中常用的分词算法有哪些?BPE和BBPE的区别是什么

分词(Tokenization)是将文本切分为模型可处理的最小单元(token)的过程.

- **三种分词粒度:**
| 粒度 | 优点 | 缺点 |
|------|------|------|
| Word(词级)| 保留语义边界 | 词表大、OOV问题 |
| Char(字符级)| 无OOV | 语义弱、序列长 |
| Subword(子词)| 平衡词表与语义 | 主流选择 |

- **主流算法对比:**
1. BPE (Byte Pair Encoding) - 从字符级开始,迭代合并最高频的相邻字符对
2. BBPE (Byte-level BPE) - 在字节级别做BPE,可处理任意字符(中文、emoji)
3. WordPiece - 类似BPE,合并准则基于似然最大化(BERT使用)
4. Unigram LM - 从大词表开始,逐步删除对总似然贡献最小的token

- **BPE vs BBPE区别:**
- BPE在Unicode字符上操作,BBPE在UTF-8字节上操作
- BBPE词表覆盖所有字节组合,天然支持多语言
- GPT-2/GPT-3使用BBPE,LLaMA使用SentencePiece+BPE
