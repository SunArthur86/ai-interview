# 强化学习算法是怎么分类的

强化学习算法主要分为基于价值、基于策略和Actor-Critic三类。基于价值的方法计算每个状态动作的价值并选取最大值，适用于离散动作空间；基于策略的方法直接学习动作的概率分布，适用于连续动作或随机策略；Actor-Critic结合两者，Actor负责策略，Critic负责评估。此外，还可按是否需要环境模型分为Model-free和Model-based。
