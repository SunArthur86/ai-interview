# 解释一下Adapter

✍大模型中的Adapter是一种轻量级的模型微调技术,旨在在大规模预训练模型的基础上,通过
插入少量可训练的参数来实现特定任务的微调,而无需重新训练整个模型.Adapter方法有效
地降低了微调大模型的计算和存储成本,同时保留了预训练模型的能力和知识.
1. 背景

大模型(如BERT、GPT等)经过大量无监督数据的预训练,拥有广泛的语言理解或生成能力.在下游
任务中,通常需要对这些预训练模型进行微调(fine-tuning),以适应特定任务(如情感分类、机器
翻译等).传统的微调方式需要更新模型的全部参数,然而这些模型通常有数十亿甚至数千亿参数,
导致微调过程的存储和计算开销巨大.此外,模型参数往往只能在特定任务之间共享,导致每个任务
的微调模型都需要单独存储一份.
为了缓解这些问题,Adapter技术应运而生,通过向模型中插入小规模的可训练参数模块,只微调这
些模块,而冻结原有的大量预训练参数.

3. Adapter的优势
参数效率高 Adapter只微调少量新引入的参 数,避免了对整个大模型的微 调.比如,BERT-base有1.1亿 参数,而典型的Adapter模块可 能只有几百万参数,微调时显 著减少了需要调整的参数量. 任务间参数共享 Adapter使得不同任务的参数模 块可以独立微调,但基础的预 训练模型参数仍然保持共享. 这种方法允许多个任务共享相 同的大模型,只需存储各个任 务的Adapter参数. 训练稳定性 由于Adapter只更新小部分参 数,相比全参数微调,训练过 程更为稳定,并且对较小的数 据集也更加友好,避免了过拟 合
4. Adapter的典型架构与变种

1. Houlsby Adapter:这是最早的Adapter之一,设计在Transformer的每一层注意力块和前馈网络
之间插入Adapter模块.
2. Parallel Adapter:不同于将Adapter插入到主网络中,平行结构的Adapter与Transformer层并行
工作,类似残差结构,可以在主干网络和Adapter之间进行特征融合.
3. Compacter:这是一种在减少参数量的基础上进一步优化的Adapter变体,使用了低秩参数分解技
术来减少Adapter中的参数量.
4. LoRA (Low-Rank Adaptation):与Adapter类似的思想,但LoRA通过在注意力权重矩阵上进行低
秩近似来减少训练参数.LoRA也是一种插入性微调技术,主要在Transformer的自注意力模块中插
入低秩矩阵更新.
5. 应用场景
Adapter在多个领域中都有广泛应用: •
自然语言处理(NLP):任务包括情感分析、命名实体识别、机器翻译等.
•
**跨语言模型微调：** Adapter被证明可以适应多语言环境中的任务,在不同语言之间共享大模型,微
调不同语言任务的Adapter. •
**多任务学习：** Adapter能够有效实现同一个大模型在多个任务之间的共享,避免了每个任务都需要
单独训练一个完整的模型.
6. 总结
🌟Adapter通过引入少量额外参数以微调预训练大模型,极大降低了微调的计算和存储成本.
它为大模型的部署和任务扩展提供了一种高效、灵活的解决方案.随着模型规模的不断增
长,Adapter技术为大规模模型的微调和部署提供了重要支持.
