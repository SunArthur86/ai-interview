# DeepSeek V3 的 DualPipe 算法解决了什么问题?和传统流水线并行有什么区别

- **DualPipe 核心思想:**

传统流水线并行(Pipeline Parallelism)存在「气泡」(bubble),即部分GPU空闲等待.

- **DualPipe 创新点:**
1. **双向流水线** - 前向和后向同时进行,从两端向中间收敛
2. **计算-通信重叠** - 在计算的同时做跨节点通信
3. **极小化气泡** - 几乎消除GPU空闲时间

- **对比:**

| 方法 | 气泡大小 | 通信开销 |
|------|---------|---------|
| GPipe | 大 | 中 |
| 1F1B | 中 | 中 |
| Interleaved 1F1B | 小 | 中 |
| **DualPipe** | **极小** | **低(重叠)** |

- **效果:** DeepSeek V3 训练效率提升显著,2048张H800 GPU上达到约 1800 TFLOPS.
