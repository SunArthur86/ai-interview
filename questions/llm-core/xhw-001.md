# Transformer前馈神经网络用什么激活函数

Transformer中的每个编码器和解码器层除了自注意力机制外，还包含一个两层的前馈神经网络
（Feed-Forward Neural Network, FFN），该网络应用在每个位置的向量上，是位置无关的。这个
前馈网络通常有以下结构：
- 两层全连接网络：第一层将输入的向量维度扩展到一个更高的维度，第二层将其缩小回原维度。
- 非线性激活函数：在两层之间插入一个非线性激活函数来引入非线性能力。

2.  激活函数选择：ReLU
在原始的Transformer模型中，前馈神经网络中使用的激活函数是ReLU（Rectified Linear Unit），
即： 
- ReLU的优势：
◦
非线性能力：通过ReLU引入非线性，使得模型可以捕捉更复杂的特征。
◦
计算效率高：ReLU的计算非常简单，只需要取输入和零的最大值，不需要像Sigmoid或Tanh那
样计算复杂的指数函数，因此它适合深层神经网络的训练。

◦
缓解梯度消失问题：相比Sigmoid或Tanh，ReLU在正区间的梯度为常数，避免了梯度消失的问
题。
3. ReLU的使用目的
ReLU在Transformer的前馈网络中被广泛应用，原因在于它能够在两个全连接层之间引入非线性变
换，帮助模型学习更复杂的特征表示。它的计算效率高且性能表现良好，尤其适合大型模型。
4. 其他激活函数的变体
尽管ReLU是原始Transformer模型中的默认激活函数，后续的模型也探索了其他激活函数，如：
GELU（Gaussian Error Linear Unit）：在某些变体中，GELU也被用作替代激活函数，尤其是在
更深层的模型中（如BERT）。GELU在零点附近平滑过渡，被认为在某些情况下能够带来更好的性
能。

GELU 函数的直观解释是：它结合了线性和非线性特性，当输入值较小的时候接近于0，当输入值较
大的时候接近于1，这使得它在处理不同范围的输入时具有不同的激活特性。这种特性使得 GELU 在
某些情况下比传统的ReLU激活函数表现得更好。
5.  总结
- 原始Transformer前馈神经网络中的激活函数是ReLU，因为它能提供简单高效的非线性能力，适
合深度学习。
- ReLU的特点：计算简单、高效，能够缓解梯度消失问题。
- 其他变体中的激活函数：有些后续模型（如BERT）使用了GELU作为替代激活函数，以实现更好的
性能。
