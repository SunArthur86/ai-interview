# PPO的原理

PPO(Proximal Policy Optimization,近端策略优化)是一种常用的强化学习算法,属于策略优化 方法.其核心思想是通过限制策略更新的幅度来提升训练的稳定性,避免策略在每次更新中发生较大 变化,从而防止训练过程中的不稳定和策略崩溃.
1. PPO的核心思想
PPO通过一种剪辑损失函数(Clipped Surrogate Objective)来实现策略的优化,它的目标是确保策 略更新时不会偏离当前策略太多.它克服了传统策略梯度方法(如TRPO,Trust Region Policy Optimization)复杂的约束问题,保持了性能的同时降低了实现难度. 传统的策略优化目标是直接优化策略 ,使得给定状态下的行为能够最大化累计奖励.PPO在此 基础上,通过引入一个重要的比例项 "probability ratio"

4. PPO的优缺点
**优点**
优点 **更新稳定：** 通过剪辑比率或KL散度约束,PPO 限制了每次更新的幅度,防止策略发生剧烈变 化,从而提高了算法的稳定性.
**缺点**
**缺点**
缺点 样本效率较低 PPO属于同策略算法,每次更新只能使用当前策 略采集的数据,导致样本效率较低.

**易于实现：** 与TRPO相比,PPO无需复杂的约束 优化算法,易于实现和调整超参数. **泛用性强：** PPO适用于离散和连续动作空间, 广泛应用于各种强化学习任务. 超参数敏感性 PPO对剪辑参数 和 GAE 中的 等超参数敏 感,需要在实际应用中进行细致的调整. ϵ λ
5. 总结
PPO通过引入剪辑或KL散度约束,在确保策略稳定更新的同时,简化了实现难度.其在连续控制任 务、机器人控制、复杂游戏等场景中表现出色,成为强化学习领域应用最为广泛的策略优化算法之 一.
