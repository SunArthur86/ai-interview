# 什么是 4D 并行训练?数据/张量/流水线/Expert 并行如何组合

- **4D 并行 = Data + Tensor + Pipeline + Expert(序列)并行**

| 维度 | 切分对象 | 通信类型 | 适合 |
|------|---------|---------|------|
| **数据并行(DP)** | Batch | AllReduce梯度 | 通用 |
| **张量并行(TP)** | 权重矩阵 | AllReduce激活 | 大模型 |
| **流水线并行(PP)** | 模型层 | 点对点激活 | 超大模型 |
| **序列并行(SP)** | 序列长度 | AllGather | 长上下文 |

- **组合策略:**
```
GPU总数 = DP × TP × PP × SP

例: 2048 GPU
DP=32, TP=8, PP=8 → 32×8×8=2048
```

- **Llama 3.1 的4D并行:**
- DP: 跨节点数据并行
- TP: 节点内8卡张量并行(NVLink高速互联)
- PP: 跨节点流水线并行
- SP: 序列并行处理长上下文

- **DeepSpeed ZeRO:**
- ZeRO-1: 优化器状态分片
- ZeRO-2: +梯度分片
- ZeRO-3: +模型参数分片(类似DP但更高效)
- 可与TP/PP叠加
