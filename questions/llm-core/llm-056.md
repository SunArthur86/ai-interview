# 手撕自注意力机制(不用Pytorch)(Self Attention)

自注意力机制是 Transformer 的核心，其数学公式为：

$$ \text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V $$

其中：
*   $Q$ (Query): 查询向量，表示当前关注点。
*   $K$ (Key): 键向量，用于被匹配。
*   $V$ (Value): 值向量，表示实际内容信息。
*   $d_k$: 缩放因子，用于防止点积过大导致梯度消失。

### 手撕代码实现

面试中要求“手撕”通常指不依赖 `nn.Linear` 等高层 API，使用基础张量运算实现。

```python
import torch
import torch.nn.functional as F

class SelfAttentionScratch:
    def __init__(self, dim_input, dim_k, dim_v):
        self.dim_input = dim_input
        self.dim_k = dim_k
        self.dim_v = dim_v
        # 手动初始化权重矩阵
        self.W_q = torch.randn(dim_input, dim_k) / dim_k**0.5
        self.W_k = torch.randn(dim_input, dim_k) / dim_k**0.5
        self.W_v = torch.randn(dim_input, dim_v) / dim_v**0.5
        
    def forward(self, x):
        # x shape: (batch_size, seq_len, dim_input)
        batch_size, seq_len, _ = x.shape
        
        # 1. 线性变换得到 Q, K, V
        Q = torch.matmul(x, self.W_q)  # (batch, seq_len, dim_k)
        K = torch.matmul(x, self.W_k)  # (batch, seq_len, dim_k)
        V = torch.matmul(x, self.W_v)  # (batch, seq_len, dim_v)
        
        # 2. 计算注意力得分
        scores = torch.matmul(Q, K.transpose(1, 2))  # (batch, seq_len, seq_len)
        scores = scores / torch.sqrt(torch.tensor(self.dim_k, dtype=torch.float32))
        
        # 3. Softmax 归一化
        attention_weights = F.softmax(scores, dim=-1)
        
        # 4. 加权求和
        output = torch.matmul(attention_weights, V)  # (batch, seq_len, dim_v)
        
        return output

# 测试
x = torch.randn(2, 5, 10) # batch=2, len=5, dim=10
attn = SelfAttentionScratch(dim_input=10, dim_k=8, dim_v=8)
out = attn.forward(x)
print(out.shape) # torch.Size([2, 5, 8])
```
