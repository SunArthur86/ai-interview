在面试中要求“手撕代码”,通常是指不依赖高层次的库函数(例如PyTorch中的nn.Linear 等),而是从头实现出自注意力机制的功能. import torch import torch.nn.functional as F class SelfAttentionScratch: def __init__(self, dim_q, dim_k, dim_v): self.dim_q = dim_q self.dim_k = dim_k self.dim_v = dim_v
# 手动初始化查询、键和值的权重矩阵
self.W_q = torch.randn(dim_q, dim_k) / dim_k**0.5 # 查询的线性变换矩阵 self.W_k = torch.randn(dim_q, dim_k) / dim_k**0.5 # 键的线性变换矩阵 self.W_v = torch.randn(dim_q, dim_v) / dim_v**0.5 # 值的线性变换矩阵 self._norm_fact = 1 / (dim_k ** 0.5) # 归一化因子 def forward(self, x): batch, n, dim_q = x.shape
# 计算查询、键和值矩阵
q = torch.matmul(x, self.W_q) # (batch, n, dim_k) k = torch.matmul(x, self.W_k) # (batch, n, dim_k) v = torch.matmul(x, self.W_v) # (batch, n, dim_v)
# 计算相似度(查询与键的点积)
scores = torch.matmul(q, k.transpose(1, 2)) * self._norm_fact # (batch, n, n)
# 对相似度进行softmax归一化,得到注意力权重
attention_weights = F.softmax(scores, dim=-1) # (batch, n, n)
# 根据注意力权重加权求和值
attention_output = torch.matmul(attention_weights, v) # (batch, n, dim_v) return attention_output
# 测试一下手撕实现

这里定义了一个 SelfAttentionScratch 类,用来实现自注意力机制.与基于 nn.Module 的 高层实现不同,它没有依赖现有的库函数,而是直接定义矩阵计算. __init__ 构造函数 def __init__(self, dim_q, dim_k, dim_v): self.dim_q = dim_q self.dim_k = dim_k self.dim_v = dim_v 1 2 3 4 这里的 __init__ 函数接收三个参数: • dim_q : 输入特征(Query)的维度. • dim_k : Key 和 Query 的映射维度. • dim_v : Value 的映射维度. 接下来,类内会基于这些维度参数初始化一些必要的权重矩阵. 初始化查询、键和值的权重矩阵 self.W_q = torch.randn(dim_q, dim_k) / dim_k**0.5 # 查询的线性变换矩阵 self.W_k = torch.randn(dim_q, dim_k) / dim_k**0.5 # 键的线性变换矩阵 self.W_v = torch.randn(dim_q, dim_v) / dim_v**0.5 # 值的线性变换矩阵 1 2 3

**在这里：**
• self.W_q , self.W_k , self.W_v 是查询(Q),键(K)和值(V)的权重矩阵,分别用 高斯分布随机初始化. • 初始化时除以 dim_k**0.5 和 dim_v**0.5 主要是为了防止数值过大引起梯度爆炸.特别是 在注意力机制中,常用这种缩放技术来稳定训练. W_q 大小为 (dim_q, dim_k) :将输入的 Query 映射到 dim_k 维度. W_k 大小为 (dim_q, dim_k) :将输入的 Key 映射到 dim_k 维度. W_v 大小为 (dim_q, dim_v) :将输入的 Value 映射到 dim_v 维度. 定义归一化因子 self._norm_fact = 1 / (dim_k ** 0.5) # 归一化因子 1

根据标准的注意力机制公式,Query 和 Key 的点积在计算之前需要除以 √(dim_k) ,这就 是归一化因子.目的是避免在点积操作中,值的绝对数值过大,从而导致梯度消失或爆炸问题. forward 函数 def forward(self, x): batch, n, dim_q = x.shape 1 2 • 这里的 forward 函数接收输入 x ,假设 x 是一个三维张量,形状为 (batch_size, sequence_length, dim_q) ,即批次大小、序列长度、和输入特征维度. • batch 是批次大小,n 是序列长度,dim_q 是输入的特征维度. 计算查询、键和值矩阵 q = torch.matmul(x, self.W_q) # (batch, n, dim_k) k = torch.matmul(x, self.W_k) # (batch, n, dim_k) v = torch.matmul(x, self.W_v) # (batch, n, dim_v) 1 2 3 • 查询矩阵 q :输入张量 x 与查询权重矩阵 self.W_q 相乘,结果是 (batch_size, sequence_length, dim_k) . • 键矩阵 k :输入张量 x 与键权重矩阵 self.W_k 相乘,结果是 (batch_size, sequence_length, dim_k) . • 值矩阵 v :输入张量 x 与值权重矩阵 self.W_v 相乘,结果是 (batch_size, sequence_length, dim_v) . 此时,q 和 k 可以用于相似性度量,而 v 会用于最终输出. 计算相似度分数 scores = torch.matmul(q, k.transpose(1, 2)) * self._norm_fact # (batch, n, n) 1 • 这里我们通过点积操作 torch.matmul(q, k.transpose(1, 2)) 计算 Query 和 Key 的相 似度. k.transpose(1, 2) 是对 k 进行转置,交换其序列长度维度和最后一个维度,方便与 q 做点积操作.最终 q 的形状是 (batch_size, sequence_length, dim_k) , k.transpose(1, 2) 的形状是 (batch_size, dim_k, sequence_length) ,二 者相乘得到 (batch_size, sequence_length, sequence_length) 的相似度矩 阵.

• 然后,结果乘以归一化因子 self._norm_fact ,用于缩放. 对相似度分数进行 softmax 归一化 attention_weights = F.softmax(scores, dim=-1) # (batch, n, n) 1 • 通过 F.softmax 函数将相似度矩阵转换为注意力权重.dim=-1 表示对最后一个维度(即 sequence_length 维度)进行 softmax,确保每个序列位置的注意力权重总和为 1. 根据注意力权重加权求和值 attention_output = torch.matmul(attention_weights, v) # (batch, n, dim_v) 1 • 使用计算得到的注意力权重 attention_weights 去加权 v ,即 torch.matmul(attention_weights, v) .这样得到的 attention_output 是注意力 机制的输出,形状为 (batch_size, sequence_length, dim_v) . 返回结果 return attention_output 1 最终输出的是 attention_output ,它表示输入序列经过自注意力机制后的输出. 测试 x = torch.randn(2, 5, 10) # batch_size=2, sequence_length=5, dim_q=10 attention = SelfAttentionScratch(dim_q=10, dim_k=8, dim_v=8) output = attention.forward(x) print(output.shape) # 输出应该是 (2, 5, 8) 1 2 3 4 这里对手撕的自注意力机制进行测试: • 输入 x 是一个随机生成的张量,形状为 (2, 5, 10) ,表示有 2 个样本,每个样本的序列长度 是 5,特征维度是 10. • 创建了一个 SelfAttentionScratch 对象,并调用 forward 方法. • 最后输出的形状是 (2, 5, 8) ,这与 dim_v=8 的设定一致.
