# 手撕自注意力机制(不用Pytorch)(Self Attention)

自注意力机制是 Transformer 的核心，其数学公式为：

$$ \text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V $$

其中：
*   $Q$ (Query): 查询向量，表示当前关注点。
*   $K$ (Key): 键向量，用于被匹配。
*   $V$ (Value): 值向量，表示实际内容信息。
*   $d_k$: 缩放因子，用于防止点积过大导致梯度消失。

### 手撕代码实现

面试中要求“手撕”通常指不依赖 `nn.Linear` 等高层 API，使用基础张量运算实现。

```python
import torch
import torch.nn.functional as F

class SelfAttentionScratch:
    def __init__(self, dim_input, dim_k, dim_v):
        self.dim_input = dim_input
        self.dim_k = dim_k
        self.dim_v = dim_v
        # 手动初始化权重矩阵 (Xavier/Glorot 初始化的简化版)
        self.W_q = torch.randn(dim_input, dim_k) / dim_k**0.5
        self.W_k = torch.randn(dim_input, dim_k) / dim_k**0.5
        self.W_v = torch.randn(dim_input, dim_v) / dim_v**0.5
        
    def forward(self, x, mask=None):
        # x shape: (batch_size, seq_len, dim_input)
        batch_size, seq_len, _ = x.shape
        
        # 1. 线性变换得到 Q, K, V
        # 矩阵乘法: (B, L, D_in) @ (D_in, D_k) -> (B, L, D_k)
        Q = torch.matmul(x, self.W_q)  
        K = torch.matmul(x, self.W_k)  
        V = torch.matmul(x, self.W_v)  
        
        # 2. 计算注意力得分
        # K.transpose(1, 2) 将 K 变为 (B, D_k, L)
        # 结果: (B, L, L)
        scores = torch.matmul(Q, K.transpose(1, 2))  
        scores = scores / torch.sqrt(torch.tensor(self.dim_k, dtype=torch.float32))
        
        # --- 补充细节：Mask 处理 ---
        if mask is not None:
            # mask 通常是 (B, L, L) 或 (L, L)，1 表示有效，0 表示屏蔽
            # 将屏蔽位置设为负无穷大，使得 softmax 后概率为 0
            scores = scores.masked_fill(mask == 0, -1e9)

        # 3. Softmax 归一化
        # 在最后一个维度 上进行归一化
        attention_weights = F.softmax(scores, dim=-1)
        
        # 4. 加权求和
        # (B, L, L) @ (B, L, D_v) -> (B, L, D_v)
        output = torch.matmul(attention_weights, V)  
        
        return output
```

**## 常见考点**
*   **复杂度分析**：Self-Attention 的时间复杂度和空间复杂度是多少？瓶颈在哪里？
*   **缩放因子**：为什么要除以 $\sqrt{d_k}$？如果不缩放会发生什么？
*   **Mask 类型**：Padding Mask 和 Causal Mask (因果掩码/未来掩码) 的区别和应用场景？
