# 介绍一下RoPE位置编码

在Transformer中,位置编码(Positional Encoding, PE)是为了让模型感知序列中元素的顺序. 传统Transformer使用的是绝对位置编码,即对序列中每个位置赋予一个唯一标识符,例如 Sinusoidal位置编码,通过一组固定的正弦和余弦函数编码每个位置.然而,绝对位置编码存在局限 性:无法直接捕捉元素之间的相对位置信息,而后者对很多自然语言任务至关重要.相对位置编码能 让模型知道两个词之间的距离,这在如句法分析等任务中非常有用.

1. RoPE的动机

**RoPE的目标是：** 通过绝对位置编码的方式实现相对位置编码,即不改变现有Transformer框架下绝对 位置编码的方式,但能让模型自动捕捉元素之间的相对位置信息. RoPE的理论基础是通过在查询(query)和键(key)向量上施加一个旋转变换,将位置信息嵌入这 些向量中.RoPE对位置编码的设计不仅适用于传统的Attention机制,还能扩展到线性Attention,这 一点是其创新所在.
