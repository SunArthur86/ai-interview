# 什么是对齐税 Alignment Tax?如何缓解

大模型在RLHF对齐过程中可能遗忘预训练能力（导致基准性能下降），这种现象称为对齐税。缓解方法包括：1）PPO-ptx：在PPO目标中增加预训练数据上的损失函数；2）模型平均：在预训练和RLHF模型权重间插值；3）异构模型平均：按层分配不同权重比例进行平均，优化对齐性能。
