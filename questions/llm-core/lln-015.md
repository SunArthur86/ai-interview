# 什么是对齐税 Alignment Tax?如何缓解

大模型在预训练过程中获得了广泛的能力,但在RLHF框架下对大语言模型进行对齐时,可能会导致模型遗忘预训练获得的能力,这
就是对齐税 Alignment Tax,即 RLHF 虽然有助于对齐人类偏好,但也可能导致模型在某些 NLP 基准上的性能下降
**解决方法：**
•
**PPO-ptx：** PPO-ptx就是在原本的PPO优化目标(带KL行为约束的最大化累积奖励)基础上,增加了一项当前policy在pretrain
数据集上的优化目标,或者说加了在pertain数据集上的预训练loss,即ptx loss:
