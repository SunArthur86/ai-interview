5. 总结
LayerNorm 和 RMSNorm 都是有效的正则化方法，但它们在核心的计算方式和应用场景上有所不同：
- LayerNorm 更适合处理均值与方差对特征影响较大的任务，特别是小批量数据和 NLP 任务中。
- RMSNorm 则适用于幅度归一化为主、特征幅度较大的场景，如深层神经网络或高维数据中，同时
它的计算效率也更高。
35. 介绍一下Bert的结构
BERT (Bidirectional Encoder Representations from Transformers) 是基于 Transformer 架构的预训
练模型。其核心在于利用 Transformer 模型中的 编码器 (Encoder) 部分，通过双向（从左到右、从右
到左）的上下文信息来理解句子中的词语。因此，BERT 能够更好地捕捉语言的语义关系，广泛应用于
自然语言处理 (NLP) 任务。

BERT模型的输入
 BERT 的输入不仅是简单的单词向量，还包含了丰富的上下文信息，它由以下三种嵌入 (Embedding) 
的求和表示：
- WordPiece Embedding ：BERT 使用 WordPiece 作为分词方法，将单词划分为子词单元。这
种处理方式既能处理未知词汇，又能提高模型的灵活性和泛化能力。例如，罕见或不规则单词会被
分成更常见的子词单位，进而能够在训练中更好地学习到词汇语义。
- 位置嵌入 (Position Embedding) ：由于 BERT 只使用 Transformer 的编码器部分，并不依
赖于序列化结构（如 RNN 或 LSTM），因此它无法从输入序列中自然地获取位置信息。为了弥补这
一点，BERT 通过位置嵌入为每个词汇添加了位置特征，使模型能够感知词汇在序列中的相对位
置。BERT 初始化了一个位置嵌入矩阵，并在训练过程中学习这些位置向量。
- 段落嵌入 (Segment Embedding) ：在 BERT 中，输入通常是两个句子拼接而成，特别是在句
子预测任务 (Next Sentence Prediction, NSP) 中。因此，BERT 为输入中的每个 token 添加一个
段落嵌入，用来区分句子 A 和句子 B，帮助模型更好地理解句子之间的关系。
最终，BERT 的输入是这三种嵌入的求和：wordpiece embedding + position embedding + 
segment embedding，这样能够同时捕捉词汇、位置、以及句子间的信息。

BERT 的核心是自注意力机制，它能在同一句

- 前馈神经网络 (Feed-Forward Network,

自注意力层之后，每个词的表示都会经过一个
前馈神经网络进行进一步处理，通常是两层线性变

- 层归一化 (Layer Normalization)
每一层的输出都会经过层归一化，以确保训练
的稳定性和加速收敛。与传统的批归一化 (Batch 
Normalization) 不同，层归一化在每个样本的内部
进行归一化处理，使其更适合处理序列化数据。

自注意力机制 (Self-Attention Mechanism)
self-attention出现的原因
◦
为了解决RNN、LSTM等常用于处理序列化数据的网络结构无法在GPU中并行加速计算的问题
◦
由于每个目标词是直接与句子中所有词分别计算相关度(attention)的，所以解决了传统的RNN
模型中长距离依赖的问题。通过attention，可以将两个距离较远的词之间的距离拉近为1直接
计算词的相关度，而传统的RNN模型中，随着距离的增加，词之间的相关度会被削弱。

预训练任务 (Pre-training Tasks)
 BERT 在大规模语料上通过自监督学习进行了预训练，主要包括两个任务：
- Masked Language Model (MLM)：
BERT 通过随机掩盖 (mask) 句子中 15% 的 token，并要求模型根据上下文预测这些被掩盖的词。
具体操作是：
◦
80%用 [MASK] 替换 token。
◦
10% 用随机 token 替换。
◦
10% 保持原 token 不变。
- 这种策略既能防止模型只依赖 [MASK] 进行猜测，同时保证了模型的泛化能力。
- Next Sentence Prediction (NSP)：
      BERT 通过给定两个句子，要求模型判断这两个句子是否为连续上下句。这一任务帮助 BERT 学习
句子间的关系，提高在问答、推理等任务中的表现。语料中50%的句子，选择其相应的下一句一起形
成上下句，作为正样本；其余50%的句子随机选择一句非下一句一起形成上下句，作为负样本。这种
设定，有利于sentence-level tasks，例如问答。
作者特意说了语料的选取很关键，要选用document-level的而不是sentence-level的，这样可以具
备抽象连续长序列特征的能力。
模型的规模与超参数
◦
BERT 的 base 版本 有 110M 参数（12 层、768 隐藏单元、12 个注意力头）。
◦
large 版本 则有 340M 参数（24 层、1024 隐藏单元、16 个注意力头）。
BERT 的预训练采用了 Adam 优化器，初始学习率经过 10,000 步的 warm-up 后逐渐衰减，总共进行
了超过 33 亿个训练样本、100 万步的训练过程。
2. BERT模型特点
- 使用transformer作为算法的主要框架，transformer能更彻底的捕捉语句中的双向关系；
- 使用了mask language model 和next sentence prediction的多任务训练目标，是一个自监督的过
程，不需要数据的标注；
- 使用tpu这种强大的机器训练了大规模的预料，是NLP的很多任务达到了全新的高度。
       BERT本质上是在海量语料的基础上，通过自监督学习的方法为单词学习一个好的特征表示。该模
型的优点是可以根据具体的人物进行微调，或者直接使用预训练的模型作为特征提取器。
