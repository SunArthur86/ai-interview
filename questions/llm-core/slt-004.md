# GRPO算法是什么？DeepSeek为什么用它？

GRPO（Group Relative Policy Optimization）是DeepSeek提出的强化学习算法，简化了PPO。

与PPO的核心区别：
- PPO需要Critic Model估计Value（baseline）
- GRPO用组内均值替代Value：对同一个prompt采样G个response，计算group mean作为baseline

Advantage计算：
- PPO: A = R - V(s)（需要Critic）
- GRPO: A_i = (R_i - mean(R)) / std(R)（不需要Critic）

优势：
1. 省去Critic Model，减少25%显存
2. 训练更稳定
3. 适合推理任务（如数学、代码），因为这类任务有明确正确答案

DeepSeek-R1使用GRPO实现推理能力涌现：
- 纯RL训练（无SFT），模型自发产生思维链（Chain-of-Thought）
- AHA moment：模型学会自我反思和验证
- 超越OpenAI o1在多个推理benchmark
