# 手撕RoPE

Rotary Position Embedding (RoPE) 通过旋转变换将位置信息注入查询和键向量。核心公式为：对于位置 m 的向量，在偶数维度 2m 和奇数维度 2m+1 上应用旋转：[q_{2m}, q_{2m+1}] -> [q_{2m}cos(mθ_i)-q_{2m+1}sin(mθ_i), q_{2m}sin(mθ_i)+q_{2m+1}cos(mθ_i)]。频率 θ_i = 10000^(-2i/d)，其中 d 是维度，i 是维度索引的一半。PyTorch 实现中，先构建频率 freqs = 10000^(-torch.arange(0, dim, 2)/dim)，然后计算旋转角度 angle = positions * freqs，最后对输入 x 的偶数和奇数维度分别应用 cos 和 sin 变换。
