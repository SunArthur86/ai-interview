# 什么任务适合上RLHF,什么任务sft效果更好

•
RLHF适合解决Reward比较明确的复杂问题,典型的是数理化、Coding等有标准答案的学科
•
对于文创、角色扮演等一般依赖于长文本能力,需要做好预训练和sft,偶尔会做DPO训练.
