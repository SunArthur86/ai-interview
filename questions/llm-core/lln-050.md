# 大模型中常用的激活函数有哪些?为什么LLaMA选择SwiGLU

- **主流激活函数演进:**

| 激活函数 | 公式 | 代表模型 |
|---------|------|----------|
| ReLU | max(0,x) | 早期模型 |
| GELU | x·Phi(x) | BERT, GPT-2 |
| SiLU/Swish | x·sigma(x) | 部分现代模型 |
| SwiGLU | Swish(xW)⊙(xV) | LLaMA, Qwen |

- **SwiGLU详解:**
**传统FFN：** FFN(x) = ReLU(xW₁)W₂
**SwiGLU：** FFN(x) = (Swish(xW) ⊙ xV)W₂

- 引入门控机制: xV作为门控信号调制Swish(xW)的输出
- 需要三个权重矩阵(W, V, W₂),参数量增加约50%
- 为保持总参数量不变,通常将隐藏层维度缩小(如从4d变为${8d \over 3}$)

- **为什么选择SwiGLU:**
1. 效果好 - 多个基准测试上优于ReLU和GELU
2. 门控机制 - 允许模型动态选择信息通路,增强表达能力
3. 平滑梯度 - SiLU的平滑性有助于深层网络训练
