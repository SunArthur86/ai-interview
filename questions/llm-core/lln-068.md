# MHA、MQA、GQA 三者的区别是什么?为什么 GQA 是最优选择

注意力机制的 KV Head 复用演进:

- **MHA(Multi-Head Attention):** Q、K、V 各有 N 个头(如 32 个),每个 Q 头对应独立的 K/V 头.效果最好,但 KV Cache 显存占用大.

- **MQA(Multi-Query Attention):** Q 仍有 N 个头,但 K/V 只有 1 个头,所有 Q 头共享同一组 K/V.KV Cache 显存降低 N 倍,但质量下降明显.

- **GQA(Grouped-Query Attention):** 折中方案,Q 有 N 个头,K/V 有 G 个头(1 < G < N),每 N/G 个 Q 头共享一组 K/V.

- **为什么 GQA 最优:** 质量接近 MHA,KV Cache 显存大幅降低,推理速度显著提升,是效果和效率的最佳平衡点.Llama 2-70B 和 Llama 3 全系列均采用 GQA.
