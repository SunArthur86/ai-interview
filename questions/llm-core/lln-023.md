# 手撕线性回归(Minimax必考题)

使用NumPy实现线性回归，包含解析解（正规方程）和数值解（梯度下降）两种方法，并在数据预处理中添加偏置项。

### 1. 解析解

利用正规方程求极值：$\theta = (X^T X)^{-1} X^T y$。适用于特征数量较少的情况。

```python
import numpy as np

class LinearRegression:
    def __init__(self, method='gd', learning_rate=0.01, n_iters=1000):
        self.method = method
        self.lr = learning_rate
        self.n_iters = n_iters
        self.weights = None

    def fit(self, X, y):
        # 数据预处理：添加偏置项 (Intercept)
        # X shape: (m_samples, n_features) -> (m_samples, n_features + 1)
        m_samples = X.shape[0]
        X_b = np.c_[np.ones((m_samples, 1)), X]
        
        if self.method == 'analytical':
            # 正规方程: theta = (X'X)^-1 X'y
            # np.linalg.inv 求逆矩阵
            self.weights = np.linalg.inv(X_b.T.dot(X_b)).dot(X_b.T).dot(y)
            
        elif self.method == 'gd':
            # 梯度下降初始化
            n_features = X_b.shape[1]
            self.weights = np.random.randn(n_features)
            
            for _ in range(self.n_iters):
                # 预测值
                y_pred = X_b.dot(self.weights)
                # 计算梯度: (1/m) * X^T * (y_pred - y)
                gradients = (1/m_samples) * X_b.T.dot(y_pred - y)
                # 更新权重
                self.weights = self.weights - self.lr * gradients
                
    def predict(self, X):
        # 预测时同样需要添加偏置项
        X_b = np.c_[np.ones((X.shape[0], 1)), X]
        return X_b.dot(self.weights)
```

### 2. 关键细节补充

*   **偏置项处理**：代码中通过 `np.c_[np.ones(...), X]` 将 $w_0$ 对应的 $x_0$ 设为 1，这样就无需单独计算偏置，统一包含在权重向量 $\theta$ 中。
*   **复杂度对比**：
    *   解析解：计算矩阵乘法和求逆，复杂度约为 $O(n^3)$（$n$ 为特征数）。当特征数很大（如 10000+）时，计算非常慢且矩阵可能不可逆。
    *   梯度下降：复杂度取决于迭代次数，适用于大规模数据集。
*   **奇异矩阵处理**：如果 $X^T X$ 不可逆（如特征存在严格线性相关性），解析解会失效。此时可以使用 `np.linalg.pinv` (伪逆) 替代 `np.linalg.inv`。

### ## 常见考点
1.  **正规方程的局限性**：什么时候不能用解析解？（特征数远大于样本数、特征多重共线性导致矩阵不可逆、数据量过大无法装入内存计算 $X^T X$）
2.  **SVD 分解**：更稳定的解析解实现通常使用 SVD（奇异值分解），而不是直接求逆，能处理奇异矩阵情况。
3.  **特征缩放**：在使用梯度下降训练线性回归前，是否需要对特征进行归一化？（必须，否则不同特征尺度不同会导致收敛震荡或极慢，虽然解析解不受影响）
