# LongLora是怎么做的

LoRA在遇到超长上下文的查询时,提升效果不够明显,并且会带来模型困惑度增加的问题.

首先将模型的输入分为几组,并在每个组中分别进行注意力计算,同时在每个组中,将原有的token移动组大小的一半来保证相邻
注意力头之间的信息交互
