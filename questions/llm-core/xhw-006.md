# GLM的自回归空白填充方法

GLM 的自回归空白填充旨在统一理解和生成任务。

**1. 基本原理**
- 在文本中随机填充连续的空白片段 [MASK]。
- 模型以自回归的方式（从左到右）重建这些空白部分。
- 在填充空白时，模型可以观察到未被掩码的上下文（即双向上下文）。

**2. 与 BERT (MLM) 的区别**
- **BERT**：预测独立的掩码词，输出是多个分类标签，难以处理生成任务；且存在 [MASK] 预训练和微调时的不一致。
- **GLM**：预测一段连续的文本，输出是文本生成任务；自回归填充避免了看到未来的信息，更符合生成范式。

**3. 优势**
- **统一框架**：既能像 BERT 一样做 NLU（理解），也能像 GPT 一样做 NLG（生成）。
- **更好的长距离依赖**：填充一段连续文本迫使模型学习更长范围的依赖关系。
