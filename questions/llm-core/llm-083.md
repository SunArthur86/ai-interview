# Group Normalization(GN)和 Instance Normalization(IN)的

区别主要在于归一化的计算范围不同：GN是将通道分组后在每个组内计算统计量（介于LayerNorm和InstanceNorm之间），而IN是将每个通道单独视为一组（即GN中分组数等于通道数）。IN更适用于风格迁移等需要去除图像特定对比度的任务，而GN是BN在Batch Size较小时的通用替代品。
