# Flash Attention的原理是什么?为什么能加速?Paged Attention解决了什么问题

- **Flash Attention - IO感知的精确注意力:**

**核心思想：** 减少HBM(高带宽内存)和SRAM之间的读写次数.

传统Attention计算S=QKᵀ需要O(N²)的中间矩阵,频繁读写HBM,显存占用巨大.

Flash Attention的优化:
1. 分块计算- 将Q/K/V分成小块,每次只加载一块到SRAM计算
2. 在线Softmax - 增量更新技巧,不需要完整注意力矩阵就能计算softmax
3. 融合内核 - QKᵀ到softmax到AV融合为一次GPU操作

**效果：** 速度提升2-4倍,显存从O(N²)降到O(N),结果与标准Attention完全一致.

- **Paged Attention - 解决KV Cache碎片化(vLLM核心):**

传统KV Cache为每个序列预分配连续显存,导致内部碎片和外部碎片.

Paged Attention借鉴OS虚拟内存的分页机制:
- KV Cache按固定大小的block存储(如每block存16个token的KV)
- 物理block可以不连续,逻辑地址通过block table映射
- 效果:显存利用率从~60%提升到~96%,吞吐量提升2-4倍
