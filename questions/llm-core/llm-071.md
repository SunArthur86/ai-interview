# 什么是交叉熵损失函数?大模型的哪里有交叉熵损失函数

### 1. 基础概念
**交叉熵** 衡量两个概率分布之间的差异。在深度学习中，它常作为损失函数，用于衡量模型预测分布 $p$ 与真实标签分布 $q$ 之间的距离。

公式：$H(q, p) = -\sum q_i \log(p_i)$

### 2. 与 KL 散度的关系
*   **KL 散度 ($D_{KL}$)：** 衡量使用分布 $p$ 来表示分布 $q$ 时产生的信息损失。
*   **关系：** $H(q, p) = H(q) + D_{KL}(q||p)$。
*   **简化：** 在分类任务中，真实标签 $q$ 通常是 One-Hot 向量（即正确类别为 1，其余为 0），此时 $H(q)=0$（熵为0，无不确定性）。因此，**最小化交叉熵等价于最小化 KL 散度**。

### 3. 为什么选择交叉熵？
*   **梯度特性：** 对数函数在预测概率较小时（即错误时）能提供较大的梯度，加速收敛。
*   **避免梯度消失：** 相比于均方误差（MSE），交叉熵配合 Sigmoid/Softmax 能提供更饱和的梯度信号。

### 4. 大模型中的应用
在大语言模型（如 GPT）训练中：
*   任务是**预测下一个 Token**。
*   真实标签是序列中实际的下一个词（One-Hot 形式）。
*   模型输出是词表中所有词的概率分布。
*   损失函数即为预测分布中“真实下一个词”概率的负对数：$Loss = -\log(P(w_{t+1} | w_{1...t}))$。
