# MHA、MQA和GQA有什么区别？

Multi-Head Attention (MHA)：每个注意力头有独立的Q、K、V。标准Transformer使用。N个头意味着N份KV Cache。

Multi-Query Attention (MQA)：所有头共享同一组K和V，只有Q是多头。极端省内存，但效果有损。如PaLM、Falcon。

Grouped-Query Attention (GQA)：MHA和MQA的折中。将Q头分成G组，每组共享一对K和V。当G=N时等价于MHA，当G=1时等价于MQA。如Llama 2 70B、Llama 3。

KV Cache对比（N=头数）：
- MHA: KV Cache = 2N × d × seq_len
- MQA: KV Cache = 2 × d × seq_len（减少N倍）
- GQA: KV Cache = 2G × d × seq_len（减少N/G倍）

效果：GQA在保持接近MHA质量的同时，大幅减少推理时的KV Cache内存和带宽。
