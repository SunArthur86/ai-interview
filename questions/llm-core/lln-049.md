# LayerNorm和RMSNorm的区别是什么?Pre-Norm和Post-Norm有什么不同

- **LayerNorm vs RMSNorm:**

| 特性 | LayerNorm | RMSNorm |
|------|-----------|---------|
| 计算方式 | 减均值+除标准差+缩放偏移 | 只除RMS(均方根)+缩放 |
| 参数量 | 2N (gamma和beta) | N (只有gamma) |
| 计算量 | 较高 | 较低 |
| 代表模型 | BERT, GPT-2 | LLaMA, Qwen |

RMSNorm是LayerNorm的简化版本,去掉了减均值和偏移项,效果相当但计算更快.

- **Pre-Norm vs Post-Norm:**

**Post-Norm：** x -> Attention -> Add(x) -> LayerNorm -> FFN -> Add -> LayerNorm
**Pre-Norm：** x -> LayerNorm -> Attention -> Add(x) -> LayerNorm -> FFN -> Add(x)

| 特性 | Post-Norm | Pre-Norm |
|------|-----------|----------|
| 训练稳定性 | 差(深层难收敛)| 好(残差直连) |
| 最终效果 | 略好 | 略差 |
| 深层模型 | 需warmup | 直接训练 |
| 代表 | 原始Transformer, BERT | GPT-2+, LLaMA |

大模型几乎全部使用Pre-Norm,因为深层模型训练稳定性更重要.
