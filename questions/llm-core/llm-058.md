# 手撕绝对位置编码

由于Transformer不具有处理序列顺序的内在机制,因此通过为每个位置添加位置编码来为模型提供顺 序信息.位置编码采用正弦和余弦函数生成,公式为:

**总结**
总结 这个 PositionalEncoder 类使用正弦和余弦函数为序列中的每个位置生成固定的位置编码,并将它们 添加到输入嵌入中.通过这种方式,Transformer 模型能够感知序列中各个词的顺序信息,帮助模型 更好地处理序列数据.
