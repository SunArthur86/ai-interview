# DeepSeek-R1-Zero的Zero含义

### DeepSeek-R1-Zero 的 Zero 含义

#### 1. 核心含义
**Zero** 源自 **AlphaZero**，意指**无需人类标注数据**，仅依靠**强化学习（RL）**从零开始自我进化。

#### 2. 方法论
- **无 SFT 阶段**：传统的模型训练流程通常包含“预训练 -> SFT（监督微调） -> RL”。DeepSeek-R1-Zero 跳过了 SFT 阶段。
- **纯 RL 训练**：直接在基础模型上应用强化学习。模型通过与环境（如数学题的验证器、代码解释器）交互，根据奖励信号自我探索推理策略，而不是模仿人类的写作风格或思维链。

#### 3. 结果与特点
- **涌现能力**：虽然起初没有学习人类的 CoT 样式，但模型在 RL 过程中自发涌现出了“回溯”、“自我反思”等高级推理行为。
- **双刃剑**：这种方法展示了纯 RL 的巨大潜力，但也带来了语言混杂、可读性差等问题（这也是后来 DeepSeek-R1 引入少量冷启动数据的原因）。
