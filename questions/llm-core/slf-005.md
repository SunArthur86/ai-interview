# 什么是RLAIF？和RLHF有什么区别？

RLAIF（Reinforcement Learning from AI Feedback）：用AI（而非人类）提供偏好标注。

RLHF流程：
1. 人工标注偏好对（A preferred over B）—— 成本高、慢
2. 训练Reward Model
3. 用PPO/DPO优化

RLAIF流程：
1. 用强LLM（如GPT-4/Claude）替代人工标注偏好
2. 训练Reward Model或直接用AI的评分
3. 用PPO/DPO优化

优势：
- 成本极低（API调用 vs 人工标注）
- 速度快（可批量生成）
- 可扩展性强

质量争议：
- AI标注可能有偏见（偏好更长/更复杂的回答）
- Constitutional AI（Anthropic）：给AI一组原则，AI根据原则评判
- 研究表明RLAIF效果接近RLHF（在某些任务上）

实践：大多数团队已经用RLAIF/Constitutional AI替代人工标注。
