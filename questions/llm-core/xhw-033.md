# Qwen通义千问的结构

Qwen (通义千问) 是阿里推出的系列大模型，其结构基于 Transformer Decoder，并有多项改进。

### 1. 整体架构
- 基于 Transformer 解码器结构，使用 causal mask 进行自回归生成。

### 2. 核心组件
- **词表**：使用 BPE 分词，词汇表较大（约 152K），优化了多语言（特别是中文）的压缩效率。
- **嵌入层**：使用 RoPE (旋转位置编码) 处理位置信息，支持长文本外推。
- **注意力机制**：使用 GQA (分组查询注意力) 以加速推理，减少显存占用。
- **前馈网络 (FFN)**：采用 SwiGLU 激活函数。
- **归一化**：使用 RMSNorm，并且采用 Pre-Norm 结构（输入先归一化再进入子层）。

### 3. 特殊创新
- **非绑定嵌入**：输入嵌入层和输出层的权重不共享，提升了模型性能。
- **偏置调整**：去除了大多数线性层的 Bias，但在 QKV 注意力层中保留了 Bias，增强了模型的外推能力。
- **长上下文**：通过 NTK-aware 插值等技术扩展上下文窗口长度。

### 4. 训练策略
- **预训练**：使用超过 3T tokens 的多语言数据进行训练。
- **对齐**：包括 SFT（有监督微调）和 RLHF（人类反馈强化学习），以及专门的代码和数学能力增强训练。
