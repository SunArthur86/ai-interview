# Reward Model常见的hacking有哪些

**参考链接：** https://www.zhihu.com/question/662061754/answer/3573741988
1. 长度hacking:RM如果数据有限会极端偏好更长的回答
2. 风格hacking:对于markdown或者latex等高阶格式的偏爱
3. CoT的hacking:对于CoT范式本身的偏爱而非CoT对于解答用户问题的促进作用
此外,事实类问题(均存在概率分布的原子知识应该相信谁),安全性问题(处于边界的问题是应该拒答,正答还是指令遵循)等
任务对于RM过于困难也可能导致RM效果差.
