# LLaMA、Qwen和DeepSeek系列的架构有什么主要区别

**三大主流开源模型系列的架构对比：** 

| 特性 | LLaMA-3 | Qwen-2.5 | DeepSeek-V2 |
|------|---------|----------|-------------|
| 注意力 | GQA | GQA | MLA |
| FFN | SwiGLU | SwiGLU | DeepMoE |
| Norm | RMSNorm | RMSNorm | RMSNorm |
| Norm位置 | Pre-Norm | Pre-Norm | Pre-Norm |
| 位置编码 | RoPE | RoPE | Decoupled RoPE |
| MoE | 否 | 否 | 是(细粒度+共享) |
| 词表 | 128K | 152K | 100K |
| 上下文 | 8K→1M | 128K | 128K |

- **DeepSeek-V2的独特设计:**

1. **MLA (Multi-head Latent Attention)** - KV Cache压缩为原来的1/4,通过低秩投影实现
2. **细粒度MoE** - 160个路由专家,每token激活6个 + 2个共享专家始终激活
3. **Decoupled RoPE** - MLA与RoPE不完全兼容,将K分为不旋转的压缩部分和独立的RoPE部分

- **Qwen系列的特色:**
- 词表最大(152K),中文处理能力强
- 支持多模态(视觉+文本统一)
- Code能力突出
