# LLaMA、Qwen和DeepSeek系列的架构有什么主要区别

### 三大主流开源模型系列的架构对比

| 特性 | LLaMA-3 | Qwen-2.5 | DeepSeek-V2 |
|------|---------|----------|-------------|
| 注意力 | GQA | GQA | MLA |
| FFN | SwiGLU | SwiGLU | DeepMoE |
| Norm | RMSNorm | RMSNorm | RMSNorm |
| Norm位置 | Pre-Norm | Pre-Norm | Pre-Norm |
| 位置编码 | RoPE | RoPE | Decoupled RoPE |
| MoE | 否 | 否 | 是(细粒度+共享) |
| 词表 | 128K | 152K | 100K |
| 上下文 | 8K→1M | 128K | 128K |

### DeepSeek-V2 的独特设计

1. **MLA (Multi-head Latent Attention)**：将 KV Cache 压缩为原来的 1/4，通过低秩投影实现，大幅降低显存。
2. **细粒度 MoE**：160 个路由专家，每 token 激活 6 个 + 2 个共享专家始终激活。
3. **Decoupled RoPE**：MLA 与 RoPE 不完全兼容，将 K 分为不旋转的压缩部分和独立的 RoPE 部分。

### Qwen 系列的特色
- 词表最大（152K），中文处理能力强。
- 支持多模态（视觉+文本统一）。
- Code 能力突出。
