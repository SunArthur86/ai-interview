# SwiGLU激活函数是什么？为什么比ReLU好？

SwiGLU（Swish-Gated Linear Unit）= Swish(xW) × (xV)，是GLU（Gated Linear Unit）的变体。

公式：SwiGLU(x) = Swish(xW) ⊗ (xV)
其中Swish(x) = x · sigmoid(x)，W和V是可学习权重。

与ReLU对比：
1. 平滑性：Swish处处可导（ReLU在x=0处不可导），梯度更平滑
2. 非单调性：Swish在x<0时也有非零梯度（ReLU直接为0），避免神经元死亡
3. 门控机制：通过(xV)项自动学习哪些信息重要

代价：需要额外的权重矩阵V，参数量增加。但效果提升明显。

使用：Llama系列、PaLM、DeepSeek等均使用SwiGLU替代ReLU/GELU。
