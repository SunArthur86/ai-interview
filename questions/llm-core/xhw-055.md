# MLA多头隐变量注意力

### MLA 多头隐变量注意力 (Multi-head Latent Attention)

#### 1. 概念
Multi-head Latent Attention (MLA) 是一种针对 Transformer 推理效率优化的注意力变体。其核心思想是通过**低秩分解**或**引入隐变量**来压缩 Key (K) 和 Value (V) 的存储与计算开销，特别是为了减少显存中的 KV Cache 占用。

#### 2. 核心机制
在标准的 Multi-head Attention (MHA) 中，每个头都有独立的 Q、K、V 投影，导致 KV Cache 随序列长度和头数线性增长，极大占用显存。

MLA 的优化思路（以 DeepSeekV2/V3 实现为例）：

1. **压缩 KV (KV Compression)**：
   - 将原本高维的 Key 和 Value 先投影到一个低维的“隐向量”空间 (Latent Vector)。
   - 注意力计算基于这个低维的隐向量进行，从而大幅减少显存占用。

2. **解耦 Query (Query Decoupling)**：
   - 为了保持模型的表达能力，Query 通常保留较高的维度或通过特定的上标矩阵与压缩后的 KV 进行交互。

#### 3. 优势
- **极低的显存占用**：KV Cache 仅需存储低维隐向量，推理时可支持更长的上下文。
- **计算加速**：由于参与注意力计算的维度降低， Attention Score 计算矩阵变小，提升了推理速度。

#### 4. 注意力公式 (示意)
假设压缩矩阵为 $W_{KV}$，则 $C$ (Compressed KV) = $X W_{KV}$。后续的 Attention 计算基于 $C$ 进行，而非原始的 $K, V$。通过这种方式，MLA 在几乎不损失模型性能的前提下，实现了极致的推理优化。
