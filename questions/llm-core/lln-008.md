# 什么是自回归语言模型和n-gram模型

1. 自回归语言模型
用历史信息预测下一个字符/token,然后加入历史信息中,再去预测下一个

x ∼ i p(x ∣ i x ) 1:i−1 1/T
其中T≥0 是一个控制我们希望从语言模型中得到多少随机性的温度参数,加了1/T之后要重新归一化概率
2. N-gram模型
在一个n-gram模型中,关于 的预测只依赖于最后
**个字符而不是整个历史：**
x i n −1

x ∼ i p(x ∣ i x ) i−(n−1):i−1 1/T
