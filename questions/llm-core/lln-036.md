# LORA 介绍,秩的数学含义,怎么计算矩阵的秩

• 秩是指矩阵中线性无关的行或列的最大数目 •
**具体地说：** **线性独立性：** 矩阵的秩是矩阵中线性独立行或列的数量.如果矩阵的某些行或列可以通过其他行或列的线性组合来表示,那
么这些行或列就不是线性独立的. **维度：** 矩阵的秩可以告诉我们矩阵的列空间或行空间的维度.秩越高,矩阵的列空间或行空间的维度就越大. **解的个数：** 在线性方程组中,矩阵的秩可以决定方程组解的个数.如果增广矩阵的秩等于系数矩阵的秩,并且等于变量的个
数,那么方程组有唯一解;如果增广矩阵的秩小于系数矩阵的秩,那么方程组无解;如果增广矩阵的秩小于变量的个数,那
么方程组有无穷多解. **矩阵的逆：** 如果矩阵是方阵,那么只有当矩阵的秩等于其阶数时,矩阵才是可逆的.
• 计算方式 通过初等行变换将矩阵转换为阶梯形或行最简形,然后计算非零行的数量
