# LORA 介绍,秩的数学含义,怎么计算矩阵的秩

1. **LoRA 介绍**：
   LoRA (Low-Rank Adaptation) 是一种高效微调技术。它冻结预训练模型的权重，并在 Transformer 层的每个线性块旁注入可训练的秩分解矩阵，从而在不增加推理开销的情况下大幅减少可训练参数量。
   - **原理公式**：$h = W_0 x + BAx$，其中 $W_0$ 冻结，$B \in \mathbb{R}^{d \times r}$，$A \in \mathbb{R}^{r \times k}$，秩 $r \ll \min(d, k)$。
   - **初始化**：$A$ 使用随机高斯分布初始化，$B$ 使用全 0 初始化，确保训练初始状态 $BA=0$，完全等同于预训练模型。
   - **推理优势**：由于 $BA$ 可以合并到 $W_0$ 中 ($W_{new} = W_0 + BA$)，推理时不存在额外的计算开销。

**LoRA 架构示意图**：
```text
       Input (x)                     Input (x)
         │                             │
         ▼                             ▼
    ┌────────┐                    ┌─────────┐
    │   A    │ (r x k)            │   LoRA  │
    │(Train)│                    │ Branch  │
    └───┬────┘                    └────┬────┘
        │                              │
        │   ┌────────┐                 │
        └──►│   B    │ (d x r)          │

            └───┬────┘                 │
                │                      │
                │  d (Hidden Dim)      │
                ▼                      ▼
            ┌──────────────────────┐   │
            │         +            │◄──┘
            └──────────┬───────────┘
                       │
                       ▼
                Output (h)
``` 

2. **秩 的数学含义**：
   - **线性无关性**：矩阵中线性无关的行向量或列向量的最大数目。
   - **维度意义**：秩代表了矩阵列空间（或行空间）的维度，即矩阵能覆盖的信息广度。秩越低，矩阵包含的冗余信息越多。
   - **方程解与逆矩阵**：秩决定了方程组解的性质（唯一解、无解、无穷多解）；方阵满秩才可逆。
   - **LoRA 中的意义**：假设模型在微调时的参数更新量矩阵 $\Delta W$ 具有低秩特性（即内在维度低），这意味着微调不需要改变所有参数的方向，只需在几个关键的低维子空间进行调整即可。

3. **秩的计算**：
   - 通过初等行变换将矩阵化为行阶梯形，非零行的数量即为矩阵的秩。
   - **奇异值分解 (SVD)**：秩等于矩阵非零奇异值的个数，这在理论分析和矩阵压缩中非常重要。

**## 常见考点**
1. 为什么 LoRA 中的 $A$ 和 $B$ 一个是随机初始化，一个是零初始化？（为了保证训练开始时与原模型一致）
2. 如何选择 LoRA 的秩 $r$？（通常取 8, 16, 32, 64，越大表达力越强但参数越多）
3. LoRA 与全量微调在性能上的差异通常是多少？（通常在 1-2% 以内，甚至更好）
4. LoRA 的推理是如何做到无额外开销的？（权重合并 $W = W_0 + BA$）
