# 讲讲attention的各个变体

Attention的变体主要包括：1. 加性Attention（Bahdanau）：通过tanh和多层感知机计算得分；2. 点积Attention：通过Query和Key的点积计算得分，常用于Transformer；3. 缩放点积Attention：点积Attention的改进，通过缩放因子解决梯度消失问题；4. 多头Attention：并行计算多个Attention头，捕捉不同子空间的特征。
