# 为什么attention要scaled

Transformer 设计 Q、K 和 V 是为了分离查询、匹配和内容提取三个步骤。Q 是查询向量，用于提问；
K 是键向量，用于匹配和判断相关性；V 是值向量，用于传递实际信息。Q 和 K 需要独立的权重矩阵，
是为了让模型有能力从不同角度表达输入序列中的依赖关系，而最终的上下文信息是从 V 中提取的，
通过 Q 和 K 的相关性来加权得到。
