# Llama 3 相比 Llama 2 有哪些关键技术改进

- **Llama 3 核心技术升级**

1. **Tokenizer 升级**
- Llama 2: 32K vocab (SentencePiece)
- Llama 3: 128K vocab (tiktoken-based)
- 效果:更好的多语言支持、代码处理能力更强、压缩率提升 15%

2. **Grouped-Query Attention (GQA)**
- Llama 2 (70B): MHA → GQA
- Llama 3 (全系 8B/70B/405B): 全部使用 GQA
- 优势:KV Cache 显存减少,推理更快

3. **RoPE 基频调整**
- Llama 2: θ=10000
- Llama 3: θ=500000(支持更长的上下文外推)

4. **SwiGLU 激活函数**
- 与 Llama 2 一致:FFN 使用 SwiGLU

5. **训练数据**
- Llama 2: 2T tokens
- Llama 3: 15T+ tokens(多语言、代码、推理密集)

6. **规模**
- 8B / 70B / 405B(400B+ 是目前最大开源稠密模型之一)
- 遵循 Chinchilla 最优 ratio

7. **对齐技术**
- RLHF + DPO 组合
- 多轮对话能力显著增强
- 工具调用(Function Calling)原生支持

- **8B vs 70B 对比**:8B 适合边缘部署,70B 接近 GPT-4 水平,405B 在部分基准超越 GPT-4.
