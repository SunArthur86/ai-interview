# QLoRA的原理是什么?如何实现4-bit量化下的高效微调

QLoRA = Quantization + LoRA,在4-bit量化的基座模型上训练LoRA适配器.

- **核心创新:**

1. **NF4 (Normal Float 4) 量化**
- 基于权重的正态分布设计量化格子,信息论最优
- 量化过程:W -> NF4(4-bit) + 量化常数(fp16)

2. **双重量化 (Double Quantization)**
- 不仅量化权重,还量化量化常数本身(fp32->fp8)
- 每参数平均节省0.37bit

3. **分页优化器 (Paged Optimizer)**
- 使用NVIDIA统一内存,显存不足时自动转移到CPU
- 避免OOM

- **QLoRA训练流程:**
1. 加载预训练模型,权重量化为NF4 (4-bit)
2. 冻结量化权重,在fp16精度下训练LoRA适配器
3. 前向传播时反量化为fp16计算
4. 反向传播梯度只流经LoRA参数

- **效果:** 65B模型可在单张48GB GPU上微调,效果接近全参数微调(差距<1%),显存减少约75%.
