# 什么是大语言模型(LLM)?它的核心能力有哪些

- **大语言模型 (LLM)** 是基于 Transformer 架构、在海量文本上训练的超大规模神经网络模型.

- **核心能力**
1. **语言理解**:理解指令、推理、翻译
2. **文本生成**:对话、写作、代码补全
3. **知识记忆**:训练数据中的事实和模式
4. **指令跟随**:按格式/约束输出

- **代表模型**:GPT-4、Claude、GLM、Llama、Qwen

- **一句话理解**:LLM = 超大规模参数 + 海量训练数据 + Transformer 架构 → 涌现出语言理解和生成能力.
