# RoPE旋转位置编码的原理是什么？

RoPE（Rotary Position Embedding）通过旋转矩阵将位置信息编码到Q和K中。

核心思想：在二维空间中，将Q和K向量按照其位置旋转一定角度。两个token的attention内积只依赖于它们的相对位置差。

公式：对于位置m的向量q，乘以旋转矩阵R(m)：
q_m' = R(m) · q_m
其中R(m)是分块对角矩阵，每个2x2块是一个角度为m·θ_i的旋转矩阵。

优势：
1. 相对位置编码：attention结果自动包含相对位置信息
2. 长度外推：训练时4K上下文，推理时可扩展到8K+（配合NTK-aware scaling）
3. 计算高效：只需逐元素乘法
4. 支持LongRoPE/YaRN等外推方法扩展到100K+

Llama、Qwen、DeepSeek等主流模型均使用RoPE。
