# 什么是 Token?为什么大模型用 Token 而不是字或词

- **Token** 是大模型处理文本的最小单位,介于字符和词之间.

- **为什么不用字?**
- 单个字信息量太少,序列太长

- **为什么不用词?**
- 中文词边界不明确
- 词表太大(百万级),模型难以处理

- **Token 的优势**
- 词表可控(通常 3万~12万)
- 可处理新词、缩写、代码、emoji
- 多语言通用

- **例子**:'Hello World' → ['Hello', ' World'] = 2 tokens

- **估算**:1 个中文 ≈ 1-2 tokens,1 个英文单词 ≈ 1-1.5 tokens.
