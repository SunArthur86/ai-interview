# LLM数据合成工作有哪些

数学推理
1. MuggleMath: Assessing the Impact of Query and Response Augmentation on Math Reasoning
• 通过对原有问题进行改写,并通过对改写后的新问题采样不同的回答,获得新的增强数据集.
2. Distilling LLMs' Decomposition Abilities into Compact Language Models
• 提出了一个基于GSM8K数据集的AI生成数学问题子问题分解基准,通过微调和离线强化学习方法在小型模型中蒸馏复杂问题的分解能力,并探索了利用模型自身反馈提升推理性能的潜在路径.
3. MathGenie: Generating Synthetic Data with Question Back-translation for Enhancing Mathematical Reasoning of LLMs
• 通过迭代解决方案增强、问题回译和验证过滤生成多样可靠的数学问题及代码集成解决方案,并基于此数据微调预训练模型,获得高性能的MathGenieLM系列模型.
4. MetaMath: Bootstrap Your Own Mathematical Questions for Large Language Models
• 通过重写技术(答案增强、问题改写、逆向问题增强)来增加训练数据集中的数学问题多样性,并在新的数据集MetaMathQA上微调模型.
5. WizardMath: Empowering Mathematical Reasoning for Large Language Models via Reinforced Evol-Instruct
• 从上行演化和下行演化角度出发,合成具有各种复杂性和多样性的数学指令,以增强大语言模型的数学能力.
6. JiuZhang3.0: Efficiently Improving Mathematical Reasoning by Training Small Data Synthesis Models
• 提出一种低成本知识蒸馏数据集构建方法,通过教育阶段提示词和梯度相似性筛选高价值文本,利用GPT-4生成数据训练小模型DeepSeekMath-7B合成高质量数学问题,并预训练JiuZhang3.0模型.
7. Unleashing Reasoning Capability of LLMs via Scalable Question Synthesis from Scratch
• 首先通过QFT和QPO训练出一个问题生成器,然后利用这些生成器进行开放式问题采样,并对生成的问题进行进一步筛选,最终基于筛选出的高质量问题生成相应的解答.
8. Augmenting Math Word Problems via Iterative Question Composing
• 通过整合预处理的高质量网络数据与多策略增强合成数据(包括GPT-4提示生成、迭代问题组合和拒绝采样),并基于此微调多个基础模型,显著提升数学推理能力. 代码生成
1. Self-play with Execution Feedback: Improving Instruction-following Capabilities of Large Language Models
• 通过自我指导和执行反馈来自动化地生成和验证指令遵循训练数据,进而通过SFT或基于人类反馈的RLHF来提升LLMs的性能.
2. AutoCoder: Enhancing Code Large Language Model with AIEV-Instruct
• 通过双代理交互模拟编程测试流程生成高质量代码数据集,结合教学与自学习阶段减少闭源模型依赖,并训练出AutoCoder系列模型.
3. CodeRL: Mastering Code Generation through Pretrained Models and Deep Reinforcement Learning
• 通过强化学习的actor-critic架构(利用预训练模型生成代码样本并由错误预测器评估),结合推理阶段的单元测试信号优化与程序修复机制,显著提升程序合成能力.
4. Genetic Instruct: Scaling up Synthetic Generation of Coding Instructions for Large Language Models
• 基于进化算法的交叉变异操作和LLM驱动的适应度评估,从少量初始样本生成高质量编码指令数据集,并以此训练开源模型.
5. WizardCoder: Empowering Code Large Language Models with Evol-Instruct
• 提出代码专用Evol-Instruct方法,通过进化提示优化和代码调试约束生成复杂指令数据,微调StarCoder得到WizardCoder模型.
6. Magicoder: Empowering Code Generation with OSS-Instruct
• 通过开源代码片段启发生成多样化、高真实性的代码指令数据,结合Evol-Instruct微调得到Magicoder系列模型.
7. SelfCodeAlign: Self-Alignment for Code Generation
• 通过从开源代码库提取编程概念生成多样化任务,结合自生成响应与测试用例验证构建高质量指令数据,无需人工标注或蒸馏闭源模型,训练出完全透明的StarCoder2-Instruct模型.
