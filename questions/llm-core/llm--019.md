# 大模型推理的输入和输出是什么?Temperature 参数有什么作用

- **推理 (Inference)** 是将输入文本送入模型,生成输出的过程.

- **输入**:一段文本序列(Token IDs)
- **输出**:模型预测的下一个 Token(逐个生成)

- **Temperature 参数**
  - **Temperature = 0**:每次选概率最高的 token(确定性输出)
  - **Temperature = 1**:按原始概率分布采样(有创造性)
  - **Temperature > 1**:概率分布更平坦(更随机,可能不连贯)

- **实用建议**
  - 代码生成/事实问答:Temperature = 0~0.3
  - 创意写作/头脑风暴:Temperature = 0.7~1.0
  - 超过 1.2 通常不建议

- **一句话理解**:Temperature 低 = 严谨(选最优),Temperature 高 = 创意(加点随机性).

**## 常见考点**
1. **Top-k 和 Top-p 采样**：除了 Temperature，这两个参数也至关重要。Top-k 限制只从概率最高的 k 个词中选；Top-p (Nucleus Sampling) 累积概率达到 p 时截断，两者常结合使用。
2. **自回归生成**：解释推理过程中“上一个 token 的输出作为下一个时间步的输入”的循环过程，以及为什么推理速度比训练慢（无法并行计算）。
3. **KV Cache**：为了加速推理，缓存每个位置的 Key 和 Value 向量，避免重复计算，是推理优化的核心技术。
