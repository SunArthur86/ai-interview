# 大模型出现幻觉的原因

大模型采用逐Token生成的自回归方式，当输出序列较长时，中间出现错误概率增加。由于生成过程无法回溯修改，模型为保持上下文连贯性，会基于前序错误Token继续编造内容，导致错误累积并产生幻觉。
