61. 手撕多头注意力机制（不用Pytorch）（Multi Head Attention）
在面试中要求“手撕代码”，有时面试官要求不依赖高层次的库函数（例如PyTorch中的
nn.Linear 等），而是从头实现出多头注意力机制的功能。
代码实现
import numpy as np
# 定义softmax函数
def softmax(x):
    e_x = np.exp(x - np.max(x, axis=-1, keepdims=True))  # 稳定性处理，避免溢出
    return e_x / np.sum(e_x, axis=-1, keepdims=True)
# 单头注意力机制
def scaled_dot_product_attention(q, k, v, mask=None):
    d_k = q.shape[-1]  # d_k = Q的最后一个维度
    scores = np.dot(q, k.T) / np.sqrt(d_k)  # 计算 Q*K^T / sqrt(d_k)
    if mask is not None:
        scores = np.where(mask == 0, -1e9, scores)  # 使用mask对scores进行填充
    # 计算注意力权重
    attention_weights = softmax(scores)
    # 计算最终输出
    output = np.dot(attention_weights, v)  # 将注意力权重与V相乘
    return output, attention_weights

# 多头注意力机制
class MultiHeadAttention:
    def __init__(self, heads, d_model):
        self.d_model = d_model  # 输入embedding的维度
        self.h = heads  # 头的数量
        self.d_k = d_model // heads  # 每个头的维度
        # 初始化Q, K, V的权重矩阵（简单起见用随机初始化）
        self.q_linear = np.random.rand(d_model, d_model)
        self.k_linear = np.random.rand(d_model, d_model)
        self.v_linear = np.random.rand(d_model, d_model)
        # 输出的全连接层
        self.out_linear = np.random.rand(d_model, d_model)
    def split_heads(self, x, batch_size):
        """
        将输入拆分为多个头。将每个输入的 d_model 维度拆成 heads 个部分，每个部分的维度
是 d_k。
        x: 输入是 (batch_size, seq_len, d_model) 形状的张量。
        """
        x = x.reshape(batch_size, -1, self.h, self.d_k)  # 形状调整为 
(batch_size, seq_len, heads, d_k)
        x = x.transpose(0, 2, 1, 3)  # 交换维度 (batch_size, heads, seq_len, 
d_k)
        return x
    def forward(self, q, k, v, mask=None):
        batch_size = q.shape[0]
        # 1. 线性投影：将输入线性映射到Q, K, V空间
        q = np.dot(q, self.q_linear)  # 形状变为 (batch_size, seq_len, d_model)
        k = np.dot(k, self.k_linear)  # 形状变为 (batch_size, seq_len, d_model)
        v = np.dot(v, self.v_linear)  # 形状变为 (batch_size, seq_len, d_model)
        # 2. 拆分多个头
        q = self.split_heads(q, batch_size)  # (batch_size, heads, seq_len, 
d_k)
        k = self.split_heads(k, batch_size)  # (batch_size, heads, seq_len, 
d_k)
        v = self.split_heads(v, batch_size)  # (batch_size, heads, seq_len, 
d_k)
        # 3. 对每个头计算单头注意力
        outputs = []
        for i in range(self.h):  # 对每个注意力头进行单独计算
            output, _ = scaled_dot_product_attention(q[:, i], k[:, i], v[:, 
i], mask)  # 每个头独立计算注意力
            outputs.append(output)
        # 4. 拼接多个头的输出

逐行解释
Softmax函数
- softmax  函数计算分数的归一化权重，确保注意力分数按比例作用于值向量。np.max(x)  用
来避免溢出问题。
scaled_dot_product_attention 函数
◦
这是单个头的注意力计算过程。
◦
q  和 k.T  矩阵相乘，生成注意力分数，并使用 np.sqrt(d_k)  进行缩放。
◦
如果有 mask ，它会用极小值填充 mask  为 0 的位置。
◦
使用 softmax  函数计算注意力权重。
◦
最终，注意力权重与 v  相乘，得到输出。
MultiHeadAttention 类
◦
初始化时，我们定义了 q_linear 、k_linear  和 v_linear ，这三个随机矩阵用于将
输入线性变换为 Q、K、V。
◦
out_linear  是多头输出的最终线性变换。
