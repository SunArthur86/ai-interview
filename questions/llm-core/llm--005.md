# SimCSE 的核心思想是什么?它如何解决句子嵌入的各向异性问题

- **SimCSE (Simple Contrastive Learning of Sentence Embeddings)**

- **背景问题:各向异性 (Anisotropy)**
- 预训练 BERT 的句向量分布在一个窄锥形空间中,所有向量挤在一个方向
- 导致余弦相似度区分度低:任意两句话的相似度都偏高

- **SimCSE 核心思想:通过对比学习拉开向量分布**

- **无监督 SimCSE**
- 正样本:同一句子两次 dropout 前向传播(Dropout 引入微小差异)
- 负样本:batch 内其他句子
- Loss = -log( exp(sim(x,x+)) / Σ exp(sim(x,x_i)) )

- **有监督 SimCSE**
- 正样本:NLI 数据集中的蕴含句子对
- 负样本:矛盾句子对(hard negative)

- **为什么有效**
1. 对比学习将向量推向均匀分布 → 缓解各向异性
2. Dropout 作为最小化增强,保持语义不变
3. 学习到的不变特征更有泛化力

- **效果**:STS-B 等语义相似度任务上大幅提升(BERT: 0.48 → SimCSE: 0.81)
