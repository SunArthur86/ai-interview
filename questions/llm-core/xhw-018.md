28. 介绍一下FlashAttention的原理
🍊在深度学习领域，Transformer模型凭借其强大的性能，成为自然语言处理和计算机视觉等领
域的主流架构。Transformer 的核心组件是自注意力机制（Self-Attention），然而，传统的
自注意力计算在处理长序列时，面临着计算量大和内存占用高的问题。FlashAttention是一
种高效的自注意力计算算法，旨在解决这些问题。

28.2 FlashAttention 的原理
问题的本质
- 内存瓶颈：GPU 的内存带宽限制了自注意力的计算速度。
- 访存开销：传统算法需要多次读取和写入 GPU 内存，导致大量的访存操作。
FlashAttention 的核心思想
FlashAttention 通过以下方式优化自注意力计算：
- 块处理（Blocking）：将计算划分为小块，逐块处理，减少一次性需要的内存。
- IO 感知（IO-Awareness）：优化内存访问模式，最大化数据在高速缓存中的使用，减少访存开
销。
- 数值稳定性：在块内进行归一化，避免数值溢出。
