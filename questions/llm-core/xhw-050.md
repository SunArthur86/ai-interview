# 什么是交叉熵损失函数

### 交叉熵损失函数

#### 1. 基本定义
交叉熵是信息论中的一个概念，用于衡量两个概率分布之间的差异。在深度学习中，它常被用作分类任务的损失函数，用来度量模型预测的概率分布与真实标签分布之间的距离。

#### 2. 公式表达
假设真实标签分布为 $q$（通常是 One-hot 编码），模型预测分布为 $p$，则交叉熵 $H(q, p)$ 定义为：
$$ H(q, p) = - \sum_{i} q_i \log(p_i) $$

对于分类任务，若样本的真实类别为 $y$，则 $q_y = 1$，其余 $q_i = 0$，公式简化为：
$$ L = - \log(p_y) $$

#### 3. 为什么使用交叉熵？
*   **直观含义**：最小化交叉熵等价于最大化真实类别发生的对数似然概率。即让模型对正确类别的预测概率 $p_y$ 尽可能接近 1。
*   **梯度特性**：相比于均方误差（MSE），交叉熵对权重参数的梯度受误差大小的影响更平滑，且在预测错误时能提供更大的梯度，从而加快收敛速度，避免梯度消失问题。

#### 4. 大模型中的应用
在大模型（如 GPT 系列）的训练中，交叉熵损失函数用于**自监督学习**阶段：
*   **任务设定**：Next Token Prediction（下一个词预测）。
*   **计算过程**：模型基于上文预测下一个 Token 的概率分布。真实标签是语料中实际的下一个 Token（即 One-hot 向量）。计算预测分布与真实标签的交叉熵。
*   **目标**：通过最小化所有训练样本上的平均交叉熵，让模型学会预测出最符合人类语言习惯的下一个词。

#### 5. 关键点
*   信息熵 $H(q)$ 衡量分布 $q$ 本身的不确定性。
*   KL 散度 $D_{KL}(q||p)$ 衡量用 $p$ 近似 $q$ 的信息损耗。
*   关系：$H(q, p) = H(q) + D_{KL}(q||p)$。由于 $H(q)$ 对于给定数据是常数，最小化交叉熵等价于最小化 KL 散度。
