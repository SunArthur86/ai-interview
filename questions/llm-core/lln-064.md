# ControlNet 的原理是什么?它如何实现可控图像生成

### ControlNet 原理

**ControlNet** 是控制 Diffusion 模型生成过程的条件控制网络。

### 核心原理
1. **复制结构**：复制预训练 UNet 的 encoder，创建一个可训练的副本。
2. **条件输入**：副本接收额外条件输入（边缘图、深度图、姿态等）。
3. **零卷积注入**：副本输出通过 zero convolution 注入回原模型。

### Zero Convolution 关键
- 初始权重为零。
- 训练开始时不影响原模型，逐步学习如何利用条件信息。
- 保证添加 ControlNet 不会破坏原模型能力（即保护原模型的预训练权重）。

### 支持的输入类型
- Canny 边缘检测
- 深度图
- 人体姿态
- 分割图
- 涂鸦

### 优势
只需少量数据微调即可精确控制生成结果（如构图、姿态、风格）。
