# CLM和MLM分别是什么,有什么区别

1. CLM(Causal Language Modeling): **定义：** CLM 是因果语言建模,训练模型预测一句话中的下一个词.每个词只能利用它前面的上 下文信息. **应用：** 它用于生成任务,例如 GPT 模型,模型按顺序生成句子的每一个单词或字符. **训练方式：** 在训练过程中,模型只能看到到当前单词之前的所有词,不能看到之后的词.这是 一种自回归模型.
2. MLM(Masked Language Modeling): **定义：** MLM 是掩码语言建模,训练模型预测在一个句子中被随机掩盖(masked)的词.模型 可以使用整个句子的上下文,包括掩盖词的前后内容. **应用：** 它主要用于表示学习任务,例如 BERT 模型,帮助模型更好地理解整个句子的语义. **训练方式：** 在训练过程中,句子中的一些词会被随机掩盖,模型的任务是根据其余词来预测被 掩盖的词.这是一种双向建模.
3. 主要区别:
• **预测目标：** CLM 只预测下一个词,而 MLM 是填补被掩盖的词. • **上下文使用：** CLM 只能使用之前的词,MLM 则可以使用整个句子的上下文(前后词). • **模型类型：** CLM 是自回归的,用于生成任务,MLM 是双向的,用于表示学习任务. • **来源：** “因果语言建模”就是上一章中说的统计语言模型,只使用前面的词来预测当前词,由 NNLM 首次运用;而“遮盖语言建模”实际上就是 Word2Vec 模型提出的 CBOW.
