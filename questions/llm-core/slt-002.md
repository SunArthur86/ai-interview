# SFT、RLHF和DPO有什么区别？

SFT（Supervised Fine-Tuning）：
- 监督微调，用高质量的指令-回答对训练
- 让模型从'续写模式'转为'指令回答模式'
- 数据格式：{"instruction": "...", "output": "..."}
- 是对齐的第一步

RLHF（Reinforcement Learning from Human Feedback）：
- 基于人类反馈的强化学习
- 流程：训练Reward Model → 用PPO优化策略模型
- Reward Model从人类标注的偏好对(A preferred over B)中学习
- PPO训练复杂、不稳定、需要同时维护4个模型（Policy、Ref、Reward、Critic）

DPO（Direct Preference Optimization）：
- 直接偏好优化，不需要Reward Model
- 数学证明：最优Reward可以直接从Policy推导
- 损失函数直接用偏好对训练Policy
- 流程简化：只需要Policy和Ref模型
- 训练更稳定，资源消耗少

关系：SFT → RLHF/DPO。先SFT让模型会对话，再RLHF/DPO对齐人类偏好。
