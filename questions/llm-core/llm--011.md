# 什么是具身智能(Embodied AI)?LLM 如何赋能机器人

- **具身智能**:AI 拥有'身体'(机器人/虚拟形象),通过感知-决策-执行的闭环与物理世界交互.

- **LLM 在具身智能中的角色**

1. **高级任务规划**
- 用户:'帮我泡杯咖啡'
- LLM 拆解:1.走到厨房 → 2.找到杯子 → 3.操作咖啡机 → 4.倒水 → 5.端回来

2. **代码生成控制**
- LLM 直接生成机器人控制代码(Python/C++)
- 如 Google RT-2:视觉 → LLM → 动作 token

3. **常识推理**
- '杯子掉在地上会碎' → 规划时避免
- '水太烫要等一下' → 执行时调整

- **核心挑战**
1. **感知-动作鸿沟**:文本/图像是离散的,动作是连续的(关节角度、力大小)
2. **实时性**:物理世界不能等 LLM 推理 5 秒
3. **安全约束**:错误动作可能造成物理损坏
4. **数据稀缺**:真实机器人交互数据采集成本极高

- **代表项目**
- **RT-2 (Google)**:VLM 直接输出机器人动作
- **NVIDIA GR00T**:人形机器人的基础模型
- **Figure 01/02**:OpenAI 支持的人形机器人
- **Tesla Optimus**:端到端神经网络控制

- **趋势**:从'仿真训练+真实部署'(Sim2Real) 到'真实数据+大规模训练'.
