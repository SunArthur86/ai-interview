# 介绍一下Qwen通义千问的结构

通义千问是阿里巴巴研发的大型语言模型系列。其核心架构基于 Transformer，主要包含以下关键技术点：

**1. 整体架构**
*   **基础组件**：采用 Decoder-only 的 Transformer 架构。
*   **位置编码**：使用 RoPE (Rotary Positional Embeddings) 增强位置信息的感知。
*   **激活函数**：使用 SwiGLU 替代传统的 ReLU 或 GeLU，提升模型表达能力。
*   **归一化**：采用 RMSNorm (Root Mean Square Layer Normalization)，提升训练稳定性。

**2. 词汇表与分词**
*   **词汇表大小**：约 152K，包含大量中文字符和多语言词汇，提高了中文和多语言的压缩效率，降低了推理成本。
*   **分词器**：基于 BPE (Byte Pair Encoding)。

**3. 训练方法**
*   **预训练**：使用了高达 3T tokens 的高质量多语言数据。
*   **长上下文**：通过 NTK-aware 插值、LogN-Scaling 等技术扩展上下文长度，支持长文本处理。
*   **对齐**：采用了 SFT (监督微调) 和 RLHF (人类反馈强化学习) 技术，特别是使用了 DPO (直接偏好优化) 进行对齐训练。

**4. 特殊设计**
*   **解耦嵌入**：输入嵌入和输出投影的权重不共享。
*   **QKV Bias**：在注意力的 QKV 层中添加偏置，增强模型外推能力。
