# 大模型的训练过程分为哪几个阶段?每个阶段的目标是什么

- **大模型训练三阶段**

1. **预训练 (Pre-training)**
- 目标:从海量文本中学习语言规律和世界知识
- 数据:万亿级 token(网页、书籍、代码)
- 方法:预测下一个 token(next-token prediction)
- 产出:基座模型(Base Model)

2. **指令微调 (SFT)**
- 目标:让模型学会遵循人类指令
- 数据:高质量指令-回答对(几万到百万条)
- 产出:对话模型(Chat Model)

3. **对齐 (RLHF / DPO)**
- 目标:让回答更安全、有用、符合人类偏好
- 数据:偏好排序数据(好回答 vs 差回答)
- 产出:对齐模型(Aligned Model)

- **一句话理解**:预训练→学知识,SFT→学指令,RLHF→学品味.
