# 什么是涌现能力（Emergent Abilities）？

涌现能力指模型规模（参数量、训练数据、计算量）达到某个阈值后突然出现的能力，在小模型中不存在。

典型涌现能力：
1. In-context Learning（上下文学习）：GPT-3（175B）展示的few-shot能力
2. Chain-of-Thought（思维链推理）：~60B参数以上出现
3. 指令遵循：需要一定规模才具备
4. 代码生成：需要专门的代码训练数据+足够规模
5. 多语言能力

争议：
- 2023年论文《Are Emergent Abilities a Mirage?》指出，涌现可能是评估指标（如exact match）的非线性导致。用连续指标（如token-level probability）则曲线平滑
- 但实践中，大模型确实'感觉'更聪明

结论：某些能力是渐进的，某些可能确实有相变点（如in-context learning）。
