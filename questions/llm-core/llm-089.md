# 怎么理解DPO的损失函数

DPO(Direct Preference Optimization)是一种用于优化语言模型(LLMs)的算法,旨在通过直接优化人类偏好数据来训练模型.DPO 的核心思想是简化传统的 RLHF(Reinforcement Learning from Human Feedback)过程,避免复杂的奖励模型训练,直接通过偏好数据来优化策略.
95. 1 DPO 损失函数的基本形式
🫐DPO 的损失函数定义如下:

**DPO**
$$L_{DPO}(\pi_\theta; \pi_{ref}) = -\mathbb{E}_{(x,y_w,y_l)\sim D} \left[ \log \sigma \left( \beta \left( \log \frac{\pi_\theta(y_w|x)}{\pi_{ref}(y_w|x)} - \log \frac{\pi_\theta(y_l|x)}{\pi_{ref}(y_l|x)} \right) \right) \right]$$

*   $\pi_\theta$: 我们要训练的模型,目标是让它学会根据人类的偏好生成更好的输出.
*   $\pi_{ref}$: 参考模型,通常是一个预训练的语言模型,它的行为是固定的,用来“约束”策略模型,防止策略模型偏离正常行为太远.
*   $y_w$: 人类偏好的输出.
*   $y_l$: 人类不喜欢的输出.
*   $D$: 包含偏好数据的数据集,每个样本由 $(x, y_w, y_l)$ 组成.
*   $\beta$: 超参数,控制奖励的缩放.

DPO Loss = -log \sigma(\beta \cdot (log \pi(y_w|x)/\pi_{ref}(y_w|x) - log \pi(y_l|x)/\pi_{ref}(y_l|x)))
通过计算这两个比率的对数差值,我们可以衡量策略模型在偏好数据上的表现.如果策略模型更倾向于选择 $y_w$ 而不是 $y_l$,那么这个差值会变大,损失函数会变小.

**策略模型和参考模型的相除是什么意思?**
公式中的 $\frac{\pi_\theta(y_w|x)}{\pi_{ref}(y_w|x)}$ 和 $\frac{\pi_\theta(y_l|x)}{\pi_{ref}(y_l|x)}$ 是关键部分.它们的含义是:
*   $\pi_{ref}(y_w|x)$: 参考模型认为 $y_w$ 是输入 $x$ 的合理输出的概率.
*   比率: 策略模型对 $y_w$ 的偏好程度 相对于 参考模型对 $y_w$ 的偏好程度.

**这个比率的意义是:**
*   如果策略模型比参考模型更喜欢 $y_w$,那么这个比率会大于 1.
*   如果策略模型比参考模型更不喜欢 $y_w$,那么这个比率会小于 1.

同理, $\frac{\pi_\theta(y_l|x)}{\pi_{ref}(y_l|x)}$ 表示策略模型对 $y_l$ 的偏好程度相对于参考模型.

**为什么要用这个比率?**
DPO 的目标是让策略模型更倾向于选择 $y_w$,而不是 $y_l$.通过计算这两个比率的差值:
$$\log \frac{\pi_\theta(y_w|x)}{\pi_{ref}(y_w|x)} - \log \frac{\pi_\theta(y_l|x)}{\pi_{ref}(y_l|x)}$$
我们可以衡量策略模型对 $y_w$ 和 $y_l$ 的偏好差异.如果策略模型更喜欢 $y_w$,这个差值会变大;如果策略模型更喜欢 $y_l$,这个差值会变小.

95. 3 损失函数的进一步解释
🫐为了更清楚地理解损失函数,我们可以将其分解为以下几个部分:
1. **Logistic 函数的作用:** 函数 $\sigma$ 将差值映射到 $(0, 1)$ 之间,表示策略模型选择 $y_w$ 而不是 $y_l$ 的概率.
2. **超参数 $\beta$ 的作用:** 控制策略模型与参考模型之间的差异.较大的 $\beta$ 会使策略模型更倾向于偏离参考模型,而较小的 $\beta$ 会使策略模型更接近参考模型.
3. **参考模型的作用:** 参考模型 $\pi_{ref}$ 用于约束策略模型的行为,防止策略模型过度偏离预训练的语言模型.这样可以避免模型生成不合理或不安全的输出.

95. 4 损失函数的简化形式
🫐为了进一步简化损失函数,我们可以对公式进行如下转换:
利用对数的性质,可以将其改写为:
$$\beta (\log \pi_\theta(y_w|x) - \log \pi_{ref}(y_w|x)) - \beta (\log \pi_\theta(y_l|x) - \log \pi_{ref}(y_l|x))$$
这个形式与源代码中的计算函数一致.其中:
*   左半部分表示策略模型对 $y_w$ 的偏好优势.
*   右半部分表示策略模型对 $y_l$ 的偏好优势.

损失函数的最终目标
DPO 的最终目标是最大化策略模型对偏好数据的偏好程度,同时保持策略模型与参考模型之间的差异在可控范围内.通过这种方式,DPO 能够有效地学习人类偏好,并生成符合人类期望的输出.
