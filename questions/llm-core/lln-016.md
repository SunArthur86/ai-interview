# 如何提升合成数据的多样性

通过明确告诉LLM生成文本的具体维度和要求来提升多样性。具体做法：1. 向ChatGPT询问生成数据时需要考虑的维度；2. 人工筛选必要的维度；3. 让ChatGPT给出每个维度的值集合；4. 再次人工筛选必要的值并去除歧义。核心代码通过生成不同维度的提示词和过滤低质量数据来实现。
