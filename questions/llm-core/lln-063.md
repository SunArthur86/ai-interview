# Sora 2 的核心技术突破是什么?它是如何实现物理规律建模的

### Sora 2 的核心技术突破

**Sora 2** 是 OpenAI 发布的视频生成模型，核心突破包括：

#### 1. 物理规律建模
- 不再是简单帧拼接，而是理解物理世界（重力、碰撞、光影）。
- 通过大规模视频-文本对学习物理规律的隐式表示。

#### 2. 视音频同步生成
- 视频画面与音频同时生成，而非后期合成。
- 支持多镜头连续叙事。

#### 3. 技术架构
- 基于 Diffusion Transformer (DiT) 架构。
- 使用 Spatiotemporal Patch 将视频转为 token 序列。
- 采用 Flow Matching 替代传统 DDPM，提升采样效率。

#### 4. 关键能力
- Cameo 肖像植入保持角色一致性。
- 最长 60 秒连贯视频。
- 复杂场景理解与物理交互模拟。
