# Sora 2 的核心技术突破是什么?它是如何实现物理规律建模的

### Sora 2 的核心技术突破

**Sora 2** 是 OpenAI 发布的新一代视频生成模型，核心突破在于对物理世界的深层理解和高效的生成架构。

#### 1. 物理规律建模 (World Simulator)
*   **3D 一致性**：不再是简单的 2D 帧间插值，而是隐式构建了 3D 空间结构。当摄像机移动时，场景中的物体透视关系正确变化。
*   **交互物理**：模型学习了物体间的物理交互规律（如重力、碰撞、流体动力学、光影反射），能够模拟“一只猫打翻玻璃杯，液体洒出”的复杂因果链。
*   **长期连贯性**：通过大规模数据训练，模型能维持长达 60 秒的状态记忆，避免物体随时间推移发生变形或消失（如“吃汉堡不减少”的问题被解决）。

#### 2. 视音频同步生成
*   **原生多模态**：不再是先生成视频再配音（TTS），而是在统一的潜在空间中联合预测视频帧和声谱图。
*   **语义对齐**：音频的节奏、音效（如脚步声、爆炸声）与画面动作精确对齐。

#### 3. 技术架构
*   **Diffusion Transformer (DiT)**：
    *   结合了 U-Net 的多尺度特征提取能力和 Transformer 的长程建模优势。
    *   相比传统 ViT，DiT 更适合处理噪声预测的去扩散过程。
*   **Spatiotemporal Patches (时空补丁)**：
    *   将视频在时间和空间维度上切分为小块，将视频数据转化为类似于语言 Token 的序列。
    *   使得模型能够像处理文本一样处理视频，实现可变的分辨率和时长。
*   **Flow Matching (Rectified Flow)**：
    *   替代了传统的 DDPM/DDIM 采样。
    *   **原理**：将噪声分布到数据分布的映射学习为一条近似直线的轨迹。
    *   **优势**：采样步数更少（推理速度更快），生成路径更直，生成的细节更锐利。

#### 4. 关键能力
*   **Cameo (角色一致性)**：通过参考图像锁定角色特征，在任意场景下保持高保真。
*   **多镜头叙事**：理解复杂的运镜指令（如“从远景推到特写”），自动生成平滑的转场。

### ## 常见考点
1.  **DiT (Diffusion Transformer) 相比传统 U-Net 有什么优势？**（追问：Transformer 的 Scaling Laws、全局注意力带来的长程一致性、更强的归纳偏置灵活性）
2.  **Flow Matching 和 DDPM 的区别？**（追问：Flow Matching 是 ODE（常微分方程）过程，求解路径更简单，采样效率更高，不仅是去噪而是直推）
3.  **Sora 如何实现“物理世界模拟”？**（追问：并不具备物理引擎，而是通过海量视频数据学习到的“视觉物理”经验统计规律，讨论其局限性）
