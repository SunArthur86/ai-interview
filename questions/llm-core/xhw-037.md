# 手撕多头注意力(用Pytorch)

总结
- 手撕自注意力机制：在面试中，要求手撕代码是指从头实现一个功能，而不依赖于现成的高层次
库。在这里，我们手动实现了自注意力机制，涉及到矩阵相乘、转置、点积、softmax 等核心操
作。
- 主要步骤：包括生成查询、键和值矩阵，计算相似度分数，使用 softmax 归一化权重，并根据注意
力权重加权 Value 。
60. 手撕多头注意力机制（用Pytorch）（Multi Head Attention）
多头注意力机制（Multi-Head Attention）是 Transformer 模型中的核心组件，用于捕捉不同位置之
间的依赖关系。通过将注意力计算分成多个头部，通过多个头，模型可以在不同的投影空间中计算注
意力，最终将各个头的输出拼接，并通过线性层进行整合。具体过程为：

MultiHead(Q, K, V ) = Concat(head1, … , headh)W

import torch
import torch.nn as nn
import torch.nn.functional as F
import math
class MultiHeadAttention(nn.Module):
    def __init__(self, heads, d_model, dropout=0.1):
        super().__init__()
        # 保存输入的d_model（模型维度）和头的数量
        self.d_model = d_model
        # d_k是每个头的特征维度，通常是d_model / heads
        self.d_k = d_model // heads
        self.h = heads
        # 三个线性变换层，分别用于将输入映射为Q、K、V
        self.q_linear = nn.Linear(d_model, d_model)  # 线性层，映射输入到Q
        self.v_linear = nn.Linear(d_model, d_model)  # 线性层，映射输入到V
        self.k_linear = nn.Linear(d_model, d_model)  # 线性层，映射输入到K
        # Dropout层，用于防止过拟合
        self.dropout = nn.Dropout(dropout)
        # 输出的线性层，最终将多头输出的结果合并回原始d_model维度
        self.out = nn.Linear(d_model, d_model)
    def attention(self, q, k, v, d_k, mask=None, dropout=None):
        # 1. 计算注意力分数（Q*K^T / sqrt(d_k)）
        # 其中q的维度是 (batch_size, heads, seq_len, d_k)
        # k的维度是 (batch_size, heads, seq_len, d_k)
        # 计算的结果是 (batch_size, heads, seq_len_q, seq_len_k)
        scores = torch.matmul(q, k.transpose(-2, -1)) / math.sqrt(d_k)

        # 2. 如果有mask，则将mask的部分填充为一个极小的值，表示这些部分不计算注意力
        if mask is not None:
            mask = mask.unsqueeze(1)  # 扩展mask的维度以适配多头的操作
            scores = scores.masked_fill(mask == 0, -1e9)  # 将mask位置设为极小值
        # 3. 使用softmax计算权重，使得权重总和为1，表示每个位置对其他位置的相关性
        scores = F.softmax(scores, dim=-1)
        
        # 4. 如果有dropout层，应用于注意力权重上
        if dropout is not None:
            scores = dropout(scores)
        
        # 5. 使用注意力权重对V矩阵进行加权求和
        # q和k的相似性决定了v的加权方式，输出的维度是 (batch_size, heads, 
seq_len_q, d_k)
        output = torch.matmul(scores, v)
        return output
    def forward(self, q, k, v, mask=None):
        bs = q.size(0)  # 获取batch_size大小
        
        # 1. 对Q、K、V分别进行线性变换
        # 输入q, k, v的维度为 (batch_size, seq_len, d_model)
        # 通过线性层后，维度依旧为 (batch_size, seq_len, d_model)
        k = self.k_linear(k)  # 线性变换后的k
        q = self.q_linear(q)  # 线性变换后的q
        v = self.v_linear(v)  # 线性变换后的v
        # 2. 维度变换：将d_model拆分为多个头 (heads)
        # 将Q、K、V重新reshape为 (batch_size, seq_len, heads, d_k)
        # 然后再交换维度，得到 (batch_size, heads, seq_len, d_k)
        k = k.view(bs, -1, self.h, self.d_k).transpose(1, 2)
        q = q.view(bs, -1, self.h, self.d_k).transpose(1, 2)
        v = v.view(bs, -1, self.h, self.d_k).transpose(1, 2)
        # 3. 计算注意力，返回的 scores 维度为 (batch_size, heads, seq_len_q, d_k)
        scores = self.attention(q, k, v, self.d_k, mask, self.dropout)
        # 4. 把多头的输出拼接起来
        # 先将 (batch_size, heads, seq_len, d_k) 变回 (batch_size, seq_len, 
heads, d_k)
        # 然后将多个头拼接为原始的 (batch_size, seq_len, d_model)
        concat = scores.transpose(1, 2).contiguous().view(bs, -1, self.d_model)
        # 5. 通过输出线性层，再次映射回 (batch_size, seq_len, d_model)
        output = self.out(concat)
        return output

维度变化示例
假设 batch_size=2 , seq_len=10 , d_model=512 , heads=8 ，那么：
1. 输入的 q, k, v  维度是 (2, 10, 512) 。
2. 经过线性层后，q, k, v  的维度仍然是 (2, 10, 512) 。
3. 使用 .view(2, 10, 8, 64)  将其拆分为8个头，每个头大小为64 ( d_k=64 )，变成 (2, 
8, 10, 64) 。
4.
transpose(1, 2)  后维度变为 (2, 8, 10, 64) ，以便每个头独立进行计算。
5. 计算注意力后的结果维度是 (2, 8, 10, 64) 。
6. 最后通过 concat  和 out  映射回原始的 (2, 10, 512) 。
