# 什么是Medusa解码？它与传统的投机解码相比有什么独特优势？

Medusa解码是一种非自回归的并行解码加速技术，通过在LLM的主头之外增加多个额外的“输出头”来预测未来的多个token。与传统投机解码使用一个小模型来草拟token不同，Medusa的这些头是主模型参数的延伸，共享底层Transformer，通过微调训练得到。推理时，Medusa可以一次性预测k个token，形成一个候选树，然后通过并行验证机制接受或拒绝这些token。其优势在于消除了对小Draft Model的依赖，避免了小模型能力不足导致acceptance rate（接受率）低的问题，且不需要运行两个独立的模型，部署更简单。在相同延迟下，Medusa通常能比标准的 speculative decoding 获得更高的加速比（2x-3x）。
