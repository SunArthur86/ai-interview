# DPO 的 loss 是什么

DPO（Direct Preference Optimization）的损失函数直接基于 Bradley-Terry 模型构建，旨在利用人类偏好数据（$x, y_w, y_l$）优化模型，使其倾向于生成更好的回复。其损失函数形式如下：

$$ L(\theta) = -\mathbb{E}_{(x,y_w,y_l)\sim D} \left[ \log \sigma \left( \beta \log \frac{\pi_\theta(y_w|x)}{\pi_\theta(y_l|x)} \right) \right] $$

其中：
- $\sigma$ 是 Sigmoid 函数。
- $\pi_\theta(y|x)$ 是模型生成的概率。
- $\beta$ 是温度系数（通常设为 0.1）。
- $y_w$ 是偏好回复，$y_l$ 是非偏好回复。

**原理简述**：DPO 并没有显式训练奖励模型，而是通过数学推导，将奖励模型的差异转化为策略模型的对数概率比。最小化该损失函数意味着最大化 $y_w$ 和 $y_l$ 的对数概率差，从而直接对齐人类偏好。
