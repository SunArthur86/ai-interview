# Deepseek中蒸馏R1是什么?是否比从零RL训练是否更好

🥥在DeepSeek的研究中,蒸馏是指将大型模型(如DeepSeek-R1)的知识通过监督微调
(SFT)传递到较小的模型中.具体而言,DeepSeek团队使用DeepSeek-R1生成的数据,对
较小的模型(如Llama和Qwen系列)进行微调,使其具备类似的推理能力.
107. 1 蒸馏与从零开始的强化学习(RL)训练的比较
🍋蒸馏与从零开始的强化学习(RL)训练的比较
**性能表现：** 研究表明,蒸馏模型在推理任务中的表现优于直接通过RL训练的小模型.例如,
DeepSeek-R1-Distill-Qwen-7B在AIME 2024基准测试中.÷÷超越了QwQ-32B-Preview.
**计算成本：** 从零开始对小模型进行RL训练需要巨大的计算资源.相比之下,蒸馏策略更为经
济高效,能够在较小的模型上实现出色的推理能力.
**基础模型质量：** 尽管蒸馏策略有效,但其效果仍取决于基础模型的质量.例如,基于1.5B参
数的模型与7B参数的模型相比,后者在蒸馏后的性能更佳.
107. 2 结果分析
**🌻数据质量：** 蒸馏过程中使用的高质量推理数据,使得小模型能够学习到有效的推理模式,而
无需经历RL训练中的探索过程.
**训练稳定性：** 直接对小模型进行RL训练可能导致不稳定性,而蒸馏通过SFT提供了稳定的训
练过程,减少了训练难度.
**资源效率：** 蒸馏策略在计算资源和时间成本上更具优势,特别是在资源有限的情况下.

在DeepSeek的研究中,通过蒸馏将大型模型的推理能力传递到小模型,比从零开始对小模型进行
RL训练更为有效和高效.然而,蒸馏效果仍依赖于基础模型的质量,选择合适的基础模型对于实现
最佳性能至关重要.
