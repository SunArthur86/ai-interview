# VLA（Vision-Language-Action）模型与传统的VLM相比，在架构和能力上有什么根本区别？

VLM主要关注视觉理解和语言生成，即“看图说话”；而VLA将动作也纳入模型的核心输出，实现了“感知-决策-执行”的闭环。在架构上，VLA通常在视觉编码器和语言解码器之间加入了动作投影层或直接预测机械臂的末端轨迹/控制指令。VLA的训练数据不仅包含图文对，还包含大量的轨迹数据。这使得VLA能够直接将视觉观察映射到物理动作，具备跨具身任务的泛化能力，无需针对每个新任务进行复杂的运动规划编程，是具身智能Agent的核心技术方向。
