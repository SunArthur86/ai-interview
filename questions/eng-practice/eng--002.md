# LLM 与推荐系统结合时会遇到哪些问题?有哪些可落地的方案

- **LLM + 推荐系统** 是当前大厂热门方向,但落地有显著挑战.

**常见问题：** 
1. 延迟:LLM 推理延迟远超传统推荐模型(秒级 vs 毫秒级)
2. 用户历史行为序列过长,超出上下文窗口
3. LLM 对推荐指标(CTR/CVR)的直接优化能力有限
4. 在线推理成本高,难以支撑高 QPS 场景
5. 冷启动场景数据稀疏,LLM 理解力优势有限

**可落地方案：** 
- 特征增强:LLM 生成用户兴趣标签/商品描述,补充传统特征
- 冷启动推荐:LLM 利用商品文本信息做语义匹配
- 重排序阶段:候选集缩小后用 LLM 做 fine-ranking
- 对话式推荐:多轮交互收集偏好
- 生成式推荐:LLM 直接生成推荐列表

**小红书实践：** 用 LLM 做内容理解和标签提取,传统模型做排序.
