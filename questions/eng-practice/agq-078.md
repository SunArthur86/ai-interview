# 你在项目中的角色是什么?最大的贡献是什么

384
**Situation：** 面试官考察个人在项目中的真实贡献和成长.
**Task：** 清晰表达个人角色和核心贡献.
**Action：** 
1. 角色定位: 后端技术负责人 / AI 工程师.
2. 核心贡献:
**架构设计：** 主导了多路检索融合架构和 Agent 编排器的设计.
**核心模块开发：** 实现了 ReAct 引擎、检索管道、熔断器、模型路由等核心模块.
**性能优化：** 主导了全链路延迟优化,将 P50 从 6.5s 降到 2.8s.
**工程体系建设：** 建立了 CI/CD 流水线、监控告警体系、评估测试框架.
3. 技术决策: 推动了从单 Agent 到多 Agent 的架构演进,解决了工具数量增长导致的准确率下
降问题.
4. 个人成长: 从"能用 LangChain 写 Demo"到"能设计和落地企业级 AI Agent 系统".
**Result：** 
系统稳定服务 20+ 企业客户.
团队从 3 人发展到 8 人.
个人从 IC 成长为技术负责人.
