# 如何检测多 Agent 系统的「死循环」

**组合策略**：
1. 全局步数上限；
2. 状态哈希去重（若连续重复同一计划/同一工具入参则停）；
3. 无进展检测（关键指标多轮不变，如 bug 数未降）；
4. 预算熔断（Token/费用/时间）。

**追问应对**：若问「误杀怎么办？」——答：提高进展定义粒度、允许人类确认继续。
