# CrewAI的核心概念是什么?和AutoGen有什么设计理念上的区别

- **CrewAI核心概念:**

1. **Crew** - 一组Agent的集合(团队)
2. **Agent** - 有角色、目标、工具的成员
3. **Task** - 具体任务,有描述和期望输出
4. **Process** - 任务执行模式(sequential/hierarchical)

- **典型代码:**
```python
researcher = Agent(
role="研究员",
goal="收集最新AI论文",
tools=[search_tool]
)
writer = Agent(
role="技术作者",
goal="写一篇高质量文章",
tools=[write_tool]
)

crew = Crew(
agents=[researcher, writer],
tasks=[research_task, write_task],
process=Process.sequential
)
result = crew.kickoff()
```

- **CrewAI vs AutoGen:**

| | CrewAI | AutoGen |
|--|--------|---------|
| 设计理念 | **结构化流水线** | **自由对话** |
| 控制流 | 预定义任务序列 | GroupChat动态协商 |
| 可预测性 | **高** | 中 |
| 灵活性 | 中 | **高** |
| 适合 | 明确分工的生产流 | 探索性/创意任务 |

- **选择建议:**
- 任务步骤明确 → CrewAI
- 需要Agent自由协作 → AutoGen
- 需要复杂状态管理 → LangGraph
