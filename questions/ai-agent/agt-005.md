5. Agent 的分类 
5.1 概念解释（经典 AI 教材视角） 
分类来自智能体理论，可与 LLM Agent 对照理解：
类型
通俗理解
反应式（Reactive）
类似反射：输入→动作，几乎不维护复杂内部模型。
基于模型（Model-based）
脑中有「世界模型」，能预测行动后果。
基于目标（Goal-based）
明确要达成什么状态，能选路径。
基于效用（Utility-based）
多目标权衡，选期望效用最大的行动。
学习型（Learning）
从反馈中更新策略或模型。
