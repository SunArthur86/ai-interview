# Agent 的分类

### Agent 的分类

#### 概念解释（基于智能体理论）

| 类型 | 通俗理解 | 说明 |
| :--- | :--- | :--- |
| **反应式** | 类似反射 | 输入→动作，几乎不维护复杂内部模型。 |
| **基于模型** | 脑中有地图 | 脑中有「世界模型」，能预测行动后果。 |
| **基于目标** | 明确终点 | 明确要达成什么状态，能选路径。 |
| **基于效用** | 权衡利弊 | 多目标权衡，选期望效用最大的行动。 |
| **学习型** | 自我进化 | 从反馈中更新策略或模型。 |

#### 分类应用
大多数现代 LLM Agent 结合了 **基于模型**（LLM 理解世界）、**基于目标**（Prompt 设定的目标）和 **基于效用**（RLHF 或评分机制）的特性。
