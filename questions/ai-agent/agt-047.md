# 冲突解决

多 Agent 可能对 同一问题给出矛盾结论（例如「能上线」vs「有高危漏洞」）。冲突解决机制用于
收敛到可执行决策。

5.3 面试问题 Q8 
Q8：为什么光有投票不够？
**A：**因为 LLM Agent 的「独立意见」往往 不独立（相似训练分布、相似 system 提示），且
缺少 真实世界证据 时，投票可能强化错误。更稳妥的是 证据门槛 + 优先级规则 + 人类在环。
**追问应对：**若问「Red Team 怎么用？」——答：专门 Agent 负责挑错、攻击假设、构造反
例，输出 必须回应的质疑清单。
