# 只用向量检索、不做摘要可以吗

可以,但你会失去低成本的全局叙事(例如任务阶段、未决事项).摘要擅长提供「主线」,向
量擅长提供「细节证据」.最佳实践常是摘要 + 检索并存.
**追问应对：** 若数据极结构化(工单系统),可用状态机字段替代部分摘要.
