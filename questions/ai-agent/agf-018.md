# 你如何为一个企业场景选择 ReAct vs Plan-and-Execute

看任务是否强流程、是否需要可审计计划书、是否允许前期规划成本；需要强工具交互与动态环境用 ReAct，强分解与多步骤交付用 Plan-and-Execute。
