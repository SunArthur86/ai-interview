# 你如何为一个企业场景选择 ReAct vs Plan-and-Execute

看任务是否 强流程、是否需要 可审计计划书、是否允许 前期规划成本;需要强工具交互与动
态环境用 ReAct,强分解与多步骤交付用 Plan-and-Execute.
