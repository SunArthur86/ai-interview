# 向量数据库选型:Milvus、Pinecone、Weaviate 各有什么优劣

**主流向量数据库对比：** 

- **Milvus:**
- 开源,支持私有化部署
- 支持十亿级向量
- 索引算法丰富(IVF、HNSW、DiskANN)
- 支持 Hybrid Search(向量+标量过滤)
- 运维复杂度高

- **Pinecone:**
- 全托管 SaaS,零运维
- 效果好,延迟稳定
- 支持 Serverless 模式
- 闭源,数据在云端
- 成本较高(按用量计费)

- **Weaviate:**
- 开源,支持私有化
- 内置模块化(可直接接 OpenAI/Cohere Embedding)
- 支持 GraphQL 查询
- 社区活跃度中等

- **其他选择:**
- Qdrant:Rust 编写,高性能,开源
- Chroma:轻量级,适合原型
- pgvector:PostgreSQL 扩展,适合已有 PG 的团队

- **选型建议:**
- 原型验证:Chroma 或 pgvector
- 生产部署(私有化):Milvus 或 Qdrant
- 生产部署(SaaS):Pinecone
