# 一句话说明什么是 AI Agent

AI Agent 是以大模型为认知核心，结合规划、记忆与工具调用，能在多步交互中根据环境反馈持续决策并完成任务的系统；其本质是闭环的感知-思考-行动循环，而不仅是单次文本生成。
