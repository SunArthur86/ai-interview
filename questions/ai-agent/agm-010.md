# 情景记忆和语义记忆为什么要区分存储

因为更新频率、隐私级别、检索特征不同:情景更个人化、更时间敏感;语义更共享、更稳
定.区分后可做不同保留策略(情景更易过期)、不同权限(情景多用户隔离更严格),并减少把
「一次性事件」误当「长期规则」.
