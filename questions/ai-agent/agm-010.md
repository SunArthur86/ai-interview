# 情景记忆和语义记忆为什么要区分存储

因为更新频率、隐私级别、检索特征不同：情景记忆更个人化、更时间敏感；语义记忆更共享、更稳定。区分存储后可实施不同的保留策略（情景记忆更易过期）、不同的权限控制（情景记忆的用户隔离更严格），并减少将「一次性事件」误当「长期规则」的概率。

**追问应对**：可补充说明从情景记忆归纳成语义记忆的离线任务（反思模块）设计。
