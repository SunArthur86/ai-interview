# 如何保证模型一定输出合法 JSON

**多层保障：** (1)Prompt 明确要求「仅 JSON、禁止 Markdown」;(2)用 JSON Schema 在服务端校验;(3)失败则 repair:用第二次调用让模型根据错误信息修正;或(4)用开源/库做 JSON repair;(5)关键路径用 Function Calling + 强校验.
