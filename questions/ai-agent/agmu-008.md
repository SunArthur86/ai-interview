# 为什么光有投票不够

因为 LLM Agent 的「独立意见」往往不独立（相似训练分布、相似 system 提示），且缺少真实世界证据时，投票可能强化错误。

**更稳妥的是**：证据门槛 + 优先级规则 + 人类在环。

**追问应对**：若问「Red Team 怎么用？」——答：专门 Agent 负责挑错、攻击假设、构造反例，输出必须回应的质疑清单。
