# 用人类记忆模型设计 Agent 记忆有什么好处

好处是模块化与可解释：短期对应上下文窗口与 scratchpad，长期对应向量库/图谱；感觉记忆对应流缓冲。风险是类比不能硬套——计算机没有神经可塑性，需要工程上显式实现「巩固、遗忘、冲突解决」。
