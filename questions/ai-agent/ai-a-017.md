# 为什么结构化输出对Agent至关重要?如何确保100%的格式正确率

- **结构化输出的重要性:**

Agent的每一步都依赖结构化数据:
- 工具调用参数必须是合法JSON
- 任务状态必须是预定义枚举
- Agent间通信需要结构化消息

- **确保格式正确的方法:**

1. **Function Calling / Tool Use(最可靠)**
- 模型层面保证输出合法JSON
- OpenAI/Anthropic/Gemini都支持

2. **JSON Mode**
- response_format=json_object
- 保证合法JSON但不保证schema

3. **约束解码(Constrained Decoding)**
- 在推理时强制输出符合grammar的token
- llama.cpp的GBNF / Outlines / SGLang
- **100%保证格式正确**

4. **Pydantic验证 + 重试**
```python
**try：** 
result = MyModel.parse_raw(llm_output)
except ValidationError:
# 把错误信息发回给LLM让其修正
result = retry_with_error(llm_output, str(e))
```

- **实际建议:** 优先用Function Calling,其次约束解码,最后才是重试
