# ReAct 为什么要显式写出 Thought

显式 Thought 有三类价值：可解释性（便于人类审计）、可调试性（定位哪一步选错工具）、学习信号（可做监督微调或评估中间步骤质量）。在工程上，Thought 也帮助模型在下一步更稳定地选择 Action。

格式示例：
**Question：** ...
**Thought：** ...
**Action：** Search[query]
**Observation：** ...
**Final Answer：** ...
