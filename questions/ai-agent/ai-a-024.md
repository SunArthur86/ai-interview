# MCTS(蒙特卡洛树搜索)如何与LLM结合用于Agent推理

- **MCTS + LLM = 有方向的试错推理**

- **MCTS四步骤:**
1. **选择** - 用UCB公式选择最有前景的节点
2. **扩展** - 扩展一个新子节点
3. **模拟** - 用LLM快速模拟到终点
4. **回传** - 更新路径上的胜率

- **LLM在MCTS中的角色:**

| 角色 | 作用 |
|------|------|
| 策略网络 | 生成候选动作(扩展) |
| 价值网络 | 评估节点前景(模拟) |
| 快速模拟 | Rollout到终局 |

- **优势:**
- 结合探索和利用
- 可以发现LLM直接推理找不到的非直觉路径
- 适合数学竞赛、程序合成等需要搜索的任务

- **代表工作:** rStar-Math、AlphaCodium、ToT

- **代价:** 计算量大(需要大量LLM调用),适合离线推理
