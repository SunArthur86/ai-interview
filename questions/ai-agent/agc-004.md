# ChatBot 加上插件是不是就变成 Agent 了

不一定.若插件调用由固定规则触发(例如关键词路由),更像「带工具的 Bot」.若由模型在
多步推理中自主选择工具与参数,并形成闭环迭代,则更贴近 Agent.关键在是否具备多步自主决
策与反馈闭环.
