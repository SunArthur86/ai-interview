# MemGPT 和简单 RAG 的本质区别是什么

RAG 多是被动检索；MemGPT 强调主动内存管理——模型或控制器决定何时把外部记忆换入上下文、何时写出、如何分页，更像 OS 管理 RAM。

**追问应对**：可以补充说明控制流可由函数调用或工具实现。
