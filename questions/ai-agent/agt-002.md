# 模型选错工具怎么办

工程上(1)优化 description  与示例边界;(2)工具路由缩小候选集;(3)执行前做规则
校验或二次确认;(4)对高风险操作要求人类确认;(5)记录 bad case 做 prompt 迭代.不能
假设模型 100% 正确.
