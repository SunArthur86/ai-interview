# 请用你自己的话定义 LLM Agent,并说明与单次调用的差异.

LLM Agent 是以大语言模型为推理核心，在多轮中与外部环境交互，通过规划、记忆与工具完成复杂任务的系统。与单次调用的差异在于：单次调用是开环生成；Agent 是闭环决策，每步可依据工具返回更新状态，直到终止条件。
