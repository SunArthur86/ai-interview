# 请用你自己的话定义 LLM Agent,并说明与单次调用的差异.

LLM Agent 是以大语言模型为推理核心,在 多轮 中与外部环境交互,通过 规划、记忆与工
具 完成复杂任务的系统.与单次调用的差异在于:单次调用是 开环生成;Agent 是 闭环决策,每
步可依据工具返回更新状态,直到终止条件.
