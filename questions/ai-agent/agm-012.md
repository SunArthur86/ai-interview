# 只做强相关性检索会有什么问题

会漏掉仍然有效但表述不相似的硬约束；也会过度偏向「像」但错误的片段。需要时间与重要性补足。

**追问应对**：若问「三路召回怎么融合？」——答：RRF（倒数排名融合）、或统一打分后线性加权 / Learning to Rank，线上 AB 测试调参。
