# 为什么用 JSON Schema 描述参数

三方面--(1)跨语言标准,各 SDK 统一;(2)可自动校验,避免脏参数进业务;(3)作为
模型「字段说明」,减少胡编参数名.缺点是 Schema 过长会占 token,需要精简描述或工具路
由.
