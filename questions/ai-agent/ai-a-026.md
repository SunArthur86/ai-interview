# AutoGen的GroupChat如何工作?发言人选择策略有哪些

- **AutoGen GroupChat架构:**

多个Agent在一个共享对话中协作,由GroupChatManager管理.

- **核心组件:**
1. **Agents** - 各有角色和工具(AssistantAgent/UserProxyAgent)
2. **GroupChatManager** - 管理对话轮次和发言人选择
3. **Speaker Selection** - 决定下一个发言者

- **发言人选择策略:**

1. **auto(默认)** - LLM根据对话上下文选择
2. **round_robin** - 按顺序轮流
3. **manual** - 人工选择
4. **random** - 随机选择
5. **custom function** - 自定义选择逻辑

- **典型配置:**
```python
manager = GroupChatManager(
groupchat=GroupChat(
agents=[planner, coder, reviewer],
messages=[],
max_round=20,
speaker_selection_method="auto"
)
)
```

- **适用场景:**
- 代码开发:Planner→Coder→Tester→Reviewer
- 研究任务:Researcher→Analyst→Writer
- 决策:多个专家Agent辩论
