# Conversation Buffer 和 Window Buffer 的区别与取舍

Buffer 强调完整保留（直到触顶）；Window 强调只保留尾部。取舍：若任务强依赖「很久以前的一条约束」，纯 Window 会丢信息，需要配合摘要或长期记忆检索。
