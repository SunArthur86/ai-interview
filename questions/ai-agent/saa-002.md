# Plan-and-Execute Agent和ReAct有什么区别？

ReAct：边想边做，每步Thought→Action→Observation循环。适合简单任务。

Plan-and-Execute：
1. Planner：LLM先制定完整计划（多步骤）
2. Executor：逐步执行每个子任务
3. Re-Plan：根据执行结果可能调整计划

优势：
- 减少LLM调用次数（Planner只需调用一次）
- 适合复杂、多步骤任务
- 每个Executor步骤可以用更小更快的模型

劣势：
- 计划可能不完整或过时
- 缺乏中间结果的即时反馈

实际应用：LangChain的PlanAndExecute、BabyAGI、AutoGPT等。
