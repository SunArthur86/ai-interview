# 生产环境记忆系统最容易出的事故是什么?怎么防

最常见事故是租户隔离失败与把敏感数据写入长期记忆未加密。

**防护措施**：强制租户过滤的集成测试、检索审计日志、敏感字段检测、密钥与 PII 脱敏、向量库最小权限访问。
