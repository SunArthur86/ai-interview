# OpenAI Function Calling中tool_choice和tool_call_id有什么作用

- **tool_choice - 控制工具选择行为:**

| 值 | 含义 |
|------|------|
| `"auto"` | 模型自己决定是否调用工具(默认) |
| `"none"` | 禁止调用任何工具 |
| `"required"` | **必须**调用至少一个工具 |
| `{"type":"function","function":{"name":"xxx"}}` | 强制调用指定工具 |

- **tool_call_id - 关联请求和响应:**

工具调用的两步流程中,tool_call_id将「模型请求调用」和「工具执行结果」关联起来.
```python
# Step 1: 模型决定调用工具
response = client.chat.completions.create(...)
tool_call = response.choices[0].message.tool_calls[0]
tool_call_id = tool_call.id  # "call_abc123"

# Step 2: 执行工具后返回结果
result = execute_tool(tool_call)

# Step 3: 把结果传回模型(必须带tool_call_id)
messages.append({
**"role"：** "tool",
**"tool_call_id"：** tool_call_id,  # 关键!
**"content"：** json.dumps(result)
})
```

- **为什么需要循环 for _ in range(5):**
模型可能需要多步工具调用(A的结果决定B),循环允许模型多次调用直到finish.
