# Embedding 模型升级后旧向量怎么办

旧向量与新模型不在同一向量空间，不可直接混合比较。

**处理方案**：双写期（新旧模型并行写）、离线全量重嵌入、检索时标明 embedding_model_version，确保查询与库中向量版本一致。
