# 什么是AI Agent?它和普通LLM对话有什么本质区别

- **AI Agent = LLM + 感知 + 规划 + 行动 + 记忆**

- **核心区别:**

| 维度 | 普通LLM | AI Agent |
|------|---------|----------|
| 交互 | 单轮问答 | **多轮自主** |
| 行动 | 只输出文本 | **调用工具/API** |
| 状态 | 无状态 | **有记忆** |
| 控制 | 用户驱动 | **Agent自主决策** |
| 终止 | 用户决定 | **Agent判断完成** |

- **Agent核心循环:**
```
while not done:
1. 感知(Perceive) - 接收输入/环境状态
2. 规划(Plan) - 决定下一步做什么
3. 行动(Act) - 调用工具/API
4. 观察(Observe) - 获取行动结果
5. 更新记忆(Memory) - 存储中间结果
```

- **典型Agent:** AutoGPT / Devin / Claude Code / Codex / Manus
