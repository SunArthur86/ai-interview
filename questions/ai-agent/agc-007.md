# 如何避免 Agent 在工具调用间「迷失」

明确 停止条件 与 最大步数;
维护 任务清单(todo) 与 当前子目标;
对每步输出要求 结构化(JSON);
关键步骤 强制验证(单元测试式检查、二次 LLM 审核).
