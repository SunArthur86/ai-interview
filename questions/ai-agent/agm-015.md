# 记忆和 Tool Use 的边界是什么

工具解决「当下获取外部世界状态」;记忆解决「跨时间保留与回忆」.边界:实时动态数
据应工具查;稳定偏好与历史事件应记忆存.
- **追问应对:**若问「库存算哪边?」--答:实时库存走工具;「用户常买类目」可走记忆或画
像.
