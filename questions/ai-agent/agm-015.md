# 记忆和 Tool Use 的边界是什么

工具解决「当下获取外部世界状态」；记忆解决「跨时间保留与回忆」。

**边界**：实时动态数据应使用工具查询；稳定偏好与历史事件应存储于记忆。

**追问应对**：若问「库存算哪边？」——答：实时库存走工具；「用户常买类目」可走记忆或用户画像。
