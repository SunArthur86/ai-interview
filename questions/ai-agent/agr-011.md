# BM25在中文场景下需要分词吗?中文全文检索有什么特殊处理

- **BM25在中文中的处理:**

- **需要分词!** BM25基于词项频率(TF)和逆文档频率(IDF),需要先将文本切分成词.

- **中文分词方案:**

| 工具 | 特点 | 
|------|------| 
| **jieba** | 最流行,支持自定义词典 | 
| **HanLP** | 功能全面,支持命名实体识别 | 
| **pkuseg** | 北京大学,学术领域效果好 | 
| **IK Analyzer** | Elasticsearch中文分词插件 |

- **Elasticsearch中文BM25:**
```json 
{ 
  "analysis": { 
    "analyzer": { 
      "chinese": { 
        "tokenizer": "ik_max_word" 
      } 
    } 
  } 
} 
```

- **中文BM25的坑:**
1. **分词错误** - 影响检索准确性
2. **未登录词** - 专业术语/新词需要自定义词典
3. **混合检索** - 中文+英文+数字需要特殊处理
4. **同义词** - 中文同义词丰富,需要扩展

- **替代方案:** BM25 + 向量检索混合,向量检索不需要分词
