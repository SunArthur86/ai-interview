# Agent 的记忆一般怎么设计

**分层设计最常见：** 工作记忆(当前轨迹与关键结论)+ 会话记忆(摘要滚动)+ 长期记忆(向
量检索/结构化库).写入要区分「事实」与「推断」,并带时间戳与来源,便于更新与撤销.
