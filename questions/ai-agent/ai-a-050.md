# 从Agent角度看AGI(通用人工智能)还有多远?当前Agent架构缺少什么

- **当前Agent的局限:**

| 维度 | 当前水平 | AGI需要 |
|------|---------|---------|
| 推理 | 单步/短链 | **开放世界长链推理** |
| 学习 | 无在线学习 | **持续学习+迁移** |
| 记忆 | 被动检索 | **主动整理+遗忘** |
| 规划 | 静态分解 | **动态自适应规划** |
| 世界模型 | 无 | **物理/因果/社会世界模型** |
| 自我意识 | 无 | **自我认知+目标生成** |

- **缺少的关键能力:**

1. **世界模型(World Model)**
   - 理解物理规律、因果关系
   - 能预测行动的长期后果
   - Yann LeCun的JEPA架构探索方向

2. **持续学习**
   - 不遗忘旧知识
   - 从少量经验快速适应
   - 当前模型训练后权重冻结

3. **抽象与泛化**
   - 跨领域知识迁移
   - 发现深层规律

4. **因果推理**
   - 理解因果而非相关
   - 反事实推理

- **乐观估计:** 5-10年
- **保守估计:** 20-50年
- **关键不确定:** 是否需要全新架构范式

- **AGI 路径探讨:**
   - **Scaling Laws (缩放定律派)**: 认为 Agent 只是 LLM 的 wrapper，随着模型参数和数据的指数级增长，推理能力会自然涌现。
   - **System 2 Thinking (慢思考派)**: 认为 Transformer 本质是快速直觉，需要引入类似 "AlphaGo" 的搜索算法进行深思熟虑的规划才能达到 AGI。
   - **Embodied AI (具身智能派)**: 认为 AGI 必须通过与物理世界的交互来建立 grounding，单纯的文本模型无法真正理解世界。

## 常见考点
- **当前的 Self-Attention 机制在处理超长序列时有什么瓶颈？** (答案: 计算复杂度为 O(N²)，无法处理无限长的真实世界交互流)
- **为什么说目前的 Agent 没有真正的「记忆」？** (答案: 主要是检索拼接，缺乏内化和重组知识的能力，没有类似海马体的巩固机制)
- **System 1 vs System 2 在 AI 中指什么？** (答案: System 1 是 LLM 的快速生成，System 2 是显式的搜索/规划/反思过程)
