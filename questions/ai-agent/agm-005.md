# 长期记忆为什么常用向量数据库?有什么局限

常用是因为语义检索能处理「换说法」的匹配。局限包括：相似≠正确（会召回到表面相近的噪声）、难精确匹配（账号、订单号更适合关键字/关系库）、更新一致性需业务层保障。
