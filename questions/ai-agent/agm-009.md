# 增量摘要误差累积怎么缓解

缓解方法：(1)定期全量重摘要；(2)摘要中保留关键事实清单（姓名、日期、硬约束）；(3)对摘要做一致性检查（另一个小模型挑错）；(4)让用户可编辑「长期事实」。
