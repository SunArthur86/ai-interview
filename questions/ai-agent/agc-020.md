# 为什么「让模型自己选工具」可能不如「路由器 + 规则」

在 域窄、路径稳定 的场景,路由器更 省成本、可测试、行为确定;全模型路由在 开放域 更
灵活.最佳实践常是 混合:易分类走规则,难例走模型.
