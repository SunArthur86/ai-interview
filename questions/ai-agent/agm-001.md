# 为什么 Agent 需要记忆?没有行不行

需要。原因包括：(1)多轮一致性——避免重复追问、前后矛盾；(2)长任务——工具调用链、子目标状态需要延续；(3)个性化——偏好、禁忌、领域术语；(4)成本——不必每次把全量背景塞进 prompt。
