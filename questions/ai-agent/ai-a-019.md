# 如何设计Agent的长期记忆系统?mem0/MemGPT的设计思路是什么

- **长期记忆核心挑战:**

不是所有历史都有价值.需要决定记住什么、遗忘什么、何时检索.

- **MemGPT设计思路:**

**借鉴操作系统的虚拟内存管理：** 
- **主存(上下文窗口)** - 当前活跃的记忆
- **外部存储** - 向量数据库
- **分页机制** - 自动将记忆在主存和外部存储间调度

- **mem0设计思路:**

1. **记忆提取**
- 每轮对话后,LLM自动提取值得记住的信息
- 「用户喜欢Python」「项目用React」

2. **记忆存储**
- 结构化存储(key-value)
- 向量索引用于语义检索

3. **记忆更新**
- 新信息覆盖旧信息(如用户改了偏好)
- 冲突检测和解决

4. **记忆检索**
- 根据当前对话上下文检索相关记忆
- 注入prompt辅助生成

- **记忆类型:**
- **语义记忆** - 事实和知识
- **情景记忆** - 具体事件和经历
- **程序记忆** - 学到的技能和工作流
