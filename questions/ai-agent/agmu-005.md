# 民主讨论模式如何避免永远开不完会

需要硬终止条件：最大轮数、Token 预算、无新信息阈值（连续两轮无实质变更则停）、或主席裁决；并配合结构化发言（观点 + 证据 + 反对意见）减少废话。

**追问应对**：若问「讨论适合生产吗？」——答：适合低风险创意类或人类在环；纯自动高风险决策通常要仲裁 + 规则引擎。
