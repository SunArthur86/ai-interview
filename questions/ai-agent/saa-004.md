# Function Calling和工具调用有什么区别？

**答案**：
Function Calling：LLM厂商提供的原生API能力，LLM根据用户意图输出结构化函数调用。

流程：
1. 定义函数schema（名称、参数、描述）
2. 用户输入 + schema发给LLM
3. LLM返回要调用的函数名和参数（JSON）
4. 应用执行函数，将结果返回给LLM
5. LLM基于结果生成最终回答

工具调用 = Function Calling + 执行环境

关键区别：
- Function Calling只输出调用意图，不执行
- 工具调用包含实际执行和结果反馈

最佳实践：
1. 函数描述要清晰（让LLM理解何时用）
2. 参数schema要精确（类型、必填、枚举）
3. 并行调用：现代LLM支持一次返回多个函数调用
4. 错误处理：函数执行失败时返回错误信息给LLM重试
