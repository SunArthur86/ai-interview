# MCP（Model Context Protocol）是什么？解决了什么问题？

MCP是Anthropic于2024年底提出的开放协议，标准化了LLM应用与外部工具/数据源之间的连接。

解决的问题：
- 每个工具需要单独实现Function Calling的schema和逻辑
- 不同LLM平台的工具调用格式不统一
- 工具开发重复劳动

MCP架构：
- MCP Host：LLM应用（如Claude Desktop）
- MCP Client：Host中管理连接的组件
- MCP Server：提供工具/资源/提示

通信方式：
- 本地：stdio（标准输入输出）
- 远程：SSE + HTTP

三大能力：
1. Tools（工具）：可执行的函数（如搜索、查数据库）
2. Resources（资源）：可读取的数据（如文件、数据库记录）
3. Prompts（提示）：预设的提示模板

类比：MCP之于AI Agent = USB之于电脑——统一接口标准。
