# 如何评测记忆系统好坏

**离线评估**：召回率/精确率（给定标注 relevant memories）、摘要一致性、冲突率。

**在线评估**：任务成功率、用户纠正次数、成本与延迟。

**追问应对**：补充人工抽检与 bad case 归因（检索错 vs 摘要错 vs 写入错）。
