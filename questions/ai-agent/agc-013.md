# ReAct 框架里三个字母代表什么?解决什么问题

Reasoning + Acting：在生成中交替进行推理与行动，并接收观察。它解决的是：模型仅「空想」容易偏离事实；通过显式推理 + 工具反馈把推理锚定在真实环境上。
