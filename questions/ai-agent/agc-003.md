# Agent 和 Prompt Chain 有什么本质区别

Prompt Chain 的拓扑与顺序主要由工程侧固定;Agent 在运行时在动作空间中做选择,并依
赖 Observation 更新信念,适合输入与路径不确定的任务.二者可结合:链负责稳定流程,
Agent 负责链内某段的灵活分支.
