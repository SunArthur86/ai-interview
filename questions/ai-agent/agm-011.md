# 混合检索怎么去重与限长

**去重**：同一事实的不同表述可使用语义去重（相似度阈值）或 Canonical Key（实体对齐）。

**限长**：按最终的 rerank_score 排序后进行 Token 装箱；或采用分层注入策略，优先注入摘要，细节按需加载。

**追问应对**：若问「去重会不会误删相似但不同约束？」——答：使用阈值 + 冲突检测（矛盾触发人工或二次 LLM 仲裁），而非基于纯相似度合并。
