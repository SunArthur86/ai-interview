# 为什么说 Prompt 是 Agent 的「软代码」

因为 Agent 的行为策略(何时推理、何时调工具、输出什么格式)大量编码在 System/User
Prompt 与模板里,变更 Prompt 就像改业务规则,需要版本管理与评审,类似代码.
