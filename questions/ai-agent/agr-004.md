# RAG系统中PDF解析有哪些常见坑?Unstructured和Apache Tika各有什么优缺点

- **PDF解析常见坑:**

1. **多栏布局** - 文字顺序错乱
2. **表格** - 被拆成碎片
3. **图片/公式** - 纯文本提取丢失
4. **页眉页脚** - 噪声混入正文
5. **扫描件** - 需要OCR
6. **编码问题** - 特殊字符乱码

- **工具对比:**

| 工具 | 优势 | 劣势 |
|------|------|------|
| **Unstructured.io** | 保留文档结构(标题/表格/图)、支持40+格式 | 安装复杂、依赖多 |
| **Apache Tika** | Java生态、支持1000+格式、轻量 | 结构化能力弱 |
| **PyMuPDF(fitz)** | 速度快、精确控制 | 需手动处理布局 |
| **Marker** | Markdown输出、表格转MD | 仅学术PDF |
| **pdfplumber** | 表格提取强 | 大文件慢 |

- **最佳实践:**
- 通用文档 → Unstructured
- 技术PDF → Marker
- 表格密集 → PyMuPDF + Camelot
- 简单文本 → Tika
