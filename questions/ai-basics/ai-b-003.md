# 什么是视觉指令微调(Visual Instruction Tuning)?它为什么能让 VLM 具备对话能力

- **视觉指令微调** 是让视觉语言模型(VLM)学会遵循人类多模态指令的技术。

- **核心思想**
- 类比 LLM 的指令微调(text-only instruction tuning)，但在输入中混合图像和文本
- 目标：让模型理解'看图回答问题'的指令格式

- **LLaVA 的做法(最经典实现)**

**Step 1: 构造指令数据**
- 用 GPT-4/Claude 基于图像描述自动生成多模态指令
- 格式：[IMAGE] + "请描述这张图片中的异常情况" → 回答
- 包含：描述、推理、对话、详细描述 4 种任务

**Step 2: 两阶段训练**
- **Stage 1: 特征对齐预训练**
  - 冻结 LLM 和 Vision Encoder
  - 只训练 Projection Layer(MLP)
  - 目标：让视觉特征对齐到 LLM 的 embedding 空间
- **Stage 2: 端到端指令微调**
  - 冻结 Vision Encoder
  - 微调 LLM + Projection Layer
  - 目标：学会遵循多模态指令

- **为什么有效**
1. 投影层让视觉特征'翻译'成 LLM 能理解的'伪文本 token'
2. 指令微调让模型学会'看图 → 理解意图 → 生成回答'的模式
3. 利用 LLM 已有的推理和对话能力

- **关键改进**：LLaVA-1.5 增加了多轮对话数据和高分辨率支持。
