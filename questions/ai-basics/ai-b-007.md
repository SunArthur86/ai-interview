# 什么是过拟合(Overfitting)?如何判断和防止

- **过拟合** = 模型'死记硬背'训练数据,对新数据表现差.

- **类比**:学生只背答案不理解,考试换个数字就不会了.

- **判断方法**
- 训练集准确率高(95%),但验证集/测试集准确率低(70%)
- 训练 loss 持续下降,验证 loss 开始上升

- **防止方法**
1. **更多数据**:数据多了不容易死记
2. **正则化**:L1/L2 惩罚大权重
3. **Dropout**:训练时随机丢一些神经元
4. **早停(Early Stopping)**:验证集变差就停止训练
5. **数据增强**:翻转/裁剪/噪声扩充数据
6. **简化模型**:减少层数/参数量

- **一句话理解**:过拟合 = 模型'背题'不'学知识',防过拟合 = 让模型'理解'而非'记忆'.
