# 神经网络是怎么'学习'的?用一句话和一个例子解释.

- **一句话**:神经网络通过'猜答案→看对错→调整'的循环不断改进.

- **例子:学认猫**
1. **猜**:看图片,输出'这是猫的概率:0.3'
2. **看对错**:实际是猫,但模型说 0.3 → 差距(loss)= 0.7
3. **调整**:反向传播调整参数,让下次猜得更接近 1.0
4. **重复**:看百万张图后,猜得很准

- **核心概念**
- **Loss(损失)**:猜的答案和正确答案的差距
- **Gradient(梯度)**:Loss 关于参数的变化方向
- **Optimizer(优化器)**:按梯度方向调整参数
- **Learning Rate(学习率)**:每次调整的步长

- **一句话理解**:神经网络学习 = '反复猜→对答案→微调'直到猜得很准.
