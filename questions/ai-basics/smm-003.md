# Stable Diffusion的原理是什么？Latent Diffusion如何工作？

🎯 本质：Stable Diffusion是一种潜在扩散模型(Latent Diffusion Model)，在压缩的潜在空间中进行扩散和去噪，而非在像素空间，大幅降低计算成本。

📊 Diffusion基本原理：
前向过程（加噪）：逐步给清晰图片加高斯噪声，直到变成纯噪声
  x_0(原图) -> x_1 -> x_2 -> ... -> x_T(纯噪声)
  每步加少量噪声，T通常为1000
  
反向过程（去噪）：训练一个UNet学习逐步去噪
  x_T(噪声) -> x_{T-1} -> ... -> x_0(生成图)
  每步预测并去除一点噪声

Latent Diffusion创新：
问题：在像素空间(512x512x3)做diffusion计算量太大
解决：先用VAE将图片压缩到潜在空间(64x64x4)，在潜在空间做diffusion
  计算量减少：512*512*3 / 64*64*4 = 48倍

Stable Diffusion组件：
1. VAE Encoder/Decoder：图像 <-> 潜在空间
2. UNet：在潜在空间去噪（核心模型）
3. Text Encoder（CLIP）：将文本prompt编码为条件
4. Scheduler：控制去噪步骤（DDIM/DPM++/Euler）

文本控制生成（Classifier-Free Guidance）：
  无条件预测：UNet(z_t) - 模型自由生成
  有条件预测：UNet(z_t, text) - 按prompt生成
  实际输出 = 无条件 + guidance_scale * (有条件 - 无条件)
  guidance_scale越大，越遵循prompt但多样性降低

关键参数：
- steps（步数）：20-50步，越多质量越好但越慢
- cfg_scale（引导强度）：7-12，越高越遵循prompt
- sampler（采样器）：DPM++ 2M Karras（速度质量平衡好）
- seed（随机种子）：相同seed=相同输出
