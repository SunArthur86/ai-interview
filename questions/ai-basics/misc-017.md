# 大模型训练中为什么大量使用合成数据?合成数据有什么风险

- **合成数据使用场景:**
1. **领域数据不足** - 医疗/法律/低资源语言
2. **SFT数据生成** - 用强模型生成指令-回复对
3. **偏好数据生成** - 自动生成排序对
4. **推理数据** - 自动生成CoT推理路径

- **主要方法:**
- Self-Instruct:用强模型自动生成训练数据
- Evol-Instruct:递增复杂度的指令生成

- **风险:**
1. **模型坍缩 (Model Collapse)** - 用模型生成数据训练下一代模型,多样性递减
2. **偏见放大** - 生成数据的偏差被放大
3. **幻觉传播** - 模型幻觉产生的错误数据被训练

- **缓解措施:**
- 混合真实数据(至少30%)
- 质量过滤(perplexity/分类器/人工抽检)
- 多样性约束(embedding聚类去重)
