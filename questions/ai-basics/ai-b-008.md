# CLIP 的训练原理是什么?它如何实现图文对齐

- **CLIP (Contrastive Language-Image Pre-training)**

OpenAI 提出,通过对比学习实现图像和文本的对齐表示.

- **训练流程**

1. **输入**:N 个(图片, 文本)配对
2. **编码**:
- Image Encoder(ViT/ResNet)→ 图像向量 I₁...Iₙ
- Text Encoder(Transformer)→ 文本向量 T₁...Tₙ
3. **对比学习**:
- 构造 N×N 相似度矩阵
- 正样本对(匹配的图文)在矩阵对角线 → 拉近
- 负样本对(不匹配的图文)在非对角线 → 推远
4. **Loss**:对称的 InfoNCE loss
- 图像→文本方向的 Cross-Entropy
- 文本→图像方向的 Cross-Entropy

- **为什么有效?**
- 4 亿图文对训练 → 学到通用图文映射
- 零样本分类:用文本 'a photo of a cat' 和 'a photo of a dog' 做分类
- 语义搜索:文本 query 检索图片

- **对后续影响**
- LLaVA/Stable Diffusion 都使用 CLIP 的 text encoder
- OpenCLIP / EVA-CLIP 扩展到更大规模
- 中国版:Chinese-CLIP(阿里)
